# SYS-017 PK（ＰＫ / プレイヤーキル）

- 分类：战斗系统 / 社会秩序
- 适用范围：全服共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：中

## 摘要
`PK`指玩家主动攻击并杀害其他玩家的行为，或实施该行为的玩家本身。原作已明确：这不是日本服独有现象，而是《幻境神话》底层规则允许存在的一部分；真正会变化的是它在不同服务器、不同社会环境中的流行程度、收益结构与约束条件。

## 规则性质
- 当前最稳妥的写法是：`PK`是系统允许存在、但不必然普遍的玩家间暴力行为。
- 因此本页要区分两层：
  - 底层规则允许与否
  - 某个服务器社会中它是否常见、被如何抑制或放大

## 游戏时代底层规则并未全面禁止PK
- 游戏时代的《幻境神话》并不全面禁止玩家之间的战斗。
- 既然城市区域会被明确标注为`禁止战斗区域`，就意味着除此之外的区域并非一律禁止玩家互相攻击。
- `PK`因此是系统容许、但不一定常见的行为。
- 关联出处：
  - PDF页码：139
  - 章节标题：`007`
  - 关键词：`ＰＫ`、`プレイヤーキル`、`禁止されていない`
  - 日文短摘录：`ＰＫ` / `プレイヤーキル` / `それ以外のゾーンにおいてそれらは「禁止されていない」`

## 日本服过去不流行有明确原因
- 开篇明确给出了几个抑制`PK`的旧条件：小地图会显示附近所有角色，高等级角色会自动回避部分攻击，运营方还会以骚扰行为为由进行事后处罚。
- 另外，文中也点明了日本服务器的文化因素，即日本玩家整体上并不偏好玩家间暴力。
- 因此，“日本服过去较少见PK”应视为日本服社会生态，而不是全服天然共性。
- 关联出处：
  - PDF页码：139-140
  - 章节标题：`007`
  - 关键词：`ミニマップ`、`回避する`、`ハラスメント`、`日本サーバーの文化的な問題`
  - 日文短摘录：`ミニマップ` / `加えられた攻撃を回避する` / `ハラスメント` / `日本サーバーの文化的な問題`

## 大灾害后风险与收益结构发生逆转
- 大灾害后，脑内菜单中不再存在小地图，玩家也不会自动做出无意识回避。
- 原本由运营公司在事后执行的骚扰处罚也一并消失，等于失去了“神之手”式的外部秩序维护。
- 与此同时，击杀其他玩家可以获得其全部现款与约半数可交易道具，使`PK`的收益显著上升。
- 文中将这种变化概括为“风险与收益的逆转”，并明确认为这是`PK`增殖的重要原因。
- 关联出处：
  - PDF页码：140-141
  - 章节标题：`007`
  - 关键词：`神の手`、`現金全て`、`アイテムの約半分`、`逆転`
  - 日文短摘录：`神の手` / `現金全てと、アイテムの約半分` / `デメリットとメリットの逆転`

## PK是否泛滥取决于服务器社会生态
- 底层规则允许`PK`，并不等于每个服务器都会以同样速度、同样形式出现掠夺玩家群体。
- 日本服后来出现的问题，是在特定社会条件与秩序崩解背景下暴露出来的。
- 因此中原服创作应区分“规则可行”与“社会是否发展出相应生态”。

## 阶段变化
- 规则阶段：先确认`PK`在系统上并未被全面禁止。
- 抑制阶段：游戏时代又有地图、自动回避、运营处罚与本地文化等抑制条件。
- 逆转阶段：大灾害后这些抑制条件消失，而收益明显上升。
- 这几层必须并列保留，不能把“灾后PK增殖”反写成它在旧游戏时代就同样常见。

## 对中原服的参考价值
- 中原服若沿用原作底层规则，则`PK`机制本身应视为可直接继承的共通设定。
- 但“是否常见、由谁主导、是否形成公会化掠夺、普通玩家如何应对”则属于各服务器的社会生态差异，不应照搬日本服的稀少或增殖速度。
- 这给了你很大空间去写中原服本地秩序、边境暴力与战争化掠夺的不同形态。

## 当前未知或仍待补证的部分
- 各服务器对`PK`的非系统性规范、报复机制和公会应对模式，仍需分别在本地词条中展开。
- 原作是否在后文补出更严格的`PK`技术限制或例外，当前尚未在本页汇总完毕。
