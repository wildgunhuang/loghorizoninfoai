# SYS-022 共感子（エンパシオム）

- 分类：系统与世界机制 / 关键资源
- 适用范围：大概率共通
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`共感子`是后期出现的关键资源概念。当前材料能够稳定确认：它是`航界种`文明高度重视、并以之为采集目标的资源；而在城惠的后期推断里，它还很可能与`冒险者`的超常能力、`典灾`的行动目标、以及“归还地球”的交易条件直接相关。不过，关于它的精确定义，现阶段仍有相当部分停留在推断层。

## 资源性质
- 现阶段最稳妥的理解方式，是把`共感子`视为高层文明争夺的关键资源专名。
- 不应把它简单翻成“魔力”“MP”“灵魂能量”中的任何一种；更合理的做法是保留专名，并按出处逐步收紧边界。

## 作为异界文明的核心资源被明示
- `成惠2`明确说，她们一方社会的任务是探索并采集`共感子`。
- 她还说明，创造者社会已经因为这一资源的上限与枯竭问题进入长期停滞与衰退。
- 这说明`共感子`并非普通燃料，而是足以制约文明扩张的根本资源。
- 关联出处：
  - PDF页码：2289
  - 章节标题：`087`
  - 关键词：`共感子`、`資源`、`停滞`、`衰退`
  - 日文短摘录：`〈共感子〉と呼ぶ資源` / `停滞、そして衰退期`

## 在赛尔迪希亚中的总量巨大
- 同一组后期资料还说明：`赛尔迪希亚`所包含的`共感子`总量极大，甚至可与其出身宇宙的一定比例总量相比。
- 这也是为什么`赛尔迪希亚`会成为被高度关注的目标。
- 关联出处：
  - PDF页码：2291
  - 章节标题：`087`
  - 关键词：`共感子の総量`、`巨大利権`
  - 日文短摘录：`〈共感子〉の総量` / `巨大利権`

## 早期材料提示它可能与情感、共鸣有关
- 在更早的接触里，`成惠2`曾说“共感子会传达”，并把它与人类那种她难以完全理解的交流方式联系起来。
- 这段不是严格定义，但至少说明：`共感子`未必只是狭义的“电池式能源”。
- 判定：`情境性表现`
- 关联出处：
  - PDF页码：1954
  - 章节标题：`067`
  - 关键词：`共感子は伝わる`、`きみたちの流儀`
  - 日文短摘录：`でも〈共感子〉は伝わる` / `きみたちの流儀`

## 与典灾行动目标高度相关
- 后期多段材料都显示，`典灾`的行动与`共感子`高度相关。
- 城惠在战斗中已经明确推断：`塔里库坦`同样在寻求`共感子`。
- 后面又把`卡麦萨尔`、`巴古利司`、`塔里库坦`、`エレイヌス`等不同`典灾`的行为统一解释为围绕`共感子`展开。
- 关联出处：
  - PDF页码：2414、3295
  - 章节标题：`087` / `117`
  - 关键词：`共感子を求めている`、`同意`、`移譲`
  - 日文短摘录：`タリクタンもまた〈共感子〉を求めている` / `提供者の同意を必要とする`

## 冒险者超常能力的源泉说法属于后期高概率推断
- 到更后期，城惠已经提出一个非常重要的推断：`冒险者`的超人能力，可能正是以`共感子`为源泉驱动。
- 他同时推断：
  - `共感子`平时在个人内部循环
  - 个体单位之外的运用受到严格限制
  - 若要真正大规模取得，往往需要提供者同意
- 这段并不是官方规则说明，而是城惠根据`典灾`行为、`契约术式`逻辑与自身经验做出的高强度推断。
- 判定：`高概率效果`
- 关联出处：
  - PDF页码：3295
  - 章节标题：`117`
  - 关键词：`超人的能力`、`源泉`、`循環`、`同意`
  - 日文短摘录：`超人的能力はこの〈共感子〉` / `個人内部で循環` / `提供者の同意を必要とする`

## 与归还地球交易的关联目前仍属线索
- `塔里库坦`事件后，城惠又把昏睡者的“归还”问题与`共感子`联系起来怀疑。
- 这一层目前仍不能写成既成事实，但已足够视作“归还地球机制”的重要线索。
- 判定：`待考证`

## 阶段变化
- 较早材料里，`共感子`首先只是`成惠2`提到的高层资源名。
- 随后，它又被明确推进为足以制约异界文明兴衰、并让`赛尔迪希亚`成为巨大利权目标的关键资源。
- 到更后期，城惠进一步把它与`典灾`行动目标、`冒险者`超常能力来源、以及“归还地球”的交易条件联系起来，使其从“资源名”发展成后期世界观中枢线索。

## 对中原服的参考价值
- `共感子`非常适合保留为中原服后期世界观的关键资源名。
- 但现阶段最稳的继承方式是：
  - 承认它是高层资源
  - 承认它与`航界种 / 典灾 / 归还`密切相关
  - 对它的精确物理定义与运作方程保持开放

## 当前未知或仍待补证的部分
- `共感子`的精确定义究竟更接近能量、信息、情感载体还是复合资源，当前仍未被正式写死。
- “提供者同意”的机制究竟是物理限制、契约限制还是宇宙原则限制，现阶段也还不能确定。
- 它与`一致`、`神隠し`、`归还`、`航界种`技术体系之间的完整转换关系，仍需更多材料补证。
