# WLD-013 古代种（古来種）

- 分类：世界观 / 居民群体
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`古代种`是《记录的地平线》后期逐渐被明确出来的一类强力住民。当前材料能够稳定确认：它并不是独立于`大地人`之外的新种族，而更接近“大地人中的强者总称”。在日本服语境里，它常与`全界十三骑士团`、`妖精乡`等设定连在一起；到了中原服，则进一步出现了`仙人 / 仙女 / 道人 / 真人 / 仙君 / 圣母`等本地化异称。

## 群体性质
- 现阶段最稳妥的理解方式，是把`古代种`视为一种“高位大地人”总称，而不是新的生物分类。
- 它是一层能力与地位上的上位标签，会在不同地区被包进不同文化语言中。

## 不是独立种族，而是大地人中的强者
- 文中明确说明：`古代种`这个名字只是便宜称呼。
- 它虽然带有“种”字，但并不意味着独立种族或生物分类。
- 至少在艾利亚斯的理解里，`古代种`只是`大地人`之中那些“特别强”的存在；若要找近似词，更接近“超人”。
- 关联出处：
  - PDF页码：2499
  - 章节标题：`096 Chapter 2.01`
  - 关键词：`古来種`、`大地人`、`強い`、`超人`
  - 日文短摘录：`種族でもなければ種別でもない` / `〈大地人〉の一種` / `「超人」だろうか`

## 与骑士团体系常有联系，但并非绝对从属
- 同一段还说明，许多`古代种`确实属于`全界十三骑士团`。
- 但这并不是绝对保证，只是长期运作后形成的结果。
- 正因为`古代种`本质上只是“持有强大力量的大地人总称”，所以也存在邪恶的、遁世的、未加入骑士团的一类。
- 这条边界很重要，它避免把`古代种`误写成统一服从同一组织的固定阶层。
- 关联出处：
  - PDF页码：2499
  - 章节标题：`096 Chapter 2.01`
  - 关键词：`全界十三騎士団`、`絶対の保証はない`、`悪の古来種`
  - 日文短摘录：`多くが〈全界十三騎士団〉に所属` / `絶対の保証はない` / `悪の〈古来種〉`

## 也会管理特殊隐居地与异质区域
- 后文对`仙境 / 妖精乡`的解释进一步显示，`古代种`会管理某些隐藏于深山或密林中的特殊区域。
- 这些地点往往带有强烈的古老魔力，并与稀有魔法道具、异常现象相连。
- 因此，`古代种`不仅是“战斗很强的大地人”，也常常与特定地域、秘境与守护职责结合。
- 关联出处：
  - PDF页码：2512
  - 章节标题：`097 Chapter 2.02`
  - 关键词：`古来種`、`管理`、`特殊な地域`
  - 日文短摘录：`〈古来種〉の管理する特殊な地域` / `深い森の中や山の中に存在`

## 中原服存在成体系的本地异称
- 到中原服语境时，`古代种`不再只以统一称呼出现。
- 文中明确说，中原服务器里被称为`仙人`的一系列人物，其实就是`古代种`。
- 男性可称`仙人`，女性可称`仙女`；除此以外还有`道人`、`真人`、`仙君`、`圣母`等异称。
- 这说明中原服并非简单照搬西式奇幻命名，而是把同一住民类别包进了本地神话语言中。
- 关联出处：
  - PDF页码：2499
  - 章节标题：`096 Chapter 2.01`
  - 关键词：`仙人`、`仙女`、`道人`、`真人`、`仙君`、`聖母`
  - 日文短摘录：`〈仙人〉と呼ばれる一連の人々が〈古来種〉` / `男性であれば〈仙人〉女性であれば〈仙女〉`

## 中原服中的古代种具有地方名望体系
- 文中还顺手举了中原服本地知名个体的例子。
- 例如被称为`白翼姬`的`铃香凤`，就是中原侧高知名度的`古代种`，并且属于`翡翠骑士团`。
- 这一点说明：中原服的`古代种`并不是背景板，它们也会拥有本地英雄名号、组织归属和社会辨识度。
- 关联出处：
  - PDF页码：2499
  - 章节标题：`096 Chapter 2.01`
  - 关键词：`白翼姫`、`鈴香鳳`、`翡翠騎士団`
  - 日文短摘录：`〈白翼姫〉と呼ばれる鈴香鳳` / `〈翡翠騎士団〉の筆頭仙姫`

## 这些中原称号仍然是活着的社会称呼
- 后文里，花貂等本地住民会把带有异常状态的克拉斯提直接认成`仙君`。
- 这说明`仙人 / 仙君 / 仙女`这套说法并不是只存在于古书或设定说明里的死名词，而是中原当地仍会拿来现实辨认高位存在的活称呼。
- 这对写作很重要，因为它意味着中原服角色之间的互称、误认和敬语体系，可以直接建立在这些词上。
- 关联出处：
  - PDF页码：2502-2507
  - 章节标题：`097 Chapter 2.01`
  - 关键词：`仙君様`、`仙人`
  - 日文短摘录：`仙君様` / `仙人などに間違えられている`

## 阶段变化
- 较早材料里，`古代种`首先被明确成“并非独立种族、而是大地人中的强者总称”。
- 随后它又与`全界十三骑士团`、秘境管理、世界守护职责等日本服高层结构连接起来。
- 到中原线，这一总类进一步获得了`仙人 / 仙女 / 道人 / 真人 / 仙君 / 圣母`等成体系本地化异称，成为你写中原服时最关键的接桥设定之一。

## 对中原服的参考价值
- 这条设定对你很关键，因为它把“中原的仙人体系”与原作已有大框架直接接上了。
- 也就是说，中原服可以保留高度本土化的神话语言与社会形象，但底层上仍与全服共通的`古代种`框架连通。
- 这样既能遵守原作，又能写出鲜明的中原服务器气质。

## 当前未知或仍待补证的部分
- `古代种`内部是否存在更严格的能力阶序、法统等级或寿命差异，当前仍未被系统化说明。
- 日本服与中原服这两套称呼体系之间是否一一对应，还是存在地方额外细分，目前也还不能写死。
- `古代种`与`天吏`、`仙境`、`西王母娘娘`等中原高位结构之间的制度关系，正文仍需更多材料补证。
