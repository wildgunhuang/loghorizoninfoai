# ORG-007 苍王派（蒼王閥）

- 分类：组织与制度 / 中原服政治派系
- 适用范围：特定服务器特有
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`苍王派`是当前正文中最早被明确点名的中原服大型王号派系之一。现有材料能够稳定确认的不是其完整组织章程，而是它在边境现实中的作用方式：会通过`歌剑团`这类同阵营公会向地方势力施压，要求归附；同时又以“文治派”包装自身，使扩张不只表现为武力兼并，也表现为对秩序与治理名义的争夺。

## 派系性质
- 现阶段最稳妥的理解方式，是把`苍王派`视为中原服大战争生态中的大型政治派系，而不是单一公会。
- 正文里边境公会被要求加入的对象并不是`歌剑团`本身，而是`苍王派`。
- 这说明`苍王派`至少是高于单一公会的一层政治归属框架。
- 关联出处：
  - PDF页码：2481-2482
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`蒼王閥`、`歌剣団`、`傘下`
  - 日文短摘录：`蒼王閥に入らないと後悔をする` / `〈歌剣団〉の使者` / `傘下に入れ`

## 对地方公会的扩张方式
- `乐浪狼骑兵`接到的警告非常直接：如果不加入`苍王派`，就会“后悔”。
- 这表明`苍王派`的扩张并不是抽象招募，而是已经能向边境中小公会形成现实压力。
- 关联出处：
  - PDF页码：2481
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`蒼王閥`、`後悔`
  - 日文短摘录：`蒼王閥に入らないと後悔をする`

## 会借同阵营公会向地方施压
- 朱桓视角下，反复上门的使者来自`歌剑团`。
- 但对方要求的不是单纯加入`歌剑团`，而是进入`苍王派`一侧。
- 因而目前最合理的整理方式，是把`歌剑团`视为`苍王派`阵营对地方推进时的直接触手之一。
- 关联出处：
  - PDF页码：2481-2482
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`歌剣団`、`蒼王閥`、`傘下`
  - 日文短摘录：`〈歌剣団〉の使者` / `蒼王閥に入らないと` / `傘下に入れ`

## 施压逻辑带有“保护”与“恐吓”双重性
- 来访者一方面强调不归附会吃亏，另一方面又把地方`大地人`的安危一并拉进谈判。
- 这种说法不是普通外交辞令，而是把“接受统合”包装成“避免更坏后果”的现实威胁。
- 这使`苍王派`更像会把人口、治安和战争风险一起纳入控制逻辑的大战争派系。
- 关联出处：
  - PDF页码：2481
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`大地人も皆殺し`、`臣従`
  - 日文短摘录：`この都の〈大地人〉も皆殺し` / `臣従せよ`

## 对外会经营“文治派”形象
- 马抱特别提到，`苍王`一侧对外的招牌是“文治派”。
- 这条信息很重要，因为它说明中原服的大势力并不只是靠武力压服，也会主动争夺“谁更会治理”的合法性叙事。
- 关联出处：
  - PDF页码：2485
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`文治派`
  - 日文短摘录：`蒼王さん文治派ってふれこみ`

## 其扩张可能与长期战争的社会成本有关
- 朱桓与马抱的讨论补出一个很重要的判断：长期大战已经开始伤害`大地人`、农业和商业。
- 如果这一判断成立，那么`苍王派`近期加强对边境地方的吸纳，未必只为了继续扩地，也可能带有重新整编支配区、降低内耗的意图。
- 判定：`高概率效果`
- 关联出处：
  - PDF页码：2485
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`大地人に被害`、`畑も商業も止まっちゃう`
  - 日文短摘录：`〈大地人〉に被害出ちゃう` / `畑も商業も止まっちゃう`

## 与白王、红王派构成并列竞争框架
- 威胁话术里直接把`白王`与`红王派`拿来并提。
- 这说明中原服内部至少存在若干互相竞争的王号派系，而`苍王派`只是其中之一。
- 关联出处：
  - PDF页码：2481
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`白王`、`紅王`
  - 日文短摘录：`白王や紅王じゃ`

## 当前已知的活动边界
- 以当前正文可确认的信息看，`苍王派`已经能把影响力投到`草原之都`这种中原外缘城市。
- 但它的中心据点、常设军力规模、内部会议或领导层构成，正文还没有足够展开。
- 因而目前更适合把它写成“已确认会向边境扩张的王号派系”，而不是已经完整还原的政体词条。

## 阶段变化
- 命名阶段：先作为与`白王`、`红王派`并列的王号派系名出现。
- 扩张阶段：很快补出它会借`歌剑团`等同阵营公会向边境地方施压、要求归附。
- 合法性阶段：后文又补出其“文治派”包装，说明它争夺的不只是武力优势，也包括治理正当性叙事。
- 这几层都应并列保留，不能只把`苍王派`写成简单的军事威胁标签。

## 对中原服的参考价值
- `苍王派`提供了一种很有中原特色的大型势力模板：
  - 不是单一公会，而是更高一层的政治归属框架
  - 会借附属或同阵营公会向边境推进
  - 会把兼并叙事包装成“文治”“秩序”“保护”
  - 会把人口、经济与战争一并纳入派系统治逻辑
