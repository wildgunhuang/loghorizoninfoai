# GEO-004 大神殿（大神殿）

- 分类：场景与地理 / 复活设施
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`大神殿`是作品中逐步明确下来的`冒险者`标准复活设施与转送目的地。前期它首先以“死后会在城市的大神殿复活”的观察出现；后期则进一步被纳入`魂魄理论`，成为`冒险者`自我苏生与肉体重构的关键地点。与此同时，作品也补出了区域入口复活点与`北风之移动神殿`等变体，说明复活设施并非永远只限于固定建筑。

## 设施性质
- 当前最稳妥的写法是：`大神殿`是城市级标准复活设施的基准形态。
- 它既是机制性地点，也是城市秩序、心理安全感与战术选择中的重要节点。
- 因此本页应同时记录“建筑 / 设施 / 机制目的地”三层含义。

## 前期先作为实际复活地点被观察到
- 在开篇后不久，文本已经把`大神殿`当作`冒险者`死后复活的实际地点来描述。
- 当时角色虽然还不能解释其原理，但已经默认“死后会被送回秋叶原的大神殿”。
- 关联出处：
  - PDF页码：119
  - 章节标题：`005`
  - 关键词：`大神殿`、`復活`、`アキバ`
  - 日文短摘录：`アキバの街の大神殿で復活`

## 大地人社会也把它视作冒险者不死性的象征
- 从`大地人`侧的理解来看，`冒险者`之所以可畏，一个原因就是他们致命伤后仍不会完全消灭，而会返回`大神殿`复活。
- 这说明至少在`大地人`社会认知中，`大神殿`已经是“`冒险者`不死性”的公共象征之一，而不只是玩家内部便利设施。
- 关联出处：
  - PDF页码：305, 559
  - 章节标题：`012`；`020`
  - 关键词：`大神殿`、`復活`、`無限の魂`
  - 日文短摘录：`大神殿へと戻って復活` / `無限の魂`

## 还会反过来影响城市内行动选择
- 到秋叶原自治整备期，文中又补出一个很实际的功能面：因为`大神殿`位于秋叶原中央，部分玩家会把“自杀后在大神殿复活”当成一种带代价的快速脱离与移动手段。
- 这说明`大神殿`不仅是死亡后的被动去处，也会反过来影响城市内的战术与行动选择。
- 关联出处：
  - PDF页码：505
  - 章节标题：`017`
  - 关键词：`大神殿`、`街の中央部`、`実質的な移動`
  - 日文短摘录：`大神殿はアキバの街の中央部にある` / `実質的な移動に等しい`

## 后续理论把它定位成重构目的地
- 到`魂魄理论`出现后，`大神殿`的地位进一步明确：对`冒险者`来说，`落魄`完成后，肉体与装备会粒子化并被转送到`大神殿`，在那里进行自我苏生式的重构。
- 也就是说，前期“在大神殿复活”这一观察，在后期被升级成了完整机制的一部分。
- 关联出处：
  - PDF页码：736
  - 章节标题：`025`
  - 关键词：`大神殿`、`転送`、`自己蘇生`
  - 日文短摘录：`大神殿に転送` / `自己蘇生`

## 规则边界说明并非所有场景都回到固定大神殿
- 后文补出了一个重要边界：某些大规模战斗区域中，复活点并不会回到玩家城镇的`大圣堂 / 大神殿`，而是改在区域入口。
- 这说明“复活目的地”存在受内容规则调整的场景型变体。
- 关联出处：
  - PDF页码：1585-1586
  - 章节标题：`056`
  - 关键词：`大聖堂`、`ゾーンの入り口`
  - 日文短摘录：`大聖堂まで移動するなどということはない` / `ゾーンの入り口で復活`

## 还有道具级的移动变体
- 另有道具级变体`北风之移动神殿`，其描述已经接近“可临时替代复活地点”的设施。
- 这进一步说明“复活设施”这一功能并不绝对绑定单一固定建筑，但`大神殿`仍是最核心、最标准的基准形态。
- 关联出处：
  - PDF页码：1851
  - 章节标题：`066`
  - 关键词：`移動神殿`、`復活地点の代替`
  - 日文短摘录：`復活地点の代替`

## 阶段变化
- 观察阶段：先被认知为死后会出现的标准复活地点。
- 公共认知阶段：很快又被`大地人`社会视作`冒险者`不死性的象征。
- 理论阶段：后文把它解释为`魂魄理论`下的重构目的地。
- 变体阶段：再补出区域入口复活与移动神殿等特殊实现。
- 这几层必须同时保留，不能把后期变体反写成“大神殿从来不是标准设施”。

## 对中原服的参考价值
- 对中原服最值得直接继承的，是这几个底层点：
  - 玩家城市中存在标准复活设施
  - 该设施既是机制性地点，也是政治与心理上的重要地标
  - 某些特殊场景或特殊道具，可能临时改写复活点逻辑
- 中原服若要本地化，可以改变城市名、宗教风格与建筑外观，但不应轻易破坏“标准复活设施 + 特殊例外变体”这套结构。

## 当前未知或仍待补证的部分
- 不同城市是否都存在同规格`大神殿`，当前仍待各服务器与各城市词条继续补证。
- `大神殿`与`大圣堂`在不同章节中的用词差异，是否为同物异称还是有更严格区分，当前不能一概写死。
- 除区域入口与`北风之移动神殿`外，是否还有更多复活设施变体，当前未知。
