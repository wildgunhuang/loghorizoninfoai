# SYS-023 师徒系统（師範システム）

- 分类：系统与世界机制 / 等级同步
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`师徒系统`是《幻境神话》为了缓和新老玩家等级差而设置的同步机制。当前材料已能确认：高等级`冒险者`可以借此把自己的等级降到较低等级成员附近，以便进行带练、护送与协同行动；但到更后期，城惠又明确指出，`典灾`已经学会逆用这一系统，把它变成了敌方“口传式攻击”的基础。

## 作为弥合等级差的原生系统
- 文中明确说，`师徒系统`是为缓解老玩家与新玩家之间的等级差、让不同层级玩家能够一起游玩而搭载的系统之一。
- 这说明它从游戏时代起就不是临时技巧，而是正式的底层辅助机制。
- 关联出处：
  - PDF页码：2005
  - 章节标题：`072`
  - 关键词：`師範システム`、`格差`、`一緒に遊ぶ`
  - 日文短摘录：`『師範システム』` / `一緒に遊ぶことを妨げる要素`

## 高等级者可主动向低等级同步
- 文中多处都说明：高等级`冒险者`可以借助`师徒系统`把自己的等级压到低等级成员附近。
- 这种机制的作用不是单纯“谦让”，而是为了让仇恨、承伤、治疗和输出节奏重新回到适合新人的战斗平衡线上。
- 在训练和远征场景里，这一点尤其重要。
- 关联出处：
  - PDF页码：312-313
  - 章节标题：`011`
  - 关键词：`師範システム`、`レベルにあわせる`、`戦闘連携`
  - 日文短摘录：`「師範システム」を使ってた` / `同じようなレベルにあわせれば`

## 作为带练与护送的常规手段
- `记录的地平线`后来的日常运作里，也明确把`师徒系统`当成常规做法来使用。
- 高等级成员不以降级陪同为苦，反而依靠这套机制长期带领低等级成员活动。
- 这说明它在异世界化后没有失效，反而继续作为组织化育成工具存在。
- 关联出处：
  - PDF页码：1125
  - 章节标题：`038`
  - 关键词：`師範システム`、`低レベルにあわせる`
  - 日文短摘录：`師範システムを用いて` / `低レベルにあわせる`

## 不只用于教学，也用于长途护送
- 后文还显示，`成惠2`与`晓`都会在护送、远征、引率时主动使用`师徒系统`把等级降下来。
- 这说明它不仅适用于“练级课堂”，也适用于需要与低等级成员同步风险和同步移动节奏的实战旅程。
- 关联出处：
  - PDF页码：1841, 3107, 3119
  - 章节标题：`065`；`111`
  - 关键词：`師範システム`、`レベルを落とした`、`レベル同期`
  - 日文短摘录：`師範システムでレベルを落とした` / `レベル同期`

## 后期变化：被典灾逆用
- 到后期，城惠明确指出：那次大规模等级下降并不是普通减益状态，而是`典灾`逆用了`师徒系统`。
- 他进一步把这种手段解释为“属于敌方自己的口传式攻击”。
- 这是`师徒系统`最重要的阶段变化：它从玩家育成机制，升级成了敌我双方都可围绕其开发战术的系统接口。
- 关联出处：
  - PDF页码：3081
  - 章节标题：`111`
  - 关键词：`師範システム`、`逆用`、`口伝`、`レベルが下がっている`
  - 日文短摘录：`〈師範システム〉が逆用されている` / `彼らなりの〈口伝〉の攻撃`

## 这一效果并不表现为普通坏状态
- 同一段里，城惠还先做了一个排除判断：他们当前的等级下降并没有作为普通`坏状态`显示在状态栏里。
- 也就是说，这种现象不是常规可解咒的 `debuff`，而更像是直接调用了系统层面的等级同步接口。
- 这进一步提高了“敌方已学会逆用游戏根机制”的可信度。
- 关联出处：
  - PDF页码：3080-3081
  - 章节标题：`111`
  - 关键词：`バッドステータスじゃない`、`アイコン表記`
  - 日文短摘录：`バッドステータスじゃない` / `そんなアイコン表記はない`

## 阶段变化
- 游戏时代，`师徒系统`首先被明确为缓和新老玩家等级差、让不同层级成员能够一起活动的正式辅助机制。
- 大灾害后的前中期，它继续以公会育成、护送与陪练工具的形式稳定存在，高等级者会主动把自己同步到低等级成员附近。
- 到后期，文本已经明确写出`典灾`开始逆用这一系统接口，使其从玩家侧育成机制转变成可被敌对势力利用的战术性系统漏洞。

## 对中原服的参考价值
- `师徒系统`非常适合直接继承到中原服。
- 它可以自然支撑：
  - 高等级玩家带新人
  - 公会内部远征护送
  - 等级跨度过大的队伍临时同步
  - 后期敌对势力对系统接口的反向利用
