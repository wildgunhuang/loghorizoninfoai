# SYS-018 回城魔法（帰還呪文）

- 分类：战斗系统 / 交通与返回机制
- 适用范围：日本服实现形式，底层规则可能共通
- 对中原服的可用性：可改造沿用
- 日本服依赖度：中

## 摘要
`回城魔法`是所有玩家都可使用的返回型瞬间移动呪文，会把施术者送回“最后进入过的玩家主都市”。其底层机制很可能是全服共通，但“主都市”的具体名单与地理分布，在创作中应按各服务器本地结构改写。大灾害后，这一机制的限制被充分暴露出来，使原本理所当然的“总能回家”变成高风险假设。

## 机制性质
- 当前最稳妥的写法是：`回城魔法`属于通用返回机制，而不是自由选点传送。
- 它的核心不在“能传送”，而在“只能返回最后绑定过的主都市，且绑定会自动覆盖”。
- 因此它天然带有交通便利与撤退风险并存的双重性质。

## 基本规则
- `回城魔法`可将玩家瞬间送回最后立足过的主都市。
- 它并不是战斗中随手脱离的手段：咏唱需要数分钟，而且二十四小时只能使用一次。
- 游戏时代中，它更接近“今天的冒险结束后，回城收尾”的常用机能。
- 关联出处：
  - PDF页码：176-177
  - 章节标题：`008`
  - 关键词：`帰還呪文`、`瞬間移動呪文`、`最後に立ち寄った`、`２４時間に一回`
  - 日文短摘录：`全ての〈エルダー・テイル〉プレイヤーが使用できる` / `最後に立ち寄った五大都市へと一瞬で戻れる` / `２４時間に一回`

## 返回点会在进入主都市时自动覆盖
- 大灾害后的限制之所以明显暴露，关键就在于返回点并非由玩家自由保留，而会在进入主都市时自动改写。
- 一旦玩家被困在别的大都市，它只能把人送回那个被困都市，而不能送回原来的据点。
- 关联出处：
  - PDF页码：176-177
  - 章节标题：`008`
  - 关键词：`自動的に上書き`、`ススキノに戻れるだけ`、`この街に戻ることは出来ません`
  - 日文短摘录：`五大都市に入った時点で、自動的に上書き` / `ススキノに戻れるだけ` / `この街に戻ることは出来ません`

## 大灾害后它不再等于稳定回撤手段
- 在传送门失效的前提下，`回城魔法`的自动覆盖机制使原本被视作理所当然的“总能回家”失效。
- 这让长距离远征、救援和跨城行动的风险大幅上升，因为“撤离”与“返回己方据点”不再是同义词。

## 阶段变化
- 游戏阶段：更像日常结束冒险的便利返回功能。
- 危机阶段：大灾害后，在跨城移动受限背景下暴露出绑定覆盖与无法真正回撤的问题。
- 这两层要同时保留，不能把“危机中的回撤失败”误写成它从一开始就毫无实用价值。

## 对中原服的继承方式
- 若中原服沿用原作体系，最稳的写法不是照搬“日本五大都市”，而是继承：
  - 存在若干主都市
  - 回城点会绑定到最近进入的主都市
  - 绑定会自动覆盖
  - 灾后长途移动会因这一机制而更难回撤
- 也就是说，中原服应继承机制，不直接继承日本城市名单。

## 当前未知或仍待补证的部分
- 各服务器“主都市”的数量、判定边界与具体名录，当前应分别在服务器词条中处理。
- 是否存在能绕开自动覆盖、固定回城点或替代`回城魔法`的手段，当前尚未在本页汇总。
