# EVT-001 大灾害（大災害）

- 分类：事件与现象 / 世界级剧变
- 适用范围：全服共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`大灾害`是玩家被卷入异世界后，用来指称这一剧变及其后续影响的核心事件名。它不仅意味着玩家转移本身，也意味着原有游戏逻辑与现实生活逻辑发生了大规模错位。到后期材料里，`大灾害`又不应再只被理解成“一次已经结束的事故”，而更像一个仍在持续改造世界与社会的长期过程。

## 事件性质
- 现阶段最稳妥的理解方式，是把`大灾害`视作玩家社会的核心命名。
- 它首先是玩家和居民共同面对的现实剧变；至于它与`一致`、`森罗变化`等上位解释如何套合，应分层整理，而不是混写成一个单义术语。

## 命名层与解释层应分开处理
- `大灾害`首先是玩家社会对剧变的现实命名，因此本页优先记录“人们经历了什么、世界发生了哪些可观察变化”。
- `一致`更偏`航界种`侧对跨世界接触的上位解释，`森罗变化`则更偏作品内历史与宇宙论框架。
- 三者高度相关，但不应互相覆盖或替代。

## 初期直接影响是信息网络断裂
- 玩家失去了现实世界中的 `Web` 攻略站支援，必须重新通过口头交流、手绘接续图、实地调查等方式重建信息网络。
- 原本可依赖的外部知识库消失后，区域连接、资源产地、危险地带与新旧地图信息迅速升值。
- 市场中的上架物品开始被大量撤回，因为玩家意识到旧物品的用途与价值可能发生变化。
- 多数玩家在剧变后的数日内还没有建立稳定秩序，只能依靠公会、熟人网络或临时合作关系求生。
- 关联出处：
  - PDF页码：18-19
  - 章节标题：`001 異世界の始まり（上）`
  - 关键词：`Ｗｅｂ`、`攻略サイト`、`接続`
  - 日文短摘录：`もうＷｅｂが見れません` / `攻略サイト`

## 初期直接影响也包括生活系统扭曲
- 大灾害后的世界表面上沿用了游戏规则，但现实生活层面的许多细节却产生严重错位。
- 典型例子是料理系统失真：系统式制作出的成品料理普遍失味，而原材料本身仍有味道。
- 这种错位说明：此世界并不是把原游戏“完全照搬成现实”，而是一个混合了游戏规则与现实肉体体验的扭曲世界。

## 料理突破改变了灾后食物经济
- 大灾害后的早期常识是：只要通过`作成メニュー`生成食物，无论菜色名目多么丰富，成品几乎都会退化成同一种乏味口感。
- 这一点使得食物在灾后很快被重新理解为“只为止饥的廉价消耗品”，市场判断标准几乎只剩价格。
- 关联出处：
  - PDF页码：384-386
  - 章节标题：`014`
  - 关键词：`湿気た煎餅`、`値段のみ`、`値下げ合戦`、`作成メニュー`
  - 日文短摘录：`全て同じ味` / `湿気た煎餅` / `選択基準は値段のみ` / `凄まじいまでの値下げ合戦`

- 但后文明确出现了突破口：`厨师`若不通过`作成メニュー`，而是以现实中的正常步骤直接切、煮、烤食材，就能做出保留素材味道的料理。
- 文中同时说明，即使采用现实手法，仍然需要足够的`调理スキル`；不是`厨师`、或`调理スキル`过低时，仍可能只会做出失败品。
- 因此，这里应理解为“灾后料理规则被重新发现”，而不是“料理系统完全失效”。
- 关联出处：
  - PDF页码：290-291
  - 章节标题：`011`
  - 关键词：`料理人`、`調理スキル`、`メニューを開かない`、`普通の手順`
  - 日文短摘录：`メニューを開かないで` / `普通の手順で料理をする` / `素材の味を生かした料理` / `調理スキルは必要`

## PK与治安条件发生逆转
- 大灾害使原本在游戏时代不太流行的`PK`条件发生了逆转。
- 脑内菜单里不再存在原来的小地图，玩家也不会自动作出无意识回避，原来由运营方事后裁定的骚扰处罚也随之失效。
- 与此同时，击杀其他玩家后可以夺取其全部现款与约半数可交易物品，这使`PK`从高风险低收益行为变成了高诱因行为。
- 详细规则与阶段变化见 [SYS-017_PK_ＰＫ_プレイヤーキル.md](SYS-017_PK_ＰＫ_プレイヤーキル.md)。

  - 关键词：`ＰＫ`、`ミニマップ`、`ハラスメント`、`現金全て`、`約半分`
  - 日文短摘录：`ＰＫはリスクが大きい` / `ミニマップは脳内メニュー内部にも存在しない` / `現金全てと、アイテムの約半分`

## 后续敌对存在层也发生变化
- 后期又出现了另一条重要变化：大灾害后，世界中开始逐步出现被称为`典灾`的异常新怪物。
- 当前文本已经明确两点：
  - 它们不是单一地点的孤例，而是跨多个城市方向被确认的情报对象
  - 不能简单把所有灾后新怪物都视为`典灾`，但`典灾`确实属于灾后新增威胁的一部分
- 这说明大灾害带来的改变，并不只限于玩家、料理、经济或政治秩序，也包括敌对存在本身的演化。
- 详细内容见 [MON-001_典灾_典災.md](MON-001_典灾_典災.md)。

  - 日文短摘录：`そう呼ばれる新しいモンスター` / `時間が経過するほどに増えてきている`

## 后期不应再把它视作一次性的瞬时事故
- 到更后面的讨论里，`大灾害`的理解又向前推进了一步。
- 亚因斯明确整理出两层判断：
  - 它也许并不是“当时发生完就结束”的单次爆发，而是从某个时点开始，随后持续缓慢改造世界的现象
  - 它带来的并不只是`冒险者`醒来这一件事，而是整个`赛尔迪希亚`的社会、制度、经济、文化与势力关系都被持续卷入变化之中
- 这条判断非常关键，因为它把`大灾害`从“历史事件”升级成了“仍在进行中的长期过程”。
- 关联出处：
  - PDF页码：3244-3245
  - 章节标题：`127 1.05`
  - 关键词：`大災害は終わっていない`、`継続`、`変質`、`侵食`
  - 日文短摘录：`〈大災害〉は終わっていない` / `継続しつつある` / `世界を変質させていく` / `侵食`

## 持续过程的证据包括空间异常与社会重组
- 同一段后文还给出了这种“持续性”判断的依据。
- 其中一类是更偏世界机制的征候，例如`圆桌会议`一侧已经掌握到“两点间距离会逐渐拉开”的异常报告，以及`典灾`袭击等越来越难被解释成局部偶发事件的现象。
- 另一类则是社会层面的重组：`冒险者`真正定居、技术革新、制度调用、讨伐停滞、前所未见的怪物被击倒，以及`大地人`社会秩序因此发生的激烈变化。
- 这说明“大灾害仍在继续”的判断并不是空洞感想，而是由世界规则异常与社会变动两边共同支撑起来的。
- 关联出处：

  - 关键词：`二点間の距離が徐々に開く`、`典災`、`技術革新`、`制度`、`激甚な変化`
  - 日文短摘录：`二点間の距離が徐々に開く` / `〈典災〉による襲撃` / `技術革新` / `激甚なる変化`

## 阶段变化
- 命名阶段：先被作为玩家社会对突发剧变的总事件名使用。
- 错位阶段：很快补出信息网络断裂、生活系统扭曲、市场与治安条件逆转等现实后果。
- 演化阶段：后文再补出`典灾`等敌对存在变化与世界规则异常。
- 持续阶段：更后面明确提出它不是一次已结束事故，而是仍在持续侵蚀世界的过程。
- 这几层都应并列保留，不能把后期“持续侵蚀论”倒灌成开篇所有人已知的定论。

## 对中原服的参考价值
- `大灾害`是中原服必须直接继承的总事件名。
- 你可以直接继承的，不只是“玩家被卷进异世界”这一点，还包括：
  - 信息网络断裂
  - 生活系统与现实体验错位
  - 社会秩序、治安与物资逻辑重组
  - 后期持续性的世界侵蚀和制度改变

## 当前未知或仍待补证的部分
- `大灾害`与`一致`、`第三次森罗变化`之间的关系，当前仍应保留为“高度对应但不同解释框架”。
- 它是否存在单一触发因、是否可逆、以及是否还在以同样机制持续推进，现阶段都不能写死。
- 它与`开拓智域`之间的因果关系，当前也仍只到“强关联、强怀疑”的程度。
