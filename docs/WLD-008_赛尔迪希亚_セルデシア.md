# WLD-008 赛尔迪希亚（セルデシア）

- 分类：世界观 / 世界本体
- 适用范围：全服共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`赛尔迪希亚`是《记录的地平线》异世界的正式世界名。到后期文本里，它已经不再只是“像《幻境神话》的异世界”这种模糊说法，而被明确指认为一个可容纳`冒险者`、`大地人`与其他异界知性体受肉的世界；但它是否应被理解为普通行星、`亜世界`、实验场或其他更高层结构，在当前阅读范围内仍有不同层级的说法。

## 世界名性质
- 现阶段最稳妥的理解方式，是把`赛尔迪希亚`视为作品内正式世界名。
- 但它的“世界层级定位”不能只保留一种说法，而应并记不同来源提出的解释层次。

## 后期明确出现的世界名
- 后期文本里，`成惠2`已经明确把此处称为`赛尔迪希亚世界`，并把`冒险者`解释为“从不同世界受肉到赛尔迪希亚的存在”。
- 她同时又说，自己一方也是“从不同世界受肉到赛尔迪希亚的存在”。
- 这说明`赛尔迪希亚`已经不是角色口中的模糊代称，而是作品内部明确使用的世界名。
- 关联出处：
  - PDF页码：2217
  - 章节标题：`084 ノウアスフィアの開墾`
  - 关键词：`セルデシア世界`、`受肉`、`異なる世界`
  - 日文短摘录：`このセルデシア世界へと受肉した存在` / `異なる世界から`

## 在高层解释里又被称作亚世界
- 到更后面的讨论里，`成惠2`还把这里称作`亜世界セルデシア`。
- 这一表述比“异世界”更进一步，暗示它在更大的多世界结构里可能只是一个层级化、分支化、或被封装的世界。
- 但就当前材料而言，这仍主要是`成惠2`与城惠所接触到的高层解释，不宜当成已经完全证实的客观物理定义。
- 关联出处：
  - PDF页码：2295
  - 章节标题：`087`
  - 关键词：`亜世界セルデシア`、`サンドボックス`
  - 日文短摘录：`亜世界セルデシア` / `特別なサンドボックス`

## 不是只对冒险者有利的世界
- 城惠在后期推断里强调：`赛尔迪希亚`并不是只站在`冒险者`一边的世界。
- 对他而言，这世界之所以让`冒险者`能活下来，是因为它既是异世界，又保留了《幻境神话》的“游戏根”可供利用；但同样的根也可能被敌方利用。
- 这段内容属于后期认知深化，不应回写成开篇就已确认的绝对规则。
- 判定：`主角后期推断`
- 关联出处：
  - PDF页码：3079-3080
  - 章节标题：`111`
  - 关键词：`セルデシア`、`味方じゃない`、`公平`
  - 日文短摘录：`セルデシアは僕らだけの味方じゃない` / `この異世界はひどく公平だ`

## 是多类存在共同受肉与交错的场域
- 结合后期已知材料，`赛尔迪希亚`并非只承载`冒险者`与`大地人`。
- 至少`航界种`也会在这里受肉，而`典灾`等异常敌对存在则进一步说明，这个世界已成为多类存在共同活动、对抗与误接的场域。
- 这使`赛尔迪希亚`不再只是“玩家进入的异世界”，而是高层多方势力交错的共同舞台。
- 关联出处：
  - PDF页码：2217-2218、3077-3081
  - 章节标题：`084 ノウアスフィアの開墾` / `111`
  - 关键词：`受肉`、`航界種`、`典災`
  - 日文短摘录：`受肉した存在` / `〈航界種〉` / `〈典災〉`

## 阶段变化
- 在后期正式命名浮出之前，故事多以“异世界”或《幻境神话》现实化后的世界来间接描写此处。
- 到`成惠2`出场并解释相关结构时，`赛尔迪希亚`已经被明确提升为正式世界名，并被说明为多类存在共同受肉的场域。
- 再往后，文本又进一步提出`亜世界`、`沙盒`与“并不只偏袒冒险者”的公平性解释，使这个词条从地点名称升级成高层宇宙论入口。

## 对中原服的参考价值
- 对中原服创作而言，`赛尔迪希亚`这一世界名应直接沿用，不宜另造新的世界总体名称。
- 可直接继承的底层方向包括：
  - 同一服务器故事仍发生在同一个`赛尔迪希亚`
  - 不同服务器只是此世界中的不同区域、文化圈与玩家群落
  - 后期若要写到更深层宇宙论，可以把`亜世界`、`沙盒`、`公平性`等解释留作逐步揭露

## 当前未知或仍待补证的部分
- `赛尔迪希亚`究竟更接近完整宇宙中的一颗星球、被人为切出的亜世界、实验场还是沙盒场域，当前仍无单一结论。
- 它与《幻境神话》游戏数据结构之间的关系是否等同于“游戏变成现实”，现阶段也不能粗暴写死。
- `一致`、`森罗变化`、`大灾害` 与 `赛尔迪希亚` 的最终对应关系，仍需后文继续补证。
