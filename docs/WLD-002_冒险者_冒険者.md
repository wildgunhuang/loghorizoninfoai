# WLD-002 冒险者（冒険者）

- 分类：世界观 / 身份体系
- 适用范围：全服共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
玩家卷入异世界后，以游戏角色的肉体存在，并被称为`冒险者`。开篇即可确认他们仍保有高战斗能力与死后复苏能力；但随着剧情推进，`冒险者`这一身份不再只是“玩家角色”的同义词，而逐渐显露出真实肉体、生理需求、外貌偏移与长期死亡代价等现实层后果。

## 身份性质
- 当前最稳妥的写法是：`冒险者`是异世界中“由原玩家承担的高能力存在”这一身份总类。
- 这一身份同时具有两层特征：
  - 继承自旧游戏系统的职业、等级与战斗能力
  - 大灾害后逐渐现实化的身体、生活与社会属性
- 因此它既不是单纯“现实中的本人”，也不能再只当作“游戏角色数据”理解。

## 开篇已确认其以角色肉体存在
- 作品开头即明确指出，玩家如今不是隔着显示器操作角色，而是以“作为角色的肉体”在异世界活动。
- 简介还直接把`冒险者`描述为拥有高战斗能力与苏生能力的英雄性存在。
- 关联出处：
  - PDF页码：2, 8
  - 章节标题：作品简介；`001 異世界の始まり（上）`
  - 关键词：`冒険者`、`蘇生能力`、`キャラクターとしての肉体`
  - 日文短摘录：`英雄的存在〈冒険者〉` / `実在の身体を備えたプレイヤーとして`

## 作为生活主体而不再只是登入中的玩家
- 大灾害后，`冒险者`不能再像游戏时代那样通过登出回避身体管理问题。
- 他们必须处理睡眠、驻留、安全与日常生活空间等问题，现实中的“居住”开始成为必要条件。
- 关联出处：
  - PDF页码：72-73
  - 章节标题：`004`
  - 关键词：`宿屋`、`ゾーン貸し出し`、`実体としての身体`、`警備`
  - 日文短摘录：`短期間のゾーン貸し出し` / `寝ている間にも実体としての身体が存在` / `それなりの警備がしかれた場所`
- 推断：这使“生存”从单纯战斗问题扩展为后勤、卫生与日常生活问题。

## 生理现象已现实化
- 初期确认：异世界中的`冒险者`会感到空腹，需要进食。
- 初期确认：异世界中的`冒险者`会困，需要睡眠；高等级身体的高体力并不能取消睡眠欲。
- 初期确认：异世界中的`冒险者`也存在排泄需求。
- 同期补充：高等级`冒险者`的身体依然非常强韧，战斗和移动后的恢复速度很快，但这不等于完全脱离现实生理规律。
- 关联出处：
  - PDF页码：69, 72-73
  - 章节标题：`004`
  - 关键词：`腹が減る`、`睡眠`、`排泄`
  - 日文短摘录：`シロエ達は腹が減る` / `睡眠が必要` / `排泄もある`

## 死亡后通常会复苏
- 开篇后不久即通过实例确认，`冒险者`死亡后通常会在神殿侧复苏，而非直接回到地球。
- 后文又进一步说明，这一规则并非无代价，也并非永远没有例外。
- 此处只保留身份层结论，详细机制见相关词条。

## 外貌会带入现实本人的投影
- 初期状态：玩家仍以游戏角色外形为基础，但角色面貌开始体现现实本人的特征，例如城惠、直继、玛莉艾儿都被描述为“像本人”。
- 同期补充：这种“像本人”的判断并不限于主角组个别案例，而是可被观察到的普遍倾向。
- 关联出处：
  - PDF页码：24, 47-49
  - 章节标题：`002`；`003`
  - 关键词：`面影`、`元世界`、`投影`
  - 日文短摘录：`直継っぽいぞ？` / `元世界の容貌の特徴` / `こちらの世界に投影`

## 后续出现体型与性别错位问题
- 已确认个案：日本服角色`晓`在大灾害后，因原游戏角色是高个男性刺客、现实本人却是矮个女性，出现了步行不便、开口暴露性别、如厕困难等实际生活问题。
- 缓解手段：`晓`通过稀有事件道具`外形重设药水`重设外观后，身体尺寸与现实本人更接近，行动困难明显缓解。
- 目前能稳妥确认的是：模型与现实本人差距过大时，`冒险者`会遭遇现实层面的生活与社交障碍。
- 关联出处：
  - PDF页码：94-101
  - 章节标题：`005`
  - 关键词：`外観再決定ポーション`、`女性`、`身長差`、`歩くことも不自由`、`トイレ`
  - 日文短摘录：`プレイヤーが女性` / `歩くことも不自由` / `トイレが困る` / `本当の体型とリンク`

## 身份边界并非绝对不可改写
- 后文出现了极重要的例外：至少存在通过高等级契约书与魔法素材，把一名`大地人`转变为`冒险者`的个案。
- 因此现阶段最稳妥的结论不是“身份可自由转换”，而是：
  - `冒险者 / 大地人`边界并非绝对封死
  - 边界改写需要极端条件，并伴随未知变质与高风险
- 关联出处：
  - PDF页码：1014-1017
  - 章节标题：`034`
  - 关键词：`契約書`、`大地人`、`冒険者の身分`
  - 日文短摘录：`これには〈冒険者〉の身分が含まれる` / `ルンデルハウスを〈冒険者〉にする`

## 后期又出现死亡代价与例外
- 随剧情推进，`冒险者`的“不死”被进一步解释为依赖`魂魄理论`与`大神殿`的重构机制。
- 后文又补出经验值损失、记忆劣化风险，以及极少数可切断`大神殿`路线的特殊例外。
- 因此“`冒险者`会复活”必须与“有代价、且可能有例外”一并理解。

## 阶段变化
- 开篇阶段：主要确认“玩家以角色肉体存在，具备高战斗力与复苏能力”。
- 生活阶段：很快补出生理需求、住宿需求与身体管理问题。
- 偏移阶段：再补出外貌向现实本人投影，以及体型错位造成的现实麻烦。
- 高层阶段：后文又补出身份边界可被改写，以及复苏存在代价与少数例外。
- 这几层必须并列保留，不能把后期高层解释倒灌成开篇已知事实。

## 对中原服的参考价值
- 对中原服写作而言，`冒险者`最重要的可继承点不是某个日本服人物，而是这套完整身份框架：
  - 仍有游戏侧能力
  - 身体已经现实化
  - 社会生活与后勤问题真实存在
  - 外貌与身份边界可能发生偏移
  - 死亡复苏并非零代价
- 这决定了中原服里的公会、城市、远征、战争与阶层关系应如何成立。

## 当前未知或仍待补证的部分
- 外貌向现实本人投影，究竟是普遍长期收束，还是仅在部分案例里明显，当前仍不能写死。
- 身份边界改写是否还能重复实现、能否逆转，当前材料不足。
- `冒险者`在不同服务器中的法律地位、与本地住民的制度关系，需分别在各服务器词条中追踪，不宜在本页一概而论。
