# ORG-008 红王派（紅王閥）

- 分类：组织与制度 / 中原服政治派系
- 适用范围：特定服务器特有
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`红王派`是中原线当前已从“并列派系名”推进到“实际远征威胁”的大型势力。现有材料能够稳定确认的重点，不是它完整的内部制度，而是它在`草原之都 - 狼君山`战线上的现实行动：已派出以`红玉麒麟`为代表的大规模部队，目标并非单纯攻城，而更像是通过`封禅大典`改写周边支配权。

## 派系性质
- 目前最稳妥的处理方式，是把`红王派`视为中原服王号派系中的另一支大型竞争势力。
- 它并不是和`苍王派`、`白王`完全同义的称呼，而是被单独拿来作为对照对象。
- 关联出处：
  - PDF页码：2481
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`白王`、`紅王`
  - 日文短摘录：`白王や紅王じゃ`

## 已由红玉麒麟率队实质介入草原之都周边局势
- 朱桓在野营途中收到留守城市的追加报告时，首先确认的一点就是：这次来的不是`歌剑团`，而是`红玉麒麟`。
- 紧接着正文就把这条情报和“`红王派`的家伙们真正来了”连在一起。
- 这说明`红玉麒麟`当前至少可视为`红王派`在这一战线上的代表性干部或旗帜人物。
- 关联出处：
  - PDF页码：2570
  - 章节标题：`099 Chapter 3.01`
  - 关键词：`紅玉麒麟`、`紅王閥`
  - 日文短摘录：`〈紅玉麒麟〉だ` / `〈紅王閥〉の連中が、本格的に来やがった`

## 具备足以压垮边境城市的大规模兵力
- 同一段情报里，朱桓一侧直接给出量级判断：对方大约有`五千`人。
- 文中又立刻补上现实条件：`草原之都`没有像样的防壁。
- 这说明`红王派`当前给边境带来的威胁，不是抽象政治压力，而是足以直接摧毁地方防御秩序的现实军力。
- 关联出处：
  - PDF页码：2570
  - 章节标题：`099 Chapter 3.01`
  - 关键词：`五千`、`防壁`
  - 日文短摘录：`あいつら五千はいるはずだ` / `〈草原の都〉にはろくな防壁がない`

## 当前目标并不只是攻城，而是封禅祭坛
- 朱桓随后明确提出，对方真正的目标恐怕不是单纯夺取`草原之都`，而是更大的东西，即`封禅大典`。
- 他把这层推论进一步落到具体地点，指出`狼君山`很可能正是对应的山，而`红王派`盯上的就是那里的祭坛。
- 这说明`红王派`此时的作战逻辑更接近“争夺制度性支配权”，而不只是边境掠夺。
- 关联出处：
  - PDF页码：2570-2572
  - 章节标题：`099 Chapter 3.01`
  - 关键词：`封禅の儀`、`狼君山`、`祭壇`
  - 日文短摘录：`目標は〈草原の都〉じゃない` / `〈封禅の儀〉ってやつ` / `紅王閥はその祭壇に目をつけた`

## 已与中原本地神话地景和战争制度直接纠缠
- `红王派`之所以值得单列，不只是因为兵多，而是因为它已经和`狼君山`、`封禅大典`、山岳祭坛这些中原本地化高价值要素直接相连。
- 因而它代表的不是普通野战公会，而是会主动追逐制度枢纽与神话地景的大势力。
- 关联出处：
  - PDF页码：2570-2572
  - 章节标题：`099 Chapter 3.01`
  - 关键词：`封禅の儀`、`祭壇`
  - 日文短摘录：`〈封禅の儀〉` / `その祭壇に目をつけた`

## 先遣部队一度被更高阶敌对存在全灭
- 斥候后续回报时说得非常直白：`红王派`比朱桓一行更早野营，但营地已经空了。
- 接着出现的是“雷鸣、虎爪、白光”，以及“一个不剩全被送去`大神殿`”的描述。
- 这说明`红王派`虽是地方主要人类威胁之一，却并不是压倒一切的最终力量；在`狼君山`一线，它已先被更高阶敌对存在打穿过一次。
- 关联出处：
  - PDF页码：2574
  - 章节标题：`099 Chapter 3.03`
  - 关键词：`紅王閥`、`大規模戦闘`、`大神殿`
  - 日文短摘录：`紅王閥の連中だ` / `大規模戦闘エネミーだった` / `ひとり残らず大神殿送り`

## 与苍王派、歌剑团不是同一股势力
- 前期草原之都面对的主要施压者是`歌剑团`，其政治背景更多连向`苍王派`。
- 但到这一阶段，留守报告明确说“不是`歌剑团`，而是`红玉麒麟`”。
- 因而当前最稳妥的处理方式，是把`红王派`视为中原大型王号派系中的另一支独立势力，而不是`歌剑团`的别名或下位称呼。
- 关联出处：
  - PDF页码：2481、2570
  - 章节标题：`096 Chapter 1.04` / `099 Chapter 3.01`
  - 关键词：`歌剣団`、`紅玉麒麟`
  - 日文短摘录：`〈歌剣団〉の使者` / `〈紅玉麒麟〉だ`

## 当前已知的活动边界
- 以现有材料看，`红王派`已经能向中原外缘投送大兵团，并瞄准可重写领属关系的高价值目标。
- 但其中心据点、长期治理方式、内部会议结构与常设人事层级，目前仍未被正文充分展开。
- 因而现阶段更适合把它写成“已实质出兵并争夺封禅祭坛的王号派系”，而不是已经完全还原的国家型组织。

## 阶段变化
- 并列阶段：先作为与`苍王派`、`白王`并列的王号派系名出现。
- 出兵阶段：后文补出`红玉麒麟`率大兵团实质介入`草原之都`周边局势。
- 制度争夺阶段：再进一步明确其目标并不只是攻城，而是争夺`封禅大典`与祭坛支配权。
- 这几层都应并列保留，不能只把`红王派`写成单纯“来攻城的敌军”。

## 对中原服的参考价值
- `红王派`对你最有价值的地方，在于它提供了另一种中原大型势力模板：
  - 不一定先靠外交逼附，而可能直接以大兵团切入
  - 会主动争夺`封禅大典`这类可改写支配权的制度核心
  - 会与地方仙境、山岳祭坛和异常怪物危机同时纠缠
