# CHR-004 花貂（花貂）

- 分类：人物 / 中原本地住民
- 适用范围：特定服务器特有
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`花貂`是当前阅读范围里最鲜明的中原本地小角色之一。现有材料能够稳定确认：她属于`貂人族`，并自称是`天吏`；长期活动在`白桃庙 / 狼君山`一带，为克拉斯提提供路线、地方常识与生活协助，同时也承担庙宇管理与接待杂务。她的价值不只是人物趣味，而是把中原仙境里的地方住民、官吏身份、变形能力与日常生活感具体化。

## 基本身份
- 克拉斯提在她与同伴周围生活时，直接把她们写成`貂人族`。
- 花貂后来又明确回答：自己“正因为如此才是`天吏`”。
- 因而现阶段最稳妥的写法，是把她视为“貂人族出身的天吏样本”。
- 关联出处：
  - PDF页码：2505、2528、2533
  - 章节标题：`097 Chapter 2.01` / `098 Chapter 2.05`
  - 关键词：`貂人族`、`花貂`、`天吏`
  - 日文短摘录：`貂人族の精霊たち` / `貂人族の狂喜乱舞` / `だから天吏なのです`

## 其实并不真正理解“天之官吏”是什么
- 花貂后来自我反省时，明确承认自己虽然一直说`貂人族`是“管理仙境的天之官吏”，但真要问“官吏是什么”，她其实回答不出来。
- 这让她的身份显得很有地方色彩：既在秩序里，又未必真正理解那套秩序的抽象定义。
- 关联出处：
  - PDF页码：2599
  - 章节标题：`101 Chapter 4.01`
  - 关键词：`天の官吏`、`知らない`
  - 日文短摘录：`天の官吏とはなんなのか` / `花貂は知らないのである`

## 是白桃庙与狼君山一带的地方向导
- 花貂会主动向克拉斯提说明：
  - `狼君山`不能随便下山
  - `天狼洞`有位阶与人数限制
  - 从山顶侧只能走最长洞窟
- 她还熟悉`草原之都`的市集时间。
- 这说明她在角色功能上承担着稳定的“地方向导”位置。
- 关联出处：
  - PDF页码：2505、2529
  - 章节标题：`097 Chapter 2.01` / `098 Chapter 2.05`
  - 关键词：`狼君山`、`天狼洞`、`草原の都`
  - 日文短摘录：`〈狼君山〉は降りれませんよ` / `朝と夕方、大通りで市がある`

## 长期承担白桃庙的管理与接待
- 花貂回忆里补出了她们的日常：扫庙、给又长又险的石阶洒水、迎接偶尔来参拜的`朱桓`等客人。
- 这说明她不是只在主角到来时临时充当向导，而是本来就在庙中承担常驻杂务与接待。
- 关联出处：
  - PDF页码：2600
  - 章节标题：`101 Chapter 4.01`
  - 关键词：`石階段`、`参拝客`、`朱桓`
  - 日文短摘录：`長く険しい石階段に水を打ち` / `朱桓ら参拝客を迎える`

## 具有可逆变形能力
- 克拉斯提带着她从崖边跳下时，正文明确写到花貂“变身解开”，变成了布堆里的小动物。
- 后面她又会在烟雾中恢复少女姿态。
- 这说明她至少具备“少女形态 / 小型动物形态”之间的可逆变化能力。
- 关联出处：
  - PDF页码：2530、2535
  - 章节标题：`098 Chapter 2.05`
  - 关键词：`変化がとけてしまった`、`少女の姿を取り戻し`
  - 日文短摘录：`変化がとけてしまった` / `少女の姿を取り戻した`

## 日常欲望非常强，尤其与甜食绑定
- 花貂对甜食、外食和草原之都市集上的好吃的反应非常强烈。
- 她的兴奋几乎贯穿相关段落，也因此让克拉斯提 repeatedly 以“食いしん坊オコジョ”之类的方式戏弄她。
- 这不仅是角色趣味，也说明中原仙境住民并不是纯粹仙气缥缈的背景板，而有很具体的日常欲望。
- 关联出处：
  - PDF页码：2528-2535
  - 章节标题：`098 Chapter 2.05`
  - 关键词：`甘い匂い`、`外食`、`食いしん坊オコジョ`
  - 日文短摘录：`甘い匂いがする` / `向こうについたら外食` / `食いしん坊オコジョ`

## 饮食朴素，并承认仙桃具有禁忌性
- 花貂回忆自己和姊妹平时的饮食时，说主食大多只是硬木果。
- 至于仙人的桃子，若官吏擅自偷吃则属于重罪。
- 这条很有价值，因为它把`白桃庙`的阶层差序、食物禁忌和“桃”的神圣性都写实了。
- 关联出处：
  - PDF页码：2600
  - 章节标题：`101 Chapter 4.01`
  - 关键词：`木の実`、`桃`、`重罪`
  - 日文短摘录：`食事は主に硬い木の実` / `桃を勝手に食べるのは` / `重罪`

## 理论上未必依赖进食，但仍会为料理排队
- 后文更进一步写到：`貂人族`本来就算不进食也能耐受，但仍会动用`人化`、`霊化`之术排队等克拉斯提的料理。
- 这不仅加深了花貂的“贪吃”形象，也说明她所在族群并非单纯按现实生理需求行动。
- 关联出处：
  - PDF页码：2601
  - 章节标题：`101 Chapter 4.01`
  - 关键词：`食事をしなくても`、`人化`、`霊化`
  - 日文短摘录：`食事をしなくても耐えられるはずの貂人族` / `人化や霊化の術`

## 会用本地神话语言理解异常存在
- 她一直把带着`魂冥咒`异常状态的克拉斯提视作`仙君`，并据此推断他应当能云游、下山、买来好吃的东西。
- 这条误认一方面属于角色喜剧，另一方面也很有设定价值，因为它显示出中原地方住民如何用自己的神话语言去解释异常存在。
- 关联出处：
  - PDF页码：2505-2507、2531-2532
  - 章节标题：`097 Chapter 2.01` / `098 Chapter 2.05`
  - 关键词：`仙君様`、`雲`
  - 日文短摘录：`仙君様` / `雲でひとっ飛び`

## 把狼君山视为通往昆仑并负责守护的圣地
- 到地下追踪段落时，花貂又补出一层地方常识：在她理解里，`狼君山`连接着`昆仑`，并负有守护职责。
- 这说明她虽然不懂高层制度，但对“地方圣地归属哪一个更高中心”有朴素而稳定的认知。
- 关联出处：
  - PDF页码：2602、2609
  - 章节标题：`101 Chapter 4.02` / `101 Chapter 4.03`
  - 关键词：`崑崙`、`護衛`、`聖地`
  - 日文短摘录：`崑崙を護衛する任務` / `聖地崑崙へとつながり`

## 后期成为需要被同伴拼死保护的对象
- 到`狼君山`战线全面失稳时，花貂已经不再只承担地方向导和生活陪侍功能，而成了需要被本地同伴拼死护住的对象。
- 相关段落里，`求闻`为了保护花貂而战死，身体化作虹色光泡般消散。
- 这条材料很重要，因为它把花貂从“轻盈可爱的小角色”推进到了真正承受战争后果的位置，也让`白桃庙`一侧的伤亡第一次变得具体。
- 关联出处：
  - PDF页码：2641-2643
  - 章节标题：`101 Chapter 4.08`
  - 关键词：`求聞`、`花貂`、`虹色`
  - 日文短摘录：`求聞` / `花貂` / `虹色`

## 阶段变化
- 初期登场时，`花貂`首先被写成`白桃庙 / 狼君山`一带的地方住民与生活向导。
- 随着与克拉斯提相处的段落增加，她又进一步显露出`貂人族`、`天吏`、变形能力和地方俗信等多重设定层。
- 到后段战线失稳时，她的角色功能从“地方说明者”推进到“必须被共同体保护的具体住民”，从而直接承受战争后果。

## 对中原服的参考价值
- `花貂`适合作为中原本地小型住民模板：
  - 兼具地方官吏身份与非人种族特征
  - 能把山岳仙境的生活感、地方俗信和日常欲望具体化
  - 适合承担向导、陪衬和地方说明者功能
  - 也能在大战时承受地方共同体的真实伤亡

## 当前未知或仍待补证的部分
- 现阶段仍不宜把花貂的动物形态种类写死得太细，正文里更稳的是保留其“可变形的小型兽类化”这一层。
- 她在`白桃庙`中的正式上下级位置、与其他`天吏`的差序关系，目前也还没有被完整展开。
- 她是否具有更明确的战斗职能，正文同样尚未写清。
