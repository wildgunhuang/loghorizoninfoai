# CHR-003 叶莲仙女（葉蓮仙女）

- 分类：人物 / 中原本地住民
- 适用范围：特定服务器特有
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`叶莲仙女`是当前阅读范围内第一位被明确写成中原本地`仙女`的角色。现有材料能够稳定确认：她属于中原`古代种 / 仙女`体系，与`白桃庙`及其周边住民的安危直接相关；但同一个角色又在后续阶段明显转向诡异化，开始传达`昆仑`与`西王母娘娘`的高位意志，甚至高概率参与了对`艾利亚斯`的操控与弱化。因此，这个人物必须按阶段变化来整理，不能把前期求援者与后期敌对化状态互相覆盖。

## 基本身份
- 艾利亚斯与她见面时，对方先明确表示自己也是`古代种`之一。
- 结合前后文对中原称呼的解释，可以确认她属于中原服`仙女`这一套本地化命名系统。
- 因而她首先应被视为中原本地高位住民，而不是外来访客。
- 关联出处：
  - PDF页码：2499
  - 章节标题：`096 Chapter 2.01`
  - 关键词：`古来種`、`仙女`
  - 日文短摘录：`私も〈古来種〉のひとり` / `女性であれば〈仙女〉`

## 前期是白桃庙求援线的发起者
- 她向艾利亚斯俯身请求，希望他去征伐占领`白桃庙`的`魔人`。
- 这让她不仅是背景说明者，而是把整个`白桃庙 / 仙境 / 魔人占领`剧情线真正启动起来的人。
- 关联出处：
  - PDF页码：2499-2500
  - 章节标题：`096 Chapter 2.01`
  - 关键词：`お願い`、`白桃廟`、`魔人`
  - 日文短摘录：`エリスアさまにお願いがあります` / `〈白桃廟〉もそのひとつ` / `魔人を征伐くださいまし`

## 她的求援指向的是更大范围的仙境秩序崩塌
- 她的求援并不只提`白桃庙`一地，而是先说中原各地许多`仙境`已经陷落或被废弃。
- 这说明她所代表的并非单个山头的小危机，而是更大范围的中原本地秩序崩塌。
- 关联出处：
  - PDF页码：2500
  - 章节标题：`096 Chapter 2.01`
  - 关键词：`各地の仙境`、`陥落`
  - 日文短摘录：`各地の仙境も陥落し` / `あるいは打ち捨てられて`

## 她的求援同时带出了天吏阶层
- 叶莲仙女在描述现状时，特别提到“弱小的`天吏`”正每天在恐惧中数着自己的命数过活。
- 这条信息很重要，因为它把中原仙境里的住民层级从“只有仙人 / 仙女”推进到了更具体的地方社会构成。
- 关联出处：
  - PDF页码：2500
  - 章节标题：`096 Chapter 2.01`
  - 关键词：`天吏`、`恐怖`
  - 日文短摘录：`か弱き天吏は` / `恐怖のうちで暮らす毎日`

## 中段开始显出明显异常感
- 到克拉斯提与花貂的日常描写里，又补出一个带明显主观色彩的后续观察：他觉得叶莲仙女虽然容姿端整，却缺少“自己在活着”的感觉，像背景一样。
- 这不是客观设定，而是克拉斯提的角色观感，不能当成绝对人物事实。
- 但它是一个非常重要的阶段变化线索，因为后文确实出现了更明显的异常转向。
- 判定：`主观体感`
- 关联出处：
  - PDF页码：2533-2534
  - 章节标题：`098 Chapter 2.05`
  - 关键词：`葉蓮仙女`、`気持ちが希薄`、`背景`
  - 日文短摘录：`葉蓮仙女` / `気持ちが希薄` / `背景のような女性`

## 后期疑似转为昆仑意志的传达者
- 到`狼君山`地下战线时，叶莲仙女重新现身，语气和立场都已明显变化。
- 她不再只是单纯求援，而开始转述“`召唤术`概要已送往`昆仑`、各地可按资源派兵”的说明。
- 这说明她在后续阶段至少承担了更高层情报与命令的传达功能。
- 关联出处：
  - PDF页码：2610
  - 章节标题：`101 Chapter 4.04`
  - 关键词：`崑崙`、`召喚術の概要`
  - 日文短摘录：`召喚術の概要が崑崙へと送られて` / `各地には資源に応じた軍勢`

## 会直接提及西王母娘娘与资格判断
- 她随后更进一步宣称：`昆仑`上的`西王母`正在催促时刻，并判断“此地没有有资格者”。
- 这已经明显不是前期那个“来求人救庙”的姿态，而更像在宣读高位意志。
- 由于她当时的状态本身就非常可疑，这条材料应保留为后续阶段变化，而不是覆盖前期角色印象。
- 关联出处：
  - PDF页码：2611
  - 章节标题：`101 Chapter 4.04`
  - 关键词：`西王母`、`資格`
  - 日文短摘录：`西王母が` / `この地に資格あるものはいない`

## 后期高概率参与了对艾利亚斯的操控与弱化
- 到地下战线后，克拉斯提已经把艾利亚斯的敌对化与叶莲仙女直接连在一起判断。
- 他的结论是：这里发生的是`洗脑 / 催眠 / 意识诱导`，而且叶莲仙女正是实施者。
- 紧接着他还直接指出，“把艾利亚斯变弱的就是你”。
- 这组材料已经足够把她从单纯“诡异传话者”推进到“高概率敌对操控者”。
- 判定：`高概率效果`
- 关联出处：
  - PDF页码：2616-2617
  - 章节标题：`101 Chapter 4.05`
  - 关键词：`洗脳`、`意識誘導`、`エリアスを弱くした`
  - 日文短摘录：`洗脳あるいは催眠、意識誘導` / `葉蓮仙女による` / `エリアスを弱くしたのは貴女`

## 在后续战线中显出实际战斗能力
- 克拉斯提与她正式交锋时，文本已经不再把她写成单纯求援者或传令者，而明确表示她似乎具备“高度战斗能力”。
- 这意味着她在后续阶段的威胁不只在话语和操控层面，也包含正面交战能力。
- 关联出处：

## 当前这一个体已被击杀退场
- 到战线收束处，叶莲仙女并没有逃脱或留下悬而未决的退场，而是被克拉斯提正面斩杀。
- 正文还明确补了她的退场形态：遗体数瞬后化为沸腾般的虹色泡沫。
- 这条阶段变化非常关键，因为它说明“叶莲仙女”至少作为当前这一个战线中的敌对个体，已经走到了终点。
- 关联出处：

  - 日文短摘录：`真っ二つ` / `虹の泡`

## 阶段变化
- 初次登场时，`叶莲仙女`首先被写成`白桃庙`求援线的发起者，也是中原本地`仙女 / 古代种`体系的入口角色。
- 中段开始，她又逐渐显出异常与空洞感，从地方求援者转向带有不自然气息的高位住民。
- 到`狼君山`地下战线后，她进一步转变为`昆仑`与`西王母娘娘`意志的传达者，并高概率参与了对`艾利亚斯`的操控与弱化。
- 最后，作为当前战线中的这个体，她已经被击杀退场，因此人物页必须同时保留“前期求援者”和“后期敌对个体”两种状态。

## 对中原服的参考价值
- `叶莲仙女`适合作为中原本地高位住民模板：
  - 属于`仙女 / 古代种`体系
  - 与`仙境`和地方住民安危直接绑定
  - 可以作为求援者与地方秩序失守的见证人
  - 也可以在后期转为高层意志的传达者、操控者或敌对执行者

## 当前未知或仍待补证的部分
- 叶莲仙女后期的异常转向，究竟是主动投向`昆仑`一侧、被操控、被替换，还是某种更复杂的变化，当前仍不能写死。
- 她的主职业式能力、正式地位与更完整前史，正文也还未充分展开。
- 虽然当前这一个体已被击杀，但“叶莲仙女”究竟是唯一个体、职位称呼，还是更宽泛的身份名号，现阶段仍应保留边界。
