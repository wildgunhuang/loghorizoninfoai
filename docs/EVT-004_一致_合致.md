# EVT-004 一致（合致）

- 分类：事件与现象 / 跨世界接触事件
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`一致`是`航界种`一侧对相关跨世界接触事件的称呼，对应日文原词`合致`。当前材料显示，它至少与玩家所称的`大灾害`高度重合，或至少是其更高层的解释框架之一；并且正是这次`一致`，使`冒险者`、`大地人`、`航界种`得以在本不该相遇的前提下，于`赛尔迪希亚`发生接触。

## 术语性质
- 现阶段最稳妥的理解方式，是把`一致`视为`航界种`一侧的上位解释术语。
- 它不是玩家社会的常用名，也不是对`大灾害`的简单翻译，而是一种带有高层理论背景的命名框架。

## 航界种对大灾害的上位称呼
- 在`成惠2`的书信里，她明确说：你们所说的`大灾害`，在她们的说法里可以称作`合致`。
- 但她同时又强调，她们能提供的只是“她们自己对合致的理解”，不是原理的彻底解明，也不是解决方案。
- 这说明`一致`不是简单别名，而是带有更高层理论背景的术语。
- 关联出处：
  - PDF页码：2217-2218
  - 章节标题：`084 ノウアスフィアの開墾`
  - 关键词：`合致`、`大災害`、`理解`、`解決策`
  - 日文短摘录：`あなたたち風にいうと〈大災害〉` / `私たちなりの〈合致〉に対する理解`

## 航界种与冒险者都因一致而抵达赛尔迪希亚
- 同一封信里还明确说，`航界种`与`冒险者`一样，都是通过那次`一致`抵达`赛尔迪希亚`的。
- 差别在于：`航界种`原本距离更远、且本来不具有这里可用的物理形状，因此必须借用本地已存在的肉体作为化身。
- 这使`一致`不仅是“玩家卷入异世界”，也是多类存在同时卷入或接合到同一世界的节点。
- 关联出处：
  - PDF页码：2218
  - 章节标题：`084 ノウアスフィアの開墾`
  - 关键词：`合致`、`たどり着いた`、`化身`
  - 日文短摘录：`あの〈合致〉によりセルデシアにたどり着いた` / `既に存在した肉体を化身として借り受ける`

## 并非被完全理解的单次偶发
- `成惠2`还说明，她们一方并不是“知道全部原理后主动启动”了这次事件。
- 更准确地说，她们是以极其巨大的总量试错，偶然命中了`一致`。
- 这意味着：`一致`不是一个被完全掌控、能随便复制的现象，而是极高代价与巨大母数下偶然碰到的宇宙级异常。
- 关联出处：
  - PDF页码：2290
  - 章节标题：`087`
  - 关键词：`合致`、`総当たり`、`偶然`
  - 日文短摘录：`総当たりで〈合致〉を引き当てた` / `偶然を引き当てた`

## 影响范围覆盖整个赛尔迪希亚亚世界
- 书信还明确说，`一致`的影响及于整个`赛尔迪希亚亜世界`。
- 这让它在规模上明显高于局部区域异常，也支持把它视为世界级接合 / 重叠现象。
- 关联出处：
  - PDF页码：2291
  - 章节标题：`087`
  - 关键词：`合致の影響`、`セルデシア亜世界全体`
  - 日文短摘录：`〈合致〉の影響はこのセルデシア亜世界全体に及ぶ`

## 与大灾害的关系应写成“高度对应但不同框架”
- 现阶段最稳妥的整理方式，不是简单写成“`一致 = 大灾害` 已被完全证明”，而是写成：
  - `大灾害` 是玩家社会的命名
  - `一致` 是`航界种`侧的上位解释术语
  - 两者在当前材料里高度对应，至少描述的是同一大事件的不同认知框架

## 阶段变化
- `一致`最初作为`航界种`书信中的高层术语浮出，用来对应玩家所称的`大灾害`。
- 随后材料又把它从“术语对照”推进到“多类存在同时抵达赛尔迪希亚的接触事件”。
- 再往后，它进一步被说明为并非完全掌控的偶然命中事件，影响范围及于整个`赛尔迪希亚亜世界`，从而真正形成世界级解释框架。

## 对中原服的参考价值
- `一致`适合直接保留为中原服后期解释层的正式术语。
- 你可以在前期只用`大灾害`，到后期再逐步揭露：
  - 这不是单纯服务器事故
  - 而是更高层跨世界接触事件
  - 且其影响远不止玩家本身

## 当前未知或仍待补证的部分
- `一致`的发动条件、是否可复现、以及它与`森罗变化`的精确关系，当前仍未彻底说明。
- 它与`开拓智域`、`大灾害`的先后因果链条，现阶段也仍应保留开放。
- `航界种`对`一致`的解释是否代表全部高层真相，目前同样不能写死。
