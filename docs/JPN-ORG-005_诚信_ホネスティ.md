# JPN-ORG-005 诚信（ホネスティ）

- 分类：组织与制度 / 日本服专属组织
- 适用范围：日本服特有
- 对中原服的可用性：高参考价值
- 日本服依赖度：高

## 摘要
`诚信`是日本服`秋叶原`的大型公会之一，由`亚因斯`领导，也是`圆桌会议`成立时就明确在场并表态支持的成员势力之一。到“圆桌崩坏”阶段，它所承载的已不只是普通公会运营，而是对低活力、低收入、失去方向的玩家群体的代表诉求，以及对秋叶原治理方式的挑战。

## 在圆桌成立章中的早期位置
- 在`圆桌会议`真正成立的关键会场中，`亚因斯`就已经以`诚信`公会会长的身份明确表态同意加入。
- 更重要的是，他当场把`大地人`关系改善列为未来方向之一。
- 这意味着`诚信`并不是后期才突然跳出来的势力，而是从圆桌成立之初就处在秋叶原自治框架内部。
- 关联出处：
  - PDF页码：500-501
  - 章节标题：`017`
  - 关键词：`ホネスティ`、`同意する`、`大地人との関係改善`
  - 日文短摘录：`〈ホネスティ〉も同意する` / `〈大地人〉との関係改善に努めよう`

## 作为圆桌成员公会的基本位置
- 到后期，文中明确写出：`亚因斯`是`圆桌的十一人`之一，同时也是`诚信`的统领者。
- 这说明`诚信`并不是外围小公会，而是秋叶原自治结构内部的核心势力。
- 关联出处：
  - PDF页码：2843-2845
  - 章节标题：`112 Chapter 4.01`

## 曾承担妖精环调查任务
- 在更早一段里，`诚信`还承担过`妖精环`时刻表调查这一高优先级任务。
- 但文中明确指出，受限于公会成员数，这项任务最终没能取得成果。
- 这条信息很重要，因为它说明`诚信`并非只会提出政治主张，也曾实际承担过秋叶原公共事务。
- 关联出处：
  - PDF页码：2680-2682
  - 章节标题：`105 Chapter 1.01`
  - 关键词：`ホネスティ`、`妖精の輪`、`調査任務`
  - 日文短摘录：`〈ホネスティ〉の奮闘むなしく` / `調査任務を引き受けていた`

## 与失业、低活力层的议题密切相连
- 后期城惠反省时明确提到：秋叶原里“真正低迷到无法行动”的玩家，最大估计也不过`150`人左右，而且原本是有可能通过各公会分担吸纳来缓和的。
- 同时，文中还写到卡拉辛会先让部分“原诚信成员”暂时加入`第八商店街`，再慢慢安排实际工作与合作。
- 更稳妥的整理写法是：`诚信`与这批低收入、失去方向、需要重新安置的玩家问题强相关，但不能简单写成“所有弱势玩家都属于诚信”。
- 关联出处：
  - PDF页码：2791-2794, 2824-2826
  - 章节标题：`109 Chapter 3.03`；`111 Chapter 3.06`
  - 关键词：`元ホネスティ`、`第八商店街`、`百五十名`
  - 日文短摘录：`元〈ホネスティ〉` / `百五十名程度` / `引き受ける`

## 离脱圆桌后会直接动摇其正当性
- 后续的密谈里，作品把一件事写得很清楚：`圆桌会议`原本的合法性之一，在于它被视作“结集秋叶原诸势力的组织”。
- 而`诚信`离脱之后，这种正当性就被明显削弱了。
- 这说明`诚信`并不是可有可无的边缘成员，而是足以影响圆桌名义完整性的重量级势力。
- 关联出处：

  - 关键词：`諸勢力を結集`、`正当性`、`ホネスティが離脱`
  - 日文短摘录：`諸勢力を結集させた組織` / `〈ホネスティ〉が離脱した時点で` / `大きく揺らいでしまった`

## 后期政治转向
- 在`圆桌崩坏`阶段，`亚因斯`已经不再满足于在会内提出建议。
- 他公开指出：在当前的`圆桌会议`结构里，“提案”最终没有意义，并将问题推进到实际行动层面。
- 这里能稳定确认的是：`诚信`在后期已经成为推动秋叶原治理结构变化的主力之一；但其全部纲领与最终成形制度，仍需后文继续补证。
- 关联出处：
  - PDF页码：2802-2804
  - 章节标题：`109 Chapter 3.03`
  - 关键词：`提案`、`意味もない`、`行動`
  - 日文短摘录：`『提案』というのは最終的には何の意味もない` / `自らが行動をしなかった`

## 以大地人为对象的政治语言暴露了边界
- 再往后看，`诚信`的政治转向并不是单纯“路线更清晰”。
- 亚因斯在面对聚集起来的`大地人`时，明确意识到自己过去的理想仍然高度面向`冒险者`内部问题，而未必能直接说服那些把秋叶原当作避难地的人。
- 原文甚至直接写出：`圆桌会议`崩坏的直接原因就是`诚信`脱退，而亚因斯在说到`アキバ統治府`时当场失语。
- 这说明`诚信`后期的真正难题，不只是“要不要改革”，而是它能否把自己的诉求转换成跨阶层都能接受的政治语言。
- 关联出处：
  - PDF页码：2916-2918
  - 章节标题：`114 Chapter 5.03`
  - 关键词：`〈円卓会議〉が崩壊した直接の原因`、`〈ホネスティ〉の脱退`、`アキバ統治府`
  - 日文短摘录：`〈円卓会議〉が崩壊した直接の原因は、〈ホネスティ〉の脱退だ` / `〈大地人〉を利用した多数派工作` / `〈アキバ統治府〉は――`

## 撤退后的公会崩解
- 后续结果也已经写得很清楚：随着亚因斯放弃`アキバ統治府`，`诚信`自身迅速进入崩解。
- 投票后仅两天，原本一度逼近`1500`人的`诚信`就只剩下数十名成员，作品甚至直接写到它“事实上已经灭亡”。
- 这使`诚信`从“大型问题处理型公会”的样本，又进一步变成了“把成员诉求转化为政治行动失败后，组织本体反而会先崩掉”的案例。
- 关联出处：
  - PDF页码：2936
  - 章节标题：`115 Chapter 5.05`
  - 关键词：`千五百名`、`数十名`、`事実上、滅びた`
  - 日文短摘录：`一時は千五百名に迫っていた` / `数十名を残すのみ` / `〈ホネスティ〉は事実上、滅びた`

## 阶段变化
- 初期阶段，`诚信`首先是`圆桌会议`成立时就在场并表态支持的核心成员公会之一。
- 中期阶段，它又承担过`妖精环`调查等公共事务，并与低活力、低收入层玩家问题逐渐深度绑定。
- 到后期政治危机阶段，它转为挑战秋叶原治理结构的重要力量，并因离脱`圆桌会议`而直接动摇后者正当性。
- 最后，随着`アキバ統治府`路线失败，`诚信`自身也迅速崩解，成为“问题处理型大型公会政治失败后组织反噬”的具体案例。

## 对中原服的参考价值
- `诚信`最值得借鉴的不是名称或具体成员，而是这种公会类型：
  - 并非纯战斗强公会
  - 会卷入失业、低迷、安置、福利和公共工程预算等议题
  - 在城市治理失灵时，容易转化为要求制度重组的政治势力

## 当前未知或仍待补证的部分
- `诚信`在全盛期内部的更细成员构成、常态化业务分工与财政基础，正文仍未完全展开。
- 其残余成员之后是否被其他势力吸纳、是否有重组尝试，当前也还缺少足够后文材料。
- 亚因斯个人路线与`诚信`整体诉求之间究竟有多大重合度，现阶段仍应保留边界。
