# ORG-002 公会会馆（ギルド会館）

- 分类：组织与制度 / 公共设施
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：中

## 摘要
`公会会馆`是玩家都市中的公会综合服务设施。它不是某个单一`公会`的据点，而是处理`公会`手续、租借`公会厅`、提供银行功能等事务的公共设施。对中原服创作而言，它属于“玩家社会基础设施”的核心组成部分；而在灾后现实化条件下，它甚至可能成为城市金融、物流与治理控制的关键杠杆。

## 设施性质
- 当前最稳妥的写法是：`公会会馆`属于面向所有玩家组织的公共行政与服务枢纽。
- 它与`公会厅`的区别必须明确：
  - `公会会馆`是公共手续与服务设施
  - `公会厅`是各个`公会`实际使用的私有据点区域

## 基本功能
- 文中明确说明，`公会会馆`存在于各个城市，通常是复合多功能的大型建筑。
- 它的职员可处理`公会`设立、未所属玩家加入`公会`、退出`公会`等手续。
- 秋叶原的`公会会馆`还附带银行支店，玩家可以办理存款与贵重物品寄存。

## 它还是公会厅租借窗口
- `公会会馆`同时承担`公会厅`租借窗口的功能。
- 这说明它不仅处理名单与手续，还连接着玩家组织的实际空间配置。
- 换言之，`公会会馆`更接近“公共行政与系统服务中心”，而`公会厅`则是各`公会`实际使用的独立据点区域。
- 关联出处：
  - PDF页码：50-51
  - 章节标题：`003`
  - 关键词：`ギルドホール`、`貸し出し`
  - 日文短摘录：`ギルドホールの貸し出し` / `ギルド会館でギルドホールを借りる`

## 灾后它会变成治理与金融枢纽
- 后文进一步显示，秋叶原的`公会会馆`不仅是办手续的地方，还关联本地唯一常用银行窗口。
- 因此，一旦会馆区域的所有权与出入设定被控制，影响的就不再只是`公会`事务，而是城市金融、物资流转与自治财政。
- 这部分是日本服秋叶原的具体实现，但其底层逻辑对于中原服同样重要。
- 关联出处：
  - PDF页码：477-478, 511-512
  - 章节标题：`017`
  - 关键词：`銀行`、`入場制限`、`税`、`金貨１枚`
  - 日文短摘录：`唯一の銀行窓口` / `入場制限` / `金貨１枚が徴収`

## 阶段变化
- 行政阶段：先被视为办理`公会`手续与提供公共服务的设施。
- 枢纽阶段：后文又补出它兼具银行与空间租借枢纽功能。
- 治理阶段：灾后现实化后，它进一步显露为可左右城市金融与公共流转的控制点。
- 这几层都应并列保留，不能把后期秋叶原政治危机倒灌成所有城市一开始就同样紧张。

## 对中原服的参考价值
- 可直接继承的部分：
  - 玩家都市里存在统一的`公会`事务设施
  - 银行与公共手续可能集中在同一枢纽
  - `公会会馆`若带有区域所有权与参数设置，就能成为治理杠杆
- 中原服本地化时，可以改变建筑样式、行政风格与城市结构，但这套“公共组织基础设施”逻辑很值得保留。

## 当前未知或仍待补证的部分
- 各服务器、各城市中`公会会馆`是否都与银行强绑定，当前尚未全局汇总。
- 除秋叶原外，其他城市`公会会馆`的权限、规模与财政角色，仍需继续补证。
