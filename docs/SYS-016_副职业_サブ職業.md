# SYS-016 副职业（サブ職業）

- 分类：职业与技能 / 系统
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：中

## 摘要
`副职业`是独立于十二种主职业之外的另一套职业系统，主要提供不直接参与主战斗的便利能力、生活能力或角色扮演型特技。其底层框架大概率是全服共通的，但具体有哪些`副职业`、哪些由本地运营追加，后文显示可能存在服务器差异；而在大灾害后，`副职业`又从菜单式辅助能力进一步现实化成可直接改造生活、物资与环境的技能体系。

## 系统性质
- 当前最稳妥的写法是：`副职业`是与主职业并列存在的第二职业层。
- 它不是主职业的下位分支，也不是纯粹剧情标签，而是有独立成长、独立用途与独立技能表现的系统。
- 对原作世界来说，`副职业`的重要性在大灾害后明显被放大。

## 基本框架
- 城惠将`副职业`概括为“直接战斗之外的便利能力集”。
- 它与十二种主职业独立存在，不是主职业的下位分支。
- 在当前阅读范围内，`副职业`至少可分成两类：生产系与角色扮演系。

## 生产系副职业负责物品制造与加工
- 生产系`副职业`可以利用相应素材与设施制造物品。
- 当前明确提到的例子包括`厨师`与`抄写师`；城惠的`副职业`就是`抄写师`，可复制魔法书、地图、文书等。
- 这类`副职业`可通过城市区域内对应`大地人`处购买手引书并积累独立经验值来成长，不需要特殊任务或固定队友。
- 关联出处：
  - PDF页码：107, 124
  - 章节标题：`006`
  - 关键词：`筆写師`、`料理人`、`手引き書`、`経験値`
  - 日文短摘录：`高レベルの生産サブ職業〈筆写師〉` / `〈料理人〉` / `手引き書を購入し` / `経験は戦闘のそれとは別個`

## 角色扮演系副职业提供特殊能力
- 角色扮演系`副职业`通常不能直接制作物品，但会赋予特殊能力、珍稀特技，甚至特殊装备。
- `晓`持有的`追踪者`就是这一类，能提供追踪、尾行、消除气息、`无声移动`等偏实用型能力。
- 在异世界化后的环境里，这些原本偏便利性的能力会变得更像现实中的实际生存技巧。
- 关联出处：
  - PDF页码：121, 123-126
  - 章节标题：`006`
  - 关键词：`追跡者`、`暗視`、`無音移動`、`特殊能力`
  - 日文短摘录：`〈暗視〉` / `〈無音移動〉` / `〈追跡者〉` / `特殊能力や珍しい特技`

## 副职业目录可能存在服务器差异
- 城惠估计`副职业`至少有约五十种，实际总数可能更多。
- 文中明确指出，`副职业`会随着资料片持续增加，其中甚至包含由本地运营公司独自追加的项目。
- 这意味着：`副职业系统`本身大概率全服共通，但`副职业目录`未必全服完全一致。
- 对中原服创作而言，最稳妥的做法是沿用`副职业`框架，再谨慎决定哪些具体`副职业`属于中原服本地追加内容。
- 关联出处：
  - PDF页码：125-126
  - 章节标题：`006`
  - 关键词：`拡張パック`、`追加`、`日本サーバを運営している会社`、`５０種類程度`
  - 日文短摘录：`拡張パックの度に追加され` / `日本サーバを運営している会社などが独自に追加` / `５０種類程度は存在する`

## 大灾害后副职业开始现实化
- 后文进一步显示，`副职业`的作用方式在灾后发生了“现实化”。
- 以`厨师`为例，真正起作用的并不再只是菜单生成，而是技能等级、现实工序、本人知识与手艺共同作用的结果。
- 公开料理法后，料理品质甚至开始受到素材新鲜度和操作者技术差异影响。
- 关联出处：
  - PDF页码：290-291, 511-512, 542
  - 章节标题：`011`；`017`
  - 关键词：`調理スキル`、`普通の手順`、`知識や技術`、`素材の選択`
  - 日文短摘录：`普通の手順で料理をする` / `調理スキルは必要` / `知識や技術も重要` / `素材の選択`

## 现实化会扩展到其他副职业
- 料理突破带来的影响并没有停在`厨师`身上。
- 文中明确写到，其他生产系`副职业`、甚至角色扮演系`副职业`也开始测试自己的技术在异世界中的新适用方式。
- 这说明`副职业`在灾后不只是“菜单功能集合”，而更像“现实世界中的专门技能包”。
- 关联出处：
  - PDF页码：520-521
  - 章节标题：`018`
  - 关键词：`生産職`、`ロールプレイ系`、`適合`、`実験`
  - 日文短摘录：`あらゆる生産職` / `ロールプレイ系のサブ職業` / `どんな適合を果たすかの実験`

## 甚至可以干预环境与据点结构
- 文中还给出了一个更强的例子：生产系`副职业`的实验已经扩展到对建筑环境本身的改造，例如在建筑内部开洞、架设楼梯。
- 这意味着灾后`副职业`的现实化并不只会制造“更好的物品”，还可能改变空间结构与据点功能。
- 关联出处：
  - PDF页码：520-521

  - 关键词：`床に穴`、`階段`、`工事`
  - 日文短摘录：`床に穴を開けて` / `階段` / `工事を頼めば良い`

## 阶段变化
- 系统阶段：先被定义为主职业之外的便利能力系统。
- 分类阶段：很快补出生产系与角色扮演系两大方向，以及独立成长方式。
- 差异阶段：再补出“目录可能因服务器与运营追加而不同”的边界。
- 现实化阶段：后文进一步说明`副职业`会转化成现实中的专业技能与环境改造能力。
- 这几层必须并列保留，不能把后期现实化成果倒灌成开篇就已显而易见。

## 对中原服的参考价值
- 对中原服最值得直接继承的不是日本服具体有哪些`副职业`，而是以下底层逻辑：
  - 主职业外另有完整第二职业层
  - `副职业`在大灾害后会现实化
  - 不同服务器可能存在本地追加目录
  - 生产与角色扮演系都能在现实社会中变成重要能力
- 这对中原服的城市工坊、公会分工、地方技术差异和边境生存写法都很关键。
