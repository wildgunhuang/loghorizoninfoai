# SYS-019 任务（クエスト）

- 分类：系统 / 冒险流程
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`任务`原本是《幻境神话》中由非玩家角色提供、引导玩家获取报酬与推进内容的标准系统形式。但在大灾害后，由于`大地人`不再只是脚本化存在，`任务系统`也不应再被视作绝对稳定、可无限重复的游戏流程；后文材料还进一步说明，至少部分与任务有关的高位住民会记住过去与`冒险者`共同经历的细节。

## 系统性质
- 当前最稳妥的写法是：`任务`仍是世界的重要系统遗产，但它在异世界化后不再必然保持旧 MMO 那种“无限重复、脚本重置、完全去人格化”的稳定形态。
- 因此本页重点不在罗列任务目录，而在记录`任务系统`在现实化后的边界变化。

## 游戏时代任务本就是成长与报酬入口
- 开篇叙述中，`大地人`被说明为会卖货、提供信息与发布`任务`的原住民群体。
- 这说明在旧有认知中，`任务`是玩家日常成长与获取奖励的标准入口之一。
- 关联出处：
  - PDF页码：42
  - 章节标题：`003`
  - 关键词：`クエスト`、`大地人`
  - 日文短摘录：`クエストをくれたりする`

## 大灾害后任务的稳定性开始被怀疑
- 后文明确提出疑问：既然大量`任务`的起点原本就依附于`大地人`，那么当`大地人`变成真正会生活、会受伤、会死亡的存在后，`任务`是否还能像游戏时代那样稳定存在？
- 文中并未断言“所有`任务`都已失效”，而是指出它们的持续性已经不再理所当然。
- 关联出处：
  - PDF页码：548-549
  - 章节标题：`019`
  - 关键词：`クエスト`、`大地人`、`機能が失われる`
  - 日文短摘录：`徐々にその機能が失われている` / `そのクエストは発生しなくなってしまうかも`

## 任务失稳会直接影响新人生存路径
- 文中拿重量无视魔法背包相关`任务`举例，说明如果某些定番`任务`失效，影响的不只是剧情完整性，而是新人的基础生存条件与成长路径。
- 这意味着大灾害后的`任务系统`问题，会直接反馈到新人`冒险者`的装备获取、生活便利与社会流动性上。
- 关联出处：
  - PDF页码：549
  - 章节标题：`019`
  - 关键词：`魔法のバッグ`、`定番クエスト`、`新人数`
  - 日文短摘录：`重量無視効果のある魔法のバッグ` / `定番クエスト` / `新人プレイヤーの手には入らない`

## 至少部分任务相关住民不会像脚本那样完全重置
- 更后面的秋叶原异变篇里，曾经在旧`任务`线中与`冒险者`同行过的一对`古代种`姐妹，明确记得过去一起冒险、交付道具和共同活动的细节。
- 后文甚至补到了更具体的层面：她们不仅记得“有人来过”，还记得早期`50`级、装备朴素时期的`冒险者`们来庆祝“初次 raid”、递交袋子与物资、在秘密基地里来来往往的景象。
- 这条材料很重要，因为它说明“`任务`相关存在”并不一定会像传统 MMO 那样在`任务`结束后彻底重置。
- 但现阶段还不能因此写死“所有任务 NPC 都会长期记忆并延续关系”，更稳妥的结论是：至少部分`任务`相关角色，尤其是高位住民，并不会把过去的共同经历视作一次性脚本。
- 判定：`高概率效果`
- 关联出处：
  - PDF页码：3093-3095, 3162-3164
  - 章节标题：`121 4.01`；`125 5.02`
  - 关键词：`鱗`、`覚えてる`、`地下帝国`、`初レイド`、`レベル五十`
  - 日文短摘录：`覚えてるの？` / `鱗いっぱいくれた` / `祝！初レイド！` / `レベル五十`

## 阶段变化
- 游戏阶段：先被理解为标准成长、报酬与内容推进入口。
- 怀疑阶段：大灾害后开始被质疑是否还能无限稳定运行。
- 记忆阶段：后文又补出至少部分`任务`相关存在会记住过去互动。
- 这几层必须并列记录，不能把“后期记忆残留”反写成所有`任务`从一开始就完全人格化。

## 对中原服的参考价值
- 可直接继承的部分：
  - `任务`仍是这个世界的重要系统遗产
  - 但`任务`不再一定是永远稳定、无限重复、完全去人格化的机制
  - 若关键`大地人`、关键地点或关键关系改变，`任务`可能失效、变质或转化
- 这对中原服写作尤其重要，因为你可以在不违背原作的前提下写出“任务链断裂”“任务转化为真实委托”“旧任务遗民仍记得玩家”的情形。

## 当前未知或仍待补证的部分
- 哪些`任务`仍可稳定重复、哪些已失效，当前没有全局目录。
- 记忆保留现象究竟只集中在高位住民，还是会延伸到更广泛的`任务`相关角色，当前仍待补证。
