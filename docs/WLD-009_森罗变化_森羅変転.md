# WLD-009 森罗变化（森羅変転）

- 分类：世界观 / 超大规模世界魔法
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`森罗变化`是后期由`李・耿恩`提出的重要宇宙论与历史解释框架。按当前材料，至少可以区分`第一次森罗变化`、`第二次森罗变化`、`第三次森罗变化`三次：第一次与亜人间的出现有关，第二次与`冒险者`出现有关，第三次则对应玩家视角中的`大灾害`。但这一整套说法目前仍应视作作品内严肃提出的研究模型，而非已经完全盖棺定论的真理。

## 概念性质
- 现阶段最稳妥的理解方式，是把`森罗变化`视为“世界级变化的历史解释模型”。
- 它显然不是普通施法尺度内的魔法术语，而是用来解释种族史、召唤史和大灾害史的大框架。

## 被定义为超越个人施法尺度的世界级魔法
- `李・耿恩`明确把`森罗变化`解释为规模远超个人能力的魔法。
- 文中专门用“战略级、国防级、大陆级”这类尺度来帮助理解，强调这不是普通`魔术师`在战斗中使用的法术，而是足以影响整个世界结构的灾害级魔法。

  - 日文短摘录：`その魔法は〈森羅変転〉と呼ばれています` / `国防級` / `大陸級`

## 第三次森罗变化对应大灾害
- `李・耿恩`直接说，他把玩家所称的`大灾害`称为`第三次森罗变化`。
- 这说明至少在大地人学者视角里，`大灾害`并不是孤立事故，而是被放进了更长历史序列的第三次世界级变化。
- 关联出处：
  - PDF页码：706
  - 章节标题：`024`
  - 关键词：`第三の森羅変転`、`大災害`
  - 日文短摘录：`わたしは〈第三の森羅変転〉と呼んでいます`

## 第一次森罗变化与亜人间的出现相关
- `李・耿恩`的说明里，`第一次森罗变化`与`六倾姬`时代的高位仪式魔法有关。
- 他认为，这次变化的效果是`亜人间`的发生；甚至明确说这个世界在约三百年前原本并没有亜人间。
- 他还进一步提出假说，认为当时极大量的`魂素材`可能参与了这次变化。
- 因此，`第一次森罗变化`不仅是种族史问题，也与后来的`魂魄理论`发生连结。
- 关联出处：
  - PDF页码：712
  - 章节标题：`024`
  - 关键词：`第一の森羅変転`、`亜人間の発生`、`魂素材`
  - 日文短摘录：`発動したのが、〈第一の森羅変転〉です` / `亜人間の発生です` / `魂素材`

## 第二次森罗变化与冒险者的出现相关
- 对于`第二次森罗变化`，`李・耿恩`说现存资料远少于第一次。
- 现有少量记录提到，人类、精灵、矮人与许多`法仪族`曾动员并使用`神圣召唤术`，并在约二百四十年前带来了`冒险者`的出现。
- 换言之，`冒险者`在这套历史解释里，并不是“原本就存在于此世界的种族”，而是第二次世界变化的结果。
- 关联出处：

## 后续又被纳入魂魄理论框架重述
- 到更后期，`李・耿恩`再次把`第一 / 第二次森罗变化`与`魂魄理论`连在一起说明。
- 他把亜人间视作`第一次森罗变化`之“诅咒”的产物，又把`冒险者`视作因`第二次森罗变化`而被召唤到此世界的存在。
- 这使`森罗变化`不再只是历史传说，而成为解释种族、复活与世界生成机制的重要上位框架。
- 关联出处：
  - PDF页码：738、880

  - 关键词：`第一の森羅変転`、`呪い`、`第二の森羅変転`、`召喚`
  - 日文短摘录：`〈第一の森羅変転〉の呪い` / `〈第二の森羅変転〉によりこの世界に召喚された存在`

## 阶段变化
- `森罗变化`最初浮出时，首先被`李・耿恩`作为超越个人施法尺度的世界级魔法概念提出，并直接把`大灾害`纳入`第三次森罗变化`。
- 随后，这一框架又继续向历史深处展开，被用来解释`第一次森罗变化`导致的亜人间发生，以及`第二次森罗变化`与`冒险者`出现之间的关系。
- 到更后续的说明里，`森罗变化`进一步和`魂魄理论`、召唤史、种族史连在一起，逐步从历史解释扩展成世界生成机制的上位模型。

## 对中原服的参考价值
- 这套框架对中原服特别重要，因为它大概率不是日本服地方传说，而是世界级历史解释。
- 你可以直接继承：
  - `大灾害 = 第三次森罗变化` 这一后期解释线
  - `冒险者`与`大地人 / 亜人间`并非同源的世界生成史
  - 世界深层机制与`魂魄`、`召唤`、`世界级魔法`有关

## 当前未知或仍待补证的部分
- `李・耿恩`的整个框架仍属于作品内学说，不能直接当成完全终审定论。
- 第一次与第二次森罗变化的具体施法者、详细过程与全貌，正文仍未完全展开。
- 它与`一致`、`大灾害`、`赛尔迪希亚`高层结构之间是否完全可互相套合，目前也仍需继续补证。
