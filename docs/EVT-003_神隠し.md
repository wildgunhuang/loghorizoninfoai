# EVT-003 神隠し

- 分类：事件与现象 / 失踪现象
- 适用范围：待考证
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`神隠し`是后期逐渐被用来称呼的一类“无尸体、无通常痕迹的消失现象”。当前材料显示，这个名字至少被用在两种场景上：一是`古代种`级别守护者整体失踪，二是`冒险者`在友人列表熄灭后彻底消失、甚至在观察下从昏睡状态中淡去。后者尤其重要，因为城惠已经把它视作“归还地球 / 往还机制”的可能线索。

## 现象性质
- 现阶段最稳妥的理解方式，是把`神隠し`视作现象名，而不是已经解释完成的机制名。
- 它先是民间 / 大地人层的通俗命名，后期才逐渐进入冒险者侧的正式调查语境。

## 作为“完全消失”的通俗命名
- 较早的材料里，`大地人`已经会用“神隠し”去形容那种没有尸体、没有战斗痕迹、仿佛整个存在被拿走的消失。
- 这说明它首先是一个现象名，而不是系统公告名。
- 关联出处：
  - PDF页码：793
  - 章节标题：`027`
  - 关键词：`神隠し`、`消えてしまった`、`痕跡`
  - 日文短摘录：`ただ、彼らは消えてしまった` / `神隠しとでも`

## 在冒险者之间变成持续追查对象
- 到后期，`神隠し`已经不是单发传闻，而是`圆桌会议`层面在调查的“行方不明”案件。
- 文中说明，相关传闻在`大灾害`后不久就已出现，并长期以都市传说形式残留。
- 真正让城惠确信它值得认真追查的，是好友列表的“暗转”与熟人失联。
- 关联出处：
  - PDF页码：2712
  - 章节标题：`097`
  - 关键词：`神隠し`、`行方不明`、`フレンドリストの暗転`
  - 日文短摘录：`〈神隠し〉` / `行方不明` / `フレンドリストの暗転`

## 已出现观察下的实证案例
- 后文已经给出极其关键的一步：有昏睡中的`冒险者`在看护人员眼前“变淡并消失”。
- 这是目前阅读范围里最硬的一次实证，不再只是传闻。
- 因此，`神隠し`不能再被当成简单的逃亡、断联或误报。
- 关联出处：
  - PDF页码：2714
  - 章节标题：`097`
  - 关键词：`薄れ、消失`、`目撃証言`、`観察下`
  - 日文短摘录：`薄れ、消失した` / `目撃証言も多数` / `観察下ではっきりと発生`

## 已进入统计与报告阶段
- 同一章里还补了一条非常重要的信息：`第八商店街`已经把相关“行方不明”情报整理成报告交到城惠手里。
- 报告中提到的规模约为`千分之二`。这说明`神隠し`虽然不是大面积普发事件，但数量已经高到不能再当成个别事故。
- 结合后文对涩谷昏睡者消失案例的引用，更稳妥的整理方式是：`神隠し`已经进入“可被统计、可被追查”的阶段，而不再只是都市传说。
- 关联出处：

## 与归还地球的关系目前仍是推测线
- 城惠随后尝试把`神隠し`与`塔里库坦`提出的“支付后归还”说法联系起来理解。
- 他怀疑，昏睡并消失的人，可能是以某种代价换取了返回地球。
- 但这一步在当前材料里仍然只是强烈怀疑，不能写成既成事实。
- 判定：`待考证`
- 关联出处：

  - 章节标题：`097`
  - 关键词：`帰還`、`同意する契約者`、`睡眠`
  - 日文短摘录：`同意する契約者` / `睡眠の中で待機状態` / `地球へと「帰還」`

## 从“归还”推进到“往还”目标
- 同一段材料里，城惠还明确区分了`归还`与`往还`。
- 他承认，自己曾向同伴宣言要让`赛尔迪希亚`与地球“往还可能”，但这显然比单纯“能不能回去”要求更高。
- 因此，`神隠し`在后期不只是“失踪案件”或“逃离这个世界的可能方式”，而是被城惠主动当成了建构双向通路线索的调查入口。
- 判定：`高概率效果`
- 关联出处：
  - PDF页码：2714
  - 章节标题：`097`
  - 关键词：`往還可能`、`帰る`、`神隠し`
  - 日文短摘录：`セルデシアと地球を往還可能にする` / `往還は当然「帰る」よりも高いハードル`

## 阶段变化
- 较早材料里，`神隠し`首先只是`大地人`侧对“无痕消失”的通俗命名。
- 到后期调查阶段，它又被推进为`圆桌会议`层面可统计、可追查的失踪现象。
- 再往后，城惠已经把它进一步当作“归还地球 / 往还机制”的调查入口，因此这条词条必须同时保留“民间现象名”和“高层研究线索”两层性质。

## 对中原服的参考价值
- `神隠し`非常适合作为中原服后期悬案线直接继承。
- 可继承的核心不是具体失踪者名单，而是：
  - 友人列表熄灭与彻底消失的关联
  - 观察下消失这一实证
  - 它可能触及“归还地球”或更高层世界机制

## 当前未知或仍待补证的部分
- `神隠し`是否只对应一种机制，还是多种“彻底消失”现象的总称，当前仍不能写死。
- 它与`共感子`、`一致`、`归还`之间的具体因果顺序，现阶段也还没有定论。
- 早期大地人口中的“神隠し”与后期冒险者失踪现象是否完全同源，仍需继续补证。
