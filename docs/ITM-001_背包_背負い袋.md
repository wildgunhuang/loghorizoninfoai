# ITM-001 背包（背負い袋）

- 分类：道具与装备 / 储物
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`背包`是玩家最基础也最关键的储物型魔法物品之一。它并不只是日常装备，而是支撑远征、野营、搜集与战利品搬运的生存基础设施。

## 已确认规则
- 文中明确提到，城惠等人所持的`背包`属于`魔法物品`。
- 当前个体可收纳约`200`公斤物品；只要物品仍被收纳在包内，其重量就不会继续压在携带者身上。
- 因此玩家即便长距离行军，也能携带大量食物、野营道具、地图材料与补给品。

## 阶级与普及度
- 文中说明，这类“魔法的包”在《幻境神话》中有多个阶级，收纳对象种类与限重各不相同。
- 因其极度便利，绝大多数玩家至少都会持有一个。

## 大灾害后的实际意义
- 在游戏时代，背包主要是便利机能。
- 在异世界化后，它直接影响远征半径、宿营能力、地城探索持续时间与物资调度。
- 对中原服写作而言，这类储物道具会深刻影响物流、长征、贸易与公会后勤，值得视为底层设定，而不是可有可无的小装备。

## 阶段变化
- 游戏时代，`背包`首先只是极为便利的标准储物型`魔法物品`。
- 到大灾害后，它的意义明显上升，不再只是 UI 式便利机能，而直接变成远征、搜集、补给与物流系统的现实基础设施。

## 当前未知或仍待补证的部分
- 不同阶级`背包`之间更细的收纳差异、对象限制与稀有版本，当前正文尚未继续展开。
- 在异世界化后，`背包`是否出现新的现实副作用或极限条件，现阶段也仍需更多材料补证。
