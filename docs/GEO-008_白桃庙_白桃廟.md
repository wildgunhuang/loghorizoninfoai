# GEO-008 白桃庙（白桃廟）

- 分类：场景与地理 / 仙境据点
- 适用范围：特定服务器特有
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`白桃庙`是中原线里已经足够独立成条的具体仙境地点。现有材料能够稳定确认：它位于`狼君山`山顶，是`仙境`这一特殊区域类别下的具名据点；至少在某个阶段，它曾遭到`魔人`压制，使周边住民陷入恐惧。与此同时，正文后续又展现了它作为“桃之仙境”的生活面貌，包括庭园、厨房、天吏与貂人族活动。由于这些描写可能对应不同阶段或不同视角，整理时必须并记，而不能互相覆盖。

## 地点性质
- `白桃庙`不是普通寺庙，而是中原本地特殊区域类别`仙境`下的具体实例。
- 它同时兼具山顶圣地、住民栖居地与地方庙宇生活空间三重性质。

## 是仙境类别下的具体地点
- 叶莲仙女求援时，直接把`白桃庙`列为诸多失守`仙境`之一。
- 艾利亚斯后面也明确复述：`狼君山`山顶有个叫`白桃庙`的`仙境`。
- 因而它并不是普通寺庙，而是中原本地特殊区域类别里的具名据点。
- 关联出处：
  - PDF页码：2500、2512
  - 章节标题：`096 Chapter 2.01` / `097 Chapter 2.02`
  - 关键词：`白桃廟`、`仙境`
  - 日文短摘录：`〈白桃廟〉もそのひとつ` / `〈白桃廟〉と呼ばれる仙境`

## 位于狼君山山顶
- 正文后文明确补出它的具体地理位置：就在`狼君山`山顶。
- 这使`白桃庙`与`天狼洞`形成了稳定的“山腹迷宫 / 山顶仙境”路线关系。
- 关联出处：
  - PDF页码：2512、2529
  - 章节标题：`097 Chapter 2.02` / `098 Chapter 2.05`
  - 关键词：`狼君山の山頂`
  - 日文短摘录：`〈狼君山〉の山頂には〈白桃廟〉` / `この仙境は〈狼君山〉の山頂にある`

## 初期求援信息中，它处于被魔人压制的危机状态
- 叶莲仙女求援时，把`白桃庙`描述成被“狂猛的魔人”制压的地点。
- 她还直接把附近村镇与庙中`天吏`的恐惧生活一并说了出来。
- 这说明在当前阶段，`白桃庙`至少一度已不再是稳定安全的隐居地。
- 关联出处：
  - PDF页码：2500
  - 章节标题：`096 Chapter 2.01`
  - 关键词：`魔人`、`制圧`、`天吏`
  - 日文短摘录：`狂猛なる魔人に制圧され` / `か弱き天吏は`

## 进入条件与天狼洞路线深度绑定
- 花貂和后续山路段落都表明：从通常意义上说，`白桃庙`与`天狼洞`的筛选路线连在一起。
- 不过叶莲仙女后来又证明，现实化后的山体本身依然可以成为绕过标准迷宫路线的物理解法。
- 也就是说，`白桃庙`对外并不是“只有副本门”才能到达，但系统默认确实会把访客导向`天狼洞`。
- 关联出处：
  - PDF页码：2505-2506、2522
  - 章节标题：`097 Chapter 2.01` / `098 Chapter 2.04`
  - 关键词：`天狼洞`、`山頂`、`通らずに`
  - 日文短摘录：`山頂からは一番長い洞窟` / `〈天狼洞〉を通らずに山頂へたどり着く`

## 后续描写展现出“桃之仙境”的生活设施与景观
- 克拉斯提线里，`白桃庙`内部已经不再只是危机背景，而出现了广阔厨房、灯火、买菜用布包、以及季节不限的桃花与桃实。
- 文中还直接把那里称作“桃之仙境”。
- 这说明它至少在某个时点是一个带有稳定居住、给养和礼仪生活感的仙境据点。
- 关联出处：
  - PDF页码：2501-2502、2529
  - 章节标题：`097 Chapter 2.01` / `098 Chapter 2.05`
  - 关键词：`厨房`、`桃の仙境`、`桃の花`
  - 日文短摘录：`〈白桃廟〉の厨房` / `ここは桃の仙境` / `桃の花も実も豊富`

## 庙中的日常维护、接待与生活秩序相当具体
- 花貂的回忆把`白桃庙`的日常写得更细了：这里并非纯观念上的仙宫，而有长期维护的石阶、被打扫的庙宇、会来参拜的客人、以及以硬木果为主的朴素饮食。
- 这使`白桃庙`不仅是神秘仙境，也是一处被长期照料、具备庙务节奏的具体生活空间。
- 关联出处：
  - PDF页码：2600
  - 章节标题：`101 Chapter 4.01`
  - 关键词：`石階段`、`参拝客`、`木の実`
  - 日文短摘录：`廟を掃き清め` / `石階段に水を打ち` / `参拝客を迎える`

## 至少栖居着天宫官吏与貂人族一类本地住民
- 克拉斯提观察到，庙里除了花貂之外，还有许多自称`天宮の官吏`、形态近似精灵或小兽的本地存在。
- 花貂本人则能在少女与小兽形态间变化，并作为庙中活动成员长期出现。
- 这说明`白桃庙`并不是空壳圣地，而是有自己地方住民层的仙境据点。
- 关联出处：
  - PDF页码：2503-2504、2530
  - 章节标题：`097 Chapter 2.01` / `098 Chapter 2.05`
  - 关键词：`天宮の官吏`、`貂人族`、`花貂`
  - 日文短摘录：`本人たちは天宮の官吏だといっている` / `貂人族` / `花貂`

## 当前材料存在“危机状态 / 日常状态”并存，必须并记
- 一方面，求援段落明确写了它被`魔人`制压。
- 另一方面，克拉斯提线又明确描写了庙内仍有日常生活、供食与住民活动。
- 目前更稳妥的整理方式，不是武断断言“已经解放”或“求援是误报”，而是保留为不同阶段或不同视角下的并行材料，等待后文进一步澄清。

## 阶段变化
- 定位阶段：先被确认为`仙境`类别下的具名山顶据点。
- 危机阶段：求援材料里又明确写出其一度遭`魔人`制压。
- 生活阶段：后文再补出厨房、庭园、参拜客、`天吏`与貂人族等稳定日常空间描写。
- 这几层必须并列保留，不能把“危机状态”与“日常状态”硬并成单一时点结论。

## 对中原服的参考价值
- `白桃庙`对你很有用，因为它把“中原仙境”从抽象类别推进成了可写的具体地点：
  - 有山顶位置与路线门槛
  - 有桃花、桃实、庭园与厨房等生活化细节
  - 有本地官吏与住民层
  - 也有被外敌攻陷的脆弱性
