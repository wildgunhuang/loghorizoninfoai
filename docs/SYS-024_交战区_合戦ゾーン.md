# SYS-024 交战区（合戦ゾーン）

- 分类：系统与制度 / 服务器特色 PVP
- 适用范围：特定服务器实现形式，底层规则可能共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`交战区`是中原服务器`公会战争`制度的核心实施空间。当前材料已经足够确认：它不是普通野外`PK`，也不是单纯“大家在某片地图里乱战”，而是把公会间战斗、`公会点数`、城市领主资格和后续大片区域支配权串起来的制度化战场。

## 制度性质
- 最稳妥的定义是：`交战区`属于中原服务器为`公会战争`设置的专用战斗区。
- 因此，它和一般`PK`的区别不在于“是否会死人”，而在于它本身就带着明确的制度目标和后续统治后果。

## 不是无限制战争系统
- 文本明确说明，中原服务器虽然鼓励`公会战争`，但并不是“无限制战争系统”。
- 真正承载大规模战斗的，是各地专门设置的`交战区`。
- 这条边界很重要，因为它说明中原的战争化虽然严重，但仍是被特定制度框住的，而不是整张地图任何地方都同等有效。

  - 日文短摘录：`無制限の戦争システムではない` / `各地につくられた合戦ゾーン`

## 与公会点数直接相连
- 在`交战区`中，公会交战结果会直接转化为`公会点数`。
- 文本同时强调，虽然`公会点数`不只一种取得方式，但“公会间直接斗争”才是中原服务器设计上真正重要的部分。
- 这说明中原服权力结构并不主要建在治理与生产之上，而是被制度化战斗强力牵引。
- 关联出处：
  - PDF页码：2483
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`ギルドポイント`、`直接闘争`
  - 日文短摘录：`勝利側にはギルドポイントが手に入る` / `直接闘争があることが重要`

## 与城市领主资格相连
- 文本进一步写明，只要积累足够`公会点数`，公会就能成为战场附近城市的领主。
- 这一点甚至覆盖`燕都`、`大都`这类玩家都市。
- 这让`交战区`不再只是战斗玩法，而是与城市统治权直接挂钩的制度接口。
- 关联出处：
  - PDF页码：2483
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`領主`、`燕都`、`大都`
  - 日文短摘录：`戦場近くの都市の領主になれる` / `燕都や大都といったプレイヤータウンでさえ`

## 与封禅大典的关系
- 当前材料最清楚的结构是：
  - `交战区`负责提供持续战斗空间
  - `封禅大典`负责把既有竞争结果升级成更大范围的支配权改写
- 因此，这两条词条不应互相替代。
- 更稳妥的理解是：`交战区`是持续战场，`封禅大典`是高潮接口。
- 关联出处：

## 运营设计上的作用
- 正文直接给出了这套制度的设计背景：中原地区过于广大，运营方难以持续手工填满海量事件，于是让玩家之间的竞争本身成为可反复生成的内容。
- 因为存在攻防、资源蒐集、扩军和持续防守，`交战区`不会像普通任务那样在“通关后结束”。
- 这说明中原服很多秩序问题并不是玩家偶然玩坏的，而是运营策略主动制造出的长期生态。
- 关联出处：
  - PDF页码：2483-2484
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`プレイヤーにも負担`、`終端が発生しない`
  - 日文短摘录：`プレイヤーにも負担してもらえないか` / `終端が発生しない`

## 大灾害后的延续
- 书中明确指出，这套原本属于游戏的系统，在现实化后的`赛尔迪希亚`里仍然有效。
- 更严重的是，中原服务器里大量`冒险者`仍按这套逻辑行动。
- 因而`交战区`在灾后不再只是玩法，而成了现实战乱、领主化与住民逃散的制度来源之一。
- 关联出处：
  - PDF页码：2484-2485
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`まだ生きている`、`圧倒的多数`
  - 日文短摘录：`そのシステムはまだ生きている` / `圧倒的多数がそうである`

## 与日本服的差异
- 当前最稳妥的差异整理有两条：
  - 日本服更突出城市自治与多公会协商。
  - 中原服更突出制度化公会战争、领主化与战区边境秩序。
- 因此，日本服经验可以参考，但不能拿来直接覆盖中原服。

## 阶段变化
- 设计阶段：先被说明为中原服务器为`公会战争`设置的专用战斗区。
- 统治阶段：很快补出它与`公会点数`、城市领主资格和玩家都市支配权直接相连。
- 现实化阶段：大灾害后，这套制度继续存活并转化成真实战乱、流民化与边境秩序问题。
- 这几层都应并列记录，不能把灾后乱世只写成玩家偶然失控。

## 当前未知或仍待补证的部分
- 以下内容目前仍不能写死：
  - 中原服务器全部`交战区`的数量与分布
  - 各战区的固定规模、默认人数和战斗编成
  - `公会点数`的完整结算公式
  - 不同玩家都市周边战区的具体差异
- 这些内容后续若出现，应继续回填本页。

## 对中原服的参考价值
- 这条设定几乎可以直接作为你写中原服政治生态的骨架：
  - 为什么会有战区边境城市
  - 为什么会出现强公会领主化
  - 为什么流民、大地人和中小公会会不断迁移
  - 为什么“停止战争”在中原服会比日本服更难
