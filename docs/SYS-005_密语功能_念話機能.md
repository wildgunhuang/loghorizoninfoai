# SYS-005 密语功能（念話機能）

- 分类：战斗系统 / 通讯
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`密语功能`是《幻境神话》中最重要的远距离私人通讯手段之一。大灾害后它仍可正常使用，但不是任意对象间的开放通讯，而是严格受`好友名单`限制；因此它既是远征、联络与情报回报的关键工具，也是玩家组织能力的现实边界之一。

## 功能性质
- 当前最稳妥的写法是：`密语功能`属于长距离私人通讯系统，而不是公共广播网络。
- 它能维持熟人之间的远程协调，却不能让陌生人即时搭起联络链。
- 这使它在大灾害后既显得非常重要，也暴露出非常明显的限制。

## 开篇已确认它在异世界中仍可使用
- 在开篇阶段，玩家已确认`密语功能`仍可在异世界中正常使用。
- 它可以让相隔较远的对象进行直接对话，因此很快成为远征、联络与情报回报的关键手段。
- 关联出处：
  - PDF页码：21-22, 210-211
  - 章节标题：`002`；`008`
  - 关键词：`念話機能`、`連絡`、`定期的`
  - 日文短摘录：`念話機能` / `連絡を入れる` / `すぐさま返事が返ってくる`

## 但通讯对象严格受好友名单限制
- `密语功能`并不能联系任意玩家，只能对`好友名单`中已登记的对象使用。
- 而`好友名单`的登记，又要求双方现实地面对面接触后才能完成。
- 这使得远征救援、跨城合作、临时结识的新对象联络都受到很强限制。
- 关联出处：
  - PDF页码：211-212
  - 章节标题：`008`
  - 关键词：`フレンド・リスト`、`登録`、`実際に対面`
  - 日文短摘录：`フレンド・リストに登録した相手にしか` / `目の前で実際に対面している相手に対してしか`

## 大灾害后其战略价值被进一步放大
- 在传送门停摆、回城受限、`妖精环`风险上升之后，`密语功能`的重要性被进一步放大。
- 但它不是万能通讯网，因为陌生人之间无法靠它立即建立协作链路。
- 因此它的价值与局限是同时成立的。

## 阶段变化
- 功能阶段：先确认它在异世界中仍然存在。
- 限制阶段：很快补出它只能在`好友名单`范围内使用，且登记要求面对面接触。
- 战略阶段：在跨城移动受阻的背景下，它进一步成为远程组织的重要命脉。
- 这几层都应保留，不能只写成“还能打电话”式的扁平结论。

## 对中原服的参考价值
- 对中原服创作而言，这意味着远距离组织能力仍然受制于旧有社交关系与线下会面条件。
- 可直接继承的，是以下底层逻辑：
  - 远距离私人通讯仍存在
  - 陌生人之间不能立即用它建立协作
  - 跨城组织与远征网络会因此天然偏向熟人、旧公会与既有联盟
- 这对写中原服大范围势力协调、边境救援和跨区战争都很关键。
