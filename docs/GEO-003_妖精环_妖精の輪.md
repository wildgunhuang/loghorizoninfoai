# GEO-003 妖精环（妖精の輪）

- 分类：场景与地理 / 交通系统
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`妖精环`是一种野外转送装置，掌握其位置与使用规律能显著缩短跨区域移动时间。大灾害前，玩家可依赖攻略资料把它当作高效交通网络使用；大灾害后，原本外包给攻略站的时间表知识断裂，使它从“便利交通点”迅速转变为高风险基础设施问题。

## 设施性质
- 当前最稳妥的写法是：`妖精环`属于野外转送网络节点，而不是稳定、固定输出到单一点位的简单传送门。
- 它的关键不只在“能转送”，而在“连接规律复杂且依赖时序知识”。
- 因此本页要同时记录其交通价值与知识门槛。

## 开篇已确认它是能缩短移动时间的转送装置
- 开篇系统说明中就已提到，`妖精环`是能显著压缩移动时间的转送装置。
- 这说明它原本就是世界交通网络中的重要组成部分。
- 关联出处：

  - 章节标题：`002`

## 后续补出其自然魔法阵式结构
- 后续明确写到，`妖精环`主要分布在原野区域，是由巨大石块或蘑菇围成的天然魔法阵式转移装置。
- 这使它在视觉上也更接近“世界原生地貌中的交通节点”，而不是人工室内设备。
- 关联出处：
  - PDF页码：175-176

  - 关键词：`フィールド`、`巨大な石やキノコ`
  - 日文短摘录：`フィールドにある転移装置` / `巨大な石やキノコが円上に配置`

## 连接规律依赖复杂时序条件
- 文中进一步说明，`妖精环`会依照月相等复杂条件与另一处`妖精环`接通。
- 因此若没有正确的时间表或攻略资料，使用风险很高。
- 在大灾害之前，玩家可以依靠攻略站提供的时间表高效穿行各地；大灾害之后，这种交通方式立刻变得危险。
- 关联出处：
  - PDF页码：175-176
  - 章节标题：`008`
  - 关键词：`月の満ち欠け`、`タイムテーブル`
  - 日文短摘录：`月の満ち欠けに影響` / `タイムテーブル`

## 大灾害后它从便利交通点变成公共事业问题
- 到后期，`妖精环`时刻表的重新确认已经被视作灾后`冒险者`的重要课题，因为没有可靠时刻表，连城市之间的安全移动都无法保证。
- 文中还指出，这项调查一度由`诚信`承担，而它几乎成了当时`圆桌会议`仅存的持续性公共事业；但由于人手与条件限制，调查并未成功完成。
- 这说明`妖精环`在异世界化后不只是交通点位知识，而是直接关系到城市间联络、远征和公共治理的基础设施问题。
- 关联出处：

  - 关键词：`安全確保`、`タイムテーブル確定`、`ホネスティ`、`公共事業`
  - 日文短摘录：`安全確保の保証ではなく` / `タイムテーブル確定` / `〈ホネスティ〉の奮闘むなしく` / `継続的な公共事業`

## 阶段变化
- 交通阶段：先被认知为能大幅缩短移动时间的转送设施。
- 结构阶段：后文补出其自然魔法阵式外观与时序连接规律。
- 公共事业阶段：大灾害后又被提升为城市联络和公共治理层面的基础设施问题。
- 这几层都应并列保留，不能只把它写成“野外传送门”。

## 对中原服的参考价值
- `妖精环`是很适合中原服直接沿用的底层交通框架之一。
- 可直接继承的是：
  - 野外存在转送网络节点
  - 节点连接依赖时序或其他复杂规律
  - 大灾害后，失去攻略资料会使交通网络失稳
  - 时刻表调查可以成为公共事业与城市治理议题
- 中原服可以本地化节点地貌、命名与分布，但不宜轻易取消其“知识门槛型交通”特征。

## 当前未知或仍待补证的部分
- `妖精环`全网结构、覆盖范围与跨服务器是否一致，当前尚未全局汇总。
- 时序规律是否只受月相影响，还是还存在其他变量，当前材料仍待补证。
