# ITM-003 六道轮回（六道輪廻）

- 分类：道具与装备 / 特殊武器
- 适用范围：特定服务器实现形式，底层规则可能共通
- 对中原服的可用性：可改造沿用
- 日本服依赖度：中

## 摘要
`六道輪廻`是后期明确出现的特殊刀剑。就当前文本而言，它最关键的意义不在于一般战斗力，而在于它被直接说明为能够“杀死冒险者”，并被使用者理解为不会把死者送往`大神殿`。这使它成为“冒险者常规复活规则存在道具级例外”的关键证据。

## 基本性质
- 文本直接把`六道輪廻`说明为“在这个大和也没有第二把的、能够杀死冒险者的武器”。
- 这不是角色夸张比喻，而是紧跟在道具名称识别后的说明句，因此证据强度很高。
- 关联出处：
  - PDF页码：3251
  - 章节标题：`116`
  - 关键词：`六道輪廻`、`冒険者を殺せる武器`
  - 日文短摘录：`〈六道輪廻〉` / `〈冒険者〉を殺せる武器`

## 与大神殿复活规则的关系
- 持刀者`カズ彦`明确相信：被这把刀杀死的人会被送往“与其相称的世界”，至少不会前往`大神殿`。
- 这条信息分成两层理解更稳妥：
  - 可较稳确认：这把刀被视作绕开`大神殿`复活路径的特殊武器。
  - 仍需保留边界：其“会回到地球”的部分，当前主要来自`カズ彦`对风味文本的理解。
- 关联出处：
  - PDF页码：3251
  - 章节标题：`116`
  - 关键词：`ふさわしい世界`、`地球世界に戻れる`、`大神殿へは向かわない`
  - 日文短摘录：`ふさわしい世界へ送り込む` / `地球世界に戻れる` / `大神殿へは向かわない`

## 已确认使用案例：クオン
- 已有明确战例中，`カズ彦`用`六道輪廻`刺杀了`クオン`。
- `クオン`在死亡过程里表现为常见的虹色粒子化离脱，但文本没有描写其在`大神殿`复活，反而把这一段写成他“正在离开赛尔迪希亚世界”。
- 因此，这个案例至少能作为“文本正在认真讨论不经大神殿路线的死亡例外”之证据。
- 但现阶段仍不能仅凭这一战例就断言`クオン`已经被实证送回地球；更稳妥的整理方式，是写成“文本把他处理成离开赛尔迪希亚的案例”。
- 关联出处：
  - PDF页码：3250-3254
  - 章节标题：`116`
  - 关键词：`クオン`、`セルデシア世界を後にした`、`虹色`
  - 日文短摘录：`セルデシアで死につつある` / `新しい目覚め` / `セルデシア世界を後にした`

## 作为“后续例外”而非常规规则
- 这条道具不能拿来反推“冒险者死亡本来就不一定去大神殿”。
- 更稳妥的结构是：
  - 常规规则：`冒险者`通常在`神殿 / 大神殿`复活。
  - 后续例外：至少存在`六道輪廻`这类特殊武器，可切断这条常规路线。
- 这也正是它对中原服写作最有价值的地方：不是照抄这把刀，而是知道原作允许“常规不死规则被极罕见道具打破”。

## 阶段变化
- 在当前材料里，`六道轮回`首先作为具名特殊武器被识别出来。
- 随后它又被明确推进成“能够杀死冒险者、切断大神殿路线”的常规复活规则例外证据。
- 因此它的意义不是一般高攻击力武器，而是后期世界规则出现道具级破口的关键样本。

## 对中原服的参考价值
- 可参考的不是必须照搬“六道轮回”这把刀本身，而是这种结构：
  - 常规复活规则长期成立
  - 后期才暴露出极少数可绕开复活点的特殊装备或术式
  - 相关道具会立刻改写政治、暗杀、威慑与忠诚关系

## 当前未知或仍待补证的部分
- `六道轮回`的制造来源、使用条件、是否有次数限制与能否稳定把对象送回地球，当前都仍未坐实。
- `カズ彦`对风味文本的解读与这把刀真正机制之间是否完全一致，现阶段也仍应保留边界。
