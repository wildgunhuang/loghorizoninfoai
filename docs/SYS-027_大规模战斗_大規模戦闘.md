# SYS-027 大规模战斗（大規模戦闘）

- 分类：系统与世界机制 / 战斗系统
- 适用范围：高概率全服共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`大规模战斗`是《幻境神话》中面向多人协同攻略的大型战斗内容。当前材料能够稳定确认：它不是单纯“人多打 Boss”，而是高度依赖情报、分工、补给、消耗品、仇恨管理和场地处理的系统性战斗模式。对中原服创作而言，这条设定非常关键，因为它直接决定了公会组织力、物资能力与战术经验的价值。

## 内容位置
- `大规模战斗`并不是灾后才出现的临时叫法，而是《幻境神话》时代就存在的正式内容类别。
- `xlsx` 中对 `大規模戦闘` 有 `大规模战斗 / 大规模战役 / 大型副本` 等不同版本记录；本项目正文统一采用更中性的 `大规模战斗`。

## 历史上以五十级为起点
- 城惠明确说明，《幻境神话》最初的等级上限曾经是 `50`，最早期的大规模战斗就是围绕这个等级带设计的。
- 后来即便等级上限不断提高，大规模战斗的内容分布仍是从 `50` 级附近开始展开。
- 关联出处：

## 会随资料片按十级梯度扩充
- 后文进一步写明：`50` 级以上的大规模战斗内容，会按 `10` 级为一档不断增加。
- 这些挑战并不是一次性全部存在，而是随着各次扩张包推出而追加。
- 这说明大规模战斗本身是《幻境神话》长期运营中的核心终局内容之一。

## 取胜高度依赖情报
- 城惠明确指出，大规模战斗要获胜，情报是不可或缺的。
- 他列举的关键情报包括：己方战力、装备、特技个性、补给状况、战场地形，以及敌方能力与行动模式。
- 更稳妥的写法不是“情报越多越好”，而是：在等级相近的挑战里，若没有足够情报，往往根本无法取胜。
- 关联出处：
  - PDF页码：3034-3036
  - 章节标题：`120 2.06`
  - 关键词：`情報が不可欠`、`補給`、`敵の能力`、`十分な情報`
  - 日文短摘录：`情報が不可欠だ` / `補給の有無` / `十分な情報が手に入らなければ勝てない`

## 核心不是硬拼，而是先压制敌人的优势土俵
- 城惠还用 `召唤之典灾` 战作为例子说明：真正关键的不是“闯到 Boss 面前”，而是先切断其最强的战斗条件。
- 这包括把本体与大量配下拉开、限制召唤怪数量、改变战场空间等。
- 因此，大规模战斗的底层逻辑更接近“先破解机制与战场结构，再打数值战”，而不是单纯堆高面板。
- 关联出处：
  - PDF页码：3034-3036
  - 章节标题：`120 2.06`
  - 关键词：`召喚の典災`、`囮`、`戦場を移動`、`数を制限`
  - 日文短摘录：`囮にして` / `戦場を移動させた` / `数を制限するため`

## 前期准备的核心是耐性装备与大量消耗品
- 城惠在购物教学里说明：一般 `秘宝级` 装备平均可削减同等级属性魔法伤害约 `25%`，但不同装备会有明显属性偏科。
- 若装备和群体防护不足，短时强效的通用消耗品就会变得非常关键；以 `属性对抗水药` 为例，约 `15` 分钟内可再减轻约 `15%` 的属性攻击伤害。
- 这说明大规模战斗前的配装与备货本身就是战术的一部分。
- 关联出处：
  - PDF页码：3043-3045
  - 章节标题：`120 3.01`
  - 关键词：`秘宝級`、`二十五パーセント`、`属性対抗水薬`、`十五分程度`
  - 日文短摘录：`平均二十五パーセント程度カット` / `属性対抗水薬` / `十五分程度`

## 标准准备会按库存思维囤积通用品
- 城惠把 `解毒药`、`苏生珠`、`暗视的软膏`、`浮游的咒符` 等都列为值得提前备货的项目，甚至按整箱单位购入。
- 他还给出了非常具体的数量基准：在二十四人规模下，这类通用消耗品每个成员至少要准备十个。
- 这说明大规模战斗准备不是“人手一份刚刚好”，而是要用库存思维应对长线消耗。
- 判定：`高概率效果`

## 职业差异会把准备继续细分
- 城惠同时明确把后续准备拆到职业层面：
  - 远程职业要准备更高规格的矢弹
  - `暗杀者 / 盗剑士` 要按本人流派准备毒
  - `战士职` 要带 `补修的粉`
  - `魔法攻击职` 要准备不同咒符
- 这说明大规模战斗的补给表并不是统一清单，而是“通用底座 + 职业专属包”。

## 现场连携的核心之一是仇恨交换
- 后文在秋叶原异变后的实战里，又把一种基础的大规模战斗连携写得非常具体。
- 原文说明，所谓 `シザース`，是两名前线单位像剪刀一样交错通过，交换各自需要拘束的目标；而真正完成连携的关键，则是随之发生的`仇恨交换`。
- 这说明大型战斗里的前线协同并不只是“各打各的”，而是需要受训过的仇恨交接技术。
- 关联出处：

## 个别职业奥义可短时间跨越等级差
- 另一段战斗还补出了一个很重要的边界：并非所有防御技能都能跨越巨大等级差，但`守护战士`的`坚石堡垒`是例外之一。
- 原文明确写出，这个技能在至少十秒内能令使用者无效化一切伤害，而其再使用规制时间则长达十分钟。
- 这类“绝对型防御窗口”在大规模战斗里意义极大，因为它能为情报确认、前线重整和救援争取硬性的安全时间。

## 二十四人战斗会形成清楚的岗位结构

- 其中 `冬弥` 被明确推到 `第一防卫役` 的位置，负责开场稳定仇恨；而不同小队则分别承担主坦支援、后卫魔法攻击、蘇生和状态对应等职责。
- 这说明大规模战斗里的“二十四人”并不是简单人数堆叠，而是已经形成了清楚的岗位结构。

## 指挥需要实时处理属性判明、蘇生与消耗品调用
- 同一场战斗还把指挥层的工作展示得更具体：火炎属性被确认后，要立即口头切换到红标签`耐性水药`；同时还要调度`蘇生呪文`、处理 `战斗不能` 与后背压力。
- 这再次说明大规模战斗并不是单纯拼输出，而是实时指挥、现场情报和消耗品反应的综合考验。

## 回复职密度足够时可边打边回收战斗不能者
- 后续同一段战斗又补出一个很实用的结构性信息：若队伍里`回复职`比例足够高，即使前线一度崩盘，也能由一部分人专门负责把`战斗不能`者拖到安全处连续复苏，另一部分人继续维持战线。
- 这不是抽象的“治疗重要”，而是明确说明大规模战斗的续战力，会受到回复职数量与分工密度的直接影响。

## 识破机制后可用退避与轮换把必败局转成可解局
- `失望と憧れの典災`战的后半段，又把“读懂机制后如何重整 raid”写得非常具体。
- 实莉明确判断：紫色光线是`精神属性攻击`，会抽取`MP`；应对方法不是硬吃，而是立刻切换紫标签`水药`提高抵抗力，并让一半`回复职`常驻安全圈外，轮流接手复苏与续战。
- 这说明大规模战斗的关键并不总是提高输出，很多时候更重要的是先识破攻击属性、有效射程和循环节奏，再用岗位轮换把原本的必败局改造成可持续战。

## 个别首领可能存在隐藏中枢
- 同一战斗还出现了更少见、但很值得保留的案例：外层大体内可能还藏有较低等级的内部核心。
- 文中由五十铃率先识破，`失望の典災`胸中还藏着另一体`35`级的`憧れの典災エレイヌス`；而实莉随即把它判断为这场 raid 的中枢。
- 目前不能把这直接推广成“所有 raid Boss 都有内核”的通用规则，但至少可以确认：大规模战斗存在“外层巨体只是壳，真正中枢藏在内部”的机制型案例。

## 阶段变化
- 游戏时代，`大规模战斗`首先作为正式终局内容存在，并以`50级`附近为早期起点，随后随资料片按十级梯度扩充。
- 在成熟攻略阶段，它逐渐发展出以情报、物资、职业分工、仇恨交接与库存准备为核心的标准化攻略思路。
- 到后期异常战例里，这套模式又被推进到更动态的层次：队伍需要一边识破`典灾`级敌人的属性与机制，一边实时调整岗位、消耗品和安全圈轮换。

## 对中原服的参考价值
- 这条设定基本可以直接继承到中原服。
- 你写中原服时，可以直接利用它支撑：
  - 公会间为何需要专门军需与后勤
  - 强会为何能靠情报和补给碾压弱会
  - 为什么远征前的准备本身就是战斗的一半
  - 为什么低等级新人踏入大规模战斗会是阶段性成长节点

## 当前未知或仍待补证的部分
- `大规模战斗`在不同服务器、本地资料片与不同年代的具体人数规模和默认配置，正文仍未完全系统化。
- `シザース`、隐藏中枢这类案例，是通用高阶常识还是少数经验者战例，现阶段也仍应保留边界。
- 物资准备是否存在更明确的职业模板清单或公会标准流程，目前还缺更完整的材料。
