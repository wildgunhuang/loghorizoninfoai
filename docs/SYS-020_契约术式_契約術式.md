# SYS-020 契约术式（契約術式）

- 分类：系统与制度 / 概念魔法
- 适用范围：大概率共通
- 对中原服的可用性：可改造沿用
- 日本服依赖度：中

## 摘要
`契约术式`是后期浮出水面的高层概念魔法之一。当前已能确认，它不是普通战斗技能，而是能够通过契约书、签署与破弃等行为，直接作用于身份、权利归属与区域规则；但其完整原理、发动资格与适用上限，在当前阅读范围内仍未完全公开。

## 术式性质
- 当前最稳妥的写法是：`契约术式`属于能干预社会制度与系统接口的高层概念魔法。
- 它在作品中的重要性，不在于单次战斗威力，而在于能够让“新的规则关系”被世界承认。
- 因此它应被视为“规则改写型术式”，而不是普通的职业技能延长线。

## 已知作用层级
- 现阶段至少能确认`契约术式`曾作用于三个层级：
  - 身份层：改写`大地人 / 冒险者`边界
  - 规则层：让世界承认新的规则关系
  - 制度层：改写区域一般权利与公共空间归属
- 这意味着它不是单一用途的魔法，而是一类能触及规则接口的高层术式。

- 同一段还把它与“西方”的`凝魔锻术`并列，说明至少在这一阶段，它已被视为能改变世界秩序的高层技术之一。

## 更早出现的前驱案例是身份改写
- 在术式名称正式浮出之前，文本已经给出一个更早的前驱案例。
- 城惠以`抄写师`高等级技能、带魔法效力的书面、以及最高级魔法素材制成契约书，试图在极短时间内把一名`大地人`转变为`冒险者`。
- 文中明确强调：
  - 这不是普通文书，而是带魔法效果与拘束力的书面。
  - 契约内容极端，连城惠自己都承认近乎“诈欺”。
  - 契约完成后，当事者会发生某种变质，成为与此前不同的存在。
- 关联出处：

  - 章节标题：`034`
  - 关键词：`契約書`、`筆写師`、`大地人`、`冒険者の身分`

## 世界会承认由契约生成的新规则
- 紧接着的描写又把这次契约推进到了更接近术式层的地步：签名燃起金色燐光，城惠明确感到“这异世界承认了新的规则”。
- 随后，当事者在死亡后被转送至`大神殿`复活，城惠也因此获得了`东之外记`这一新称号。
- 这说明`契约术式`的已确认能力边界，至少包括：
  - 借由书面契约重写身份与归属
  - 让世界承认新的规则关系
  - 触发与复活机制、称号变化相关的系统性后果
- 关联出处：
  - PDF页码：1017

## 已确认的完成方式是结成后再破弃
- 后文明确说明，这一术式并不是单纯“签署合同”就结束，而是“结成后再被引き裂かれる”才算完成。
- 文中还直接展示了效果：契约书被撕裂后化为光焰，随即出现黄金，说明这不是象征性仪式，而是会触发真实系统结果的术式行为。

  - 日文短摘录：`いま、契約が結ばれて、完全になりました` / `結ばれ、引き裂かれることで完全になります`

## 对区域权利与公共制度的干预
- 当前已知最重要的使用案例，是`圆桌会议`借由`契约术式`把秋叶原市街一般权利重新让渡给`大和服务器`本身，并通过破弃契约书把该状态固定化。
- 文中点名包含在内的对象有`公会厅`、`大圣堂`、`商业会馆`以及秋叶原市街的一般权利。
- 这说明`契约术式`至少可以被用来改写“区域可被玩家持续占有”的状态。
- 关联出处：
  - PDF页码：1729
  - 章节标题：`061`
  - 关键词：`アキバ市街一般の権利`、`ヤマトサーバーそのものに譲渡`、`永遠`
  - 日文短摘录：`アキバ市街一般の権利をヤマトサーバーそのものに譲渡します` / `契約書の破棄を持ってその譲渡を永遠とします`

## 作为制度修正工具的意义
- 后文进一步说明，`圆桌会议`并不满足于只处理秋叶原核心街区，而是试图把所有`原野区域`固定到“不会再被所有”的状态。
- 提案理由非常明确：
  - `区域购买`一旦和制裁能力挂钩，就会成为骚乱与战争的火种。
  - `大地人`无法使用`区域购买`系统，这使该制度本身带有结构性不公平。
  - 土地归属应尽量回到“人与人的契约”，而不是继续依赖系统式的`区域购买`。
- 关联出处：
  - PDF页码：1730-1731
  - 章节标题：`061`
  - 关键词：`ゾーン購入`、`騒乱の種`、`フィールドゾーン`、`大地人`、`戦争の火種`
  - 日文短摘录：`ゾーン購入は、騒乱の種` / `すべてのフィールドゾーンを“所有されることがない”状態に固定化したい` / `〈大地人〉はゾーン購入ができません`

## 阶段变化
- 前驱阶段：先出现的是“以魔法契约书改写身份”的单例。
- 命名阶段：到后文才正式以`契约术式`之名浮出。
- 制度阶段：再往后，它已经被用于处理城市一般权利与区域制度。
- 这三层都需要并存记录，不能把后期“大规模制度应用”反写成它一开始就公开成熟。

## 对中原服的参考价值
- 可直接继承的，不是秋叶原这次具体操作的对象名单，而是下面这些底层思路：
  - 高层概念魔法可以作用于社会制度与系统接口，而不只是战斗
  - 它还能在极端情形下作用于“身份”“归属”“规则承认”这类更深的层面
  - 玩家社会能够通过术式干预“权利”“所有”“公共空间”这类规则层问题
  - 若某种系统权限只向`冒险者`开放、而将`大地人`排除在外，它就可能成为长期战争风险
- 对中原服写作而言，`契约术式`特别适合用来解释“如何把服务器本地秩序从旧游戏系统接回现实政治”。

## 当前未知或仍待补证的部分
- 完整发动条件是什么，是否必须依赖特定职业、素材或书面媒介，当前仍未讲透。
- 谁能稳定使用`契约术式`，是少数天才可达，还是存在更广的学习路径，当前未知。
- 它能改写到什么层级，是否还能进一步作用于复活、职业系统或服务器边界，当前材料不足以写死。
- 目前已确认的实例都偏高难度、少数人掌握，不能直接外推成各服务器普遍可用的常规技术。
