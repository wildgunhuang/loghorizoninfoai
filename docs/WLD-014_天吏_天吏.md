# WLD-014 天吏（天吏）

- 分类：世界观 / 中原本地住民阶层
- 适用范围：特定服务器特有
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`天吏`是中原线里已经出现、但仍需要谨慎处理的一类本地称呼。现有材料能够稳定确认：它至少指向`白桃庙`等`仙境`中的本地官吏或侍从阶层，地位和力量明显低于`仙人 / 仙女`；他们会受到`魔人`威胁，也会承担地方协助、引路、杂务与庙宇管理功能。至于它究竟是不是稳定官阶、种族分类，还是更接近修行阶段名称，当前仍不能写死。

## 基本性质
- 现阶段最稳妥的理解方式，是把`天吏`视为中原仙境内部的一类地方官吏或侍从阶层称呼。
- 它已经明显不是单一人物名，也不只是文学形容，而是与地方住民、庙务和危机叙事稳定绑定的一层身份。

## 至少是仙境中的本地官吏阶层称呼
- 克拉斯提线里，庙中小兽般的住民被概括为“自称天宫官吏”的存在。
- 到花貂本人那里，这个称呼又被进一步收束为`天吏`。
- 因此当前最稳的判断是：`天吏`至少是中原仙境内部的一个本地官吏或侍从阶层称呼。
- 关联出处：
  - PDF页码：2503、2533
  - 章节标题：`097 Chapter 2.01` / `098 Chapter 2.05`
  - 关键词：`天宮の官吏`、`天吏`
  - 日文短摘录：`本人たちは天宮の官吏だといっている` / `だから天吏なのです`

## 对“官吏”本身的含义其实也说不清
- 花貂后续自述时，明确承认“貂人族是管理仙境的天之官吏”这句话不假。
- 但如果进一步追问“官吏到底是什么、具体做什么”，她其实并不知道。
- 这条证据很关键，因为它说明`天吏`不只是一个客观分类，也可能是本地住民自身半懂不懂继承下来的传统身份说法。
- 关联出处：
  - PDF页码：2599
  - 章节标题：`101 Chapter 4.01`
  - 关键词：`天の官吏`、`なんなのか`
  - 日文短摘录：`貂人族は仙境を管理する天の官吏である` / `官吏とはなんなのか`

## 在层级上低于仙人、仙女
- 花貂的自我说明里，明确把`天吏`与`仙人`区分开来：若能断绝欲望，就能“成为仙人”。
- 这至少说明在她的认知体系里，`天吏`不是`仙人`同义词，而更像其下位或未达成阶段。
- 这条材料来自角色认知，不宜当成绝对宇宙法则，但已足够说明两者并非同层。
- 判定：`高概率效果`
- 关联出处：
  - PDF页码：2533
  - 章节标题：`098 Chapter 2.05`
  - 关键词：`欲を断ち切れたら`、`仙人`
  - 日文短摘录：`欲を断ち切れたら仙人になれます`

## 在危机状态下属于相对脆弱的被保护对象
- 叶莲仙女求援时，特别提到“弱小的`天吏`”正数着命数，在恐惧中度日。
- 这说明一旦`仙境`被攻陷，`天吏`首先不是主战层，而更像需要被救援的庙中住民。
- 关联出处：
  - PDF页码：2500
  - 章节标题：`096 Chapter 2.01`
  - 关键词：`か弱き天吏`、`恐怖`
  - 日文短摘录：`か弱き天吏は` / `恐怖のうちで暮らす毎日`

## 会承担引路、杂务与地方协助工作
- 花貂在`白桃庙 / 狼君山`一带负责说明路线、买市集物资、协助下山，还会给克拉斯提端茶送点心、解释地方规矩。
- 如果把她视为`天吏`样本，那么当前可确认：`天吏`至少会承担相当多的地方服务与引导工作。
- 关联出处：
  - PDF页码：2505-2507、2529-2533
  - 章节标题：`097 Chapter 2.01` / `098 Chapter 2.05`
  - 关键词：`狼君山`、`協力します`、`外食`
  - 日文短摘录：`〈狼君山〉は降りれませんよ` / `天の官吏としてご協力します` / `向こうについたら外食`

## 在狼君山仙境中长期以“管理人”方式生活
- 花貂回忆自己一族时，说她们自出生起就在`狼君山`的`仙境`里，作为那里的管理人生活。
- 她甚至补了一句：与其说是官吏，不如说更像寄居者。
- 接下来的日常内容也很具体，包括扫庙、给长石阶洒水、迎接偶尔来参拜的`朱桓`等人。
- 这说明`天吏`在地方现实里的工作，并不抽象，而是非常接近庙宇侍从与山中管理人。
- 关联出处：
  - PDF页码：2600
  - 章节标题：`101 Chapter 4.01`
  - 关键词：`管理人`、`石階段`、`朱桓`
  - 日文短摘录：`その管理人として暮らしてきた` / `官吏というよりも居候` / `朱桓ら参拝客を迎える`

## 名义上从属于天宫秩序，但未必理解高层命令体系
- 当叶莲仙女后续搬出`昆仑`、`西王母`和“资格者”这些高层话语时，花貂这个自认是“天之官吏”的角色反而完全听不懂。
- 这说明`天吏`虽然名义上属于天宫秩序，但至少在地方层面，很多成员并不真正参与或理解上位决策体系。
- 关联出处：
  - PDF页码：2611
  - 章节标题：`101 Chapter 4.04`
  - 关键词：`天の官吏`、`わからなかった`
  - 日文短摘录：`天の官吏たる花貂にもわからなかった`

## 与貂人族等地方住民形态可能存在重叠，但不能直接等同
- 目前唯一明确自称`天吏`的人物样本是花貂，而她又属于`貂人族`。
- 但仅凭这一例，还不能写成“`天吏` = `貂人族`”。
- 更稳妥的写法是：`天吏`与某些地方种族或灵性住民身份可能重叠，但当前证据不足以把两者直接并号。

## 当前不能写死它是“官职 / 种族 / 修行阶段”中的哪一种
- 现有材料同时出现了：
  - `天宮の官吏`这种官职式说法
  - `か弱き天吏`这种住民层说法
  - `欲を断ち切れたら仙人になれます`这种修行阶段式说法
- 因而现阶段不能简单把`天吏`定死成单一类别。
- 更稳妥的是把它作为“中原仙境中的本地官吏 / 侍从阶层总称，兼具修行色彩”的待补充词条。

## 阶段变化
- `天吏`最初浮出时，首先以`白桃庙`与`仙境`中的脆弱本地住民身份出现，重点落在其“需要被救援”的位置。
- 随着花貂的日常叙述展开，这一称呼又进一步被具体化为地方官吏、庙宇管理人、引路者与杂务承担者，现实职能变得更清楚。
- 到后续高层话语出现时，文本又暴露出一个新边界：即便自称“天之官吏”，地方`天吏`也未必真正理解`昆仑`、`西王母`与上位法统体系，说明它更多是地方层身份，而非高层统治阶级。

## 对中原服的参考价值
- `天吏`对你很有用，因为它为中原仙境提供了一个比“仙人 / 仙女”更接地的地方阶层：
  - 可以是庙宇与仙境中的常驻小官吏
  - 可以承担杂务、接待、引路、传令和地方管理
  - 危机来临时又会立刻暴露出其脆弱性

## 当前未知或仍待补证的部分
- `天吏`究竟是不是正式官阶、种族标签、修行阶段，还是三者兼具，目前仍不能坐实。
- 除花貂及其同类外，是否还存在别种形态的`天吏`样本，正文也还未展开。
- 它与`古代种`、`仙人 / 仙女`、`西王母`之间的法统关系，目前仍缺更系统的说明。
