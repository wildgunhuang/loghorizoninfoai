# JPN-WLD-001 自由都市同盟伊斯塔尔（自由都市同盟イースタル）

- 分类：世界观 / 日本服地方政体
- 适用范围：日本服特有
- 对中原服的可用性：高参考价值
- 日本服依赖度：高

## 摘要
`自由都市同盟伊斯塔尔`是日本服东日本地区的大地人政治联合体。它不是中原服可以直接照搬的国家名称或版图，但它作为“大地人封建同盟如何统治土地、征税、与冒险者建立关系”的样本，非常值得参考。

## 基本性质
- 文中说明，`自由都市同盟伊斯塔尔`由二十余个势力结盟构成，用以对抗西方的`神圣皇国威斯特兰迪`。
- 其统治结构是典型的封建贵族制，各领主持有城塞都市、周边农地与若干村落，并在古宫廷中召开政治会议。

## 人口与势力规模
- 文中估计，日本服务器范围内的大地人约有两百万前后；伊斯塔尔只是东日本部分的联合体。
- 单个领地的人口未必极大，但因为各地分散，仍形成了完整的贵族统治与会议结构。
- 与之相比，聚集大量冒险者的秋叶原在人口规模上已是极强势力。

## 税与保护逻辑
- 文中把领主的基本逻辑说得很清楚：领主以保护领民与领地安全为职责，并以此为前提征收税。
- 但由于这个世界距离成本高、机动兵力稀缺，许多偏远村落事实上处在“想被保护却得不到充分保护”的状态。
- 这使大地人政体天然存在“兵力不足但税基欲望仍在”的结构性张力。
- 关联出处：
  - PDF页码：607-608
  - 章节标题：`022`
  - 关键词：`税`、`保護`、`兵力を派遣できない`
  - 日文短摘录：`税を受け取っている` / `保護を受けたい` / `兵力を派遣できない`

## 与冒险者的关系
- 文中进一步指出，正因为常备兵力有限，大地人贵族长期以来就会把护卫、讨伐等任务委托给`冒险者`。
- 换言之，许多任务之所以存在，本来就是大地人统治结构与兵力短缺共同催生出的外包机制。
- 这对中原服写作很重要，因为它说明“任务”和“冒险者政治地位”并不是割裂的。
- 关联出处：
  - PDF页码：608
  - 章节标题：`022`
  - 关键词：`クエスト`、`護衛`、`殲滅`、`委託`
  - 日文短摘录：`莫大な数のクエスト` / `護衛` / `殲滅` / `委託してしまう`

## 阶段变化
- 在当前材料里，`自由都市同盟伊斯塔尔`首先被写成与西方`神圣皇国威斯特兰迪`对峙的东日本大地人封建同盟。
- 随着秋叶原与大地人政体关系被展开，它又进一步显出“人口规模庞大但兵力不足、需要依赖冒险者任务委托”的结构性特征。
- 因此这条词条当前的重点，不在戏剧性政争，而在“地方封建同盟如何与冒险者社会相互嵌合”的制度结构。

## 对中原服的参考价值
- 可参考的不是伊斯塔尔这套日本服地方名号，而是这些结构：
  - 大地人地方政体本身拥有财政、领地与封建义务
  - 远方村镇的安全缺口会催生冒险者委托体系
  - 冒险者人口集中城市会对既有大地人政体造成力量再平衡

## 当前未知或仍待补证的部分
- `伊斯塔尔`更完整的内部贵族层级、会议制度与常设军事组织，当前正文仍未系统展开。
- 它与秋叶原之间更详细的正式外交与日常行政接口，现阶段也还缺少足够稳定的出处链。
- 在更后期时间线上，这个同盟如何应对`冒险者`力量上升，当前词条仍需继续回填。
