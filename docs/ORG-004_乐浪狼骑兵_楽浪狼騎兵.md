# ORG-004 乐浪狼骑兵（楽浪狼騎兵）

- 分类：组织与制度 / 公会
- 适用范围：特定服务器特有
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`乐浪狼骑兵`是当前中原线里最重要的地方公会样本之一。它既不是中原霸权大公会，也不是单纯游荡骑兵团，而是以`草原之都`为据点、以狼骑体系闻名、在长期战乱中承担边境护卫、收容与联合防线职能的地方势力。

## 组织性质
- 目前最稳妥的定义是：`乐浪狼骑兵`属于中原服边境地方公会。
- 它的核心特征不是抽象的“会骑狼”，而是狼骑体系已经与根据地设施、远征、宿营和地方秩序维护结合在一起。

## 标志性特征：狼骑
- 正文直接说，`乐浪狼骑兵`以乘狼闻名。
- 春翠还把“中原服务器里一般会骑狼”和“`乐浪狼骑兵`以狼骑著称”放在同一段里说明，这说明狼骑不是个别成员爱好，而是公会的识别标记。
- 关联出处：
  - PDF页码：2476
  - 章节标题：`096 Chapter 1.03`
  - 关键词：`乗用狼召喚の笛`、`有名`
  - 日文短摘录：`狼に乗ることで有名`

## 根据地
- 当前材料已能确认，`草原之都`就是其本拠地与总部所在。
- 因此它并不是流动团体，而是有稳定后方、城市依托和固定设施的地方组织。
- 关联出处：
  - PDF页码：2475, 2480
  - 章节标题：`096 Chapter 1.03`；`096 Chapter 1.04`
  - 关键词：`本拠地`、`本部`
  - 日文短摘录：`ギルドの本拠地` / `街の本部`

## 设施与生活形态
- 朱桓回到总部时，文本把相关设施写得很具体：
  - 狼由`召唤笛`召出
  - 高位笛子可做到近似常时召唤
  - 成员会与狼共同生活
  - 本部专门设有狼舎与多栋宿舍
- 这说明狼骑体系在异世界化后已经设施化、生活化，而不是单纯战斗功能。
- 关联出处：
  - PDF页码：2480
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`召喚笛`、`狼舎`
  - 日文短摘录：`狼舎をもつ本部` / `常時召喚`

## 装备与生产基础
- 春翠明确提到，公会内部存在能制作狼类召唤笛的职人。
- 文本还说明，不同召唤笛有大量等阶与种类差异，高级成员常需靠任务和素材获取更高等级骑乘配置。
- 这让`乐浪狼骑兵`不只是“会用骑乘道具”，而是与任务网、生产职和地方交通深度绑定的组织。
- 关联出处：
  - PDF页码：2477-2478
  - 章节标题：`096 Chapter 1.03`
  - 关键词：`職人`、`等級`、`天狼洞`
  - 日文短摘录：`職人もいます` / `等級も種類もたくさん` / `〈天狼洞〉`

## 历史位置：从主战区退到边境
- 文本明确写到，大灾害刚发生时，朱桓等人预感旧`公会战争`制度会在现实中失控，因此主动离开`大都`，退到`草原之都`。
- 这使`乐浪狼骑兵`与纯粹扩张型大战争公会明显不同。
- 它更像主动退出主战场、在边境保存自治与地方秩序的中小公会。
- 关联出处：
  - PDF页码：2484
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`大都`、`離れ`、`逃れてきた`
  - 日文短摘录：`〈大都〉を離れて` / `〈草原の都〉まで逃れてきた`

## 对流民与逃散者的收容功能
- 书中明确写到，`草原之都`接纳了大量从中原内部逃出的`大地人`。
- 连公会中的双子成员也并非最初成员，而是在逃难路上“捡来的”。
- 因此，它在功能上不仅是战斗组织，也承担收容、护送与边境安置作用。
- 关联出处：
  - PDF页码：2485
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`拾った`、`逃げだしてきてる`
  - 日文短摘录：`途中で拾った` / `相当な数逃げだしてきてる`

## 寒地远征与边境机动能力
- 后续雪中行军段落补得更具体：
  - 即使是`冒险者`，草原寒夜里也仍需要火与热饮
  - 狼骑本身有活动极限与轮换问题
  - 成员会使用耐寒刻印斗篷、野营火堆和不同持续时间的召唤狼配置
- 这说明它是真正具备边地远征与宿营能力的地方骑兵公会。
- 关联出处：
  - PDF页码：2568-2569
  - 章节标题：`099 Chapter 3.01`
  - 关键词：`熱い飲み物`、`火`、`活動限界`
  - 日文短摘录：`熱い飲み物` / `火が必要だ` / `活動限界`

## 与草原之都的共同体关系
- 当红王派大军逼近时，朱桓一侧不愿撤离的理由已不只是公会面子，而是因为城里有大量`大地人`与成员家属。
- 这说明`乐浪狼骑兵`已与`草原之都`形成双向依存，不再是随时可迁移的纯机动团体。
- 关联出处：
  - PDF页码：2570
  - 章节标题：`099 Chapter 3.01`
  - 关键词：`大地人`、`嫁さん`
  - 日文短摘录：`〈大地人〉も大勢いる` / `嫁さんを作ったやつ`

## 指挥结构的已知线索
- 到`狼君山`战线，朱桓被直接称作`乐浪狼骑兵`的`万骑长`。
- 现阶段还不能据此写死完整军制，但至少能确认：该公会在描述核心指挥者时，会使用强烈骑兵军团色彩的称号。
- 关联出处：
  - PDF页码：2633

  - 关键词：`楽浪狼騎兵`、`万騎長`
  - 日文短摘录：`〈楽浪狼騎兵〉の万騎長`

## 对外协同能力
- 后续战线说明中，`乐浪狼骑兵`的到场被视为能改善守势的重要增援。
- 这说明它不只擅长内部机动，也具备与外来冒险者和临时队伍拼接成联合防线的组织能力。
- 对中原边境故事而言，这是极有价值的地方势力样本。
- 关联出处：
  - PDF页码：2634-2636
  - 章节标题：`101 Chapter 4.08`
  - 关键词：`協力`、`戦線`
  - 日文短摘录：`〈楽浪狼騎兵〉` / `協力`

## 阶段变化
- 边境据点阶段：先被呈现为以`草原之都`为本部的狼骑地方公会。
- 生活设施阶段：很快补出狼舎、宿舍、召唤笛职人和长期宿营体系，说明其组织已深度地方化。
- 防线阶段：后文又明确其承担流民收容、边境护卫和联合防线增援功能。
- 这几层都应并列保留，不能只把它写成“会骑狼的战斗公会”。

## 对中原服的参考价值
- `乐浪狼骑兵`最值得借鉴的不是名字，而是结构：
  - 骑乘体系与地理环境深度绑定
  - 地方公会同时承担战斗、交通、边境护卫和流民安置功能
  - 中原服并不只有霸权大战争公会，也存在这种维持地方秩序的边境势力
