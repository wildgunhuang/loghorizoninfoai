# SYS-025 魂冥咒（魂冥呪）

- 分类：战斗系统 / 异常状态
- 适用范围：待考证
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`魂冥咒`是中原线后期明确出现的具名异常状态。现有材料能够稳定确认：它是一个高达`180级`的`坏状态`，会让克拉斯提的状态显示发生异常，进而被误认为`仙人 / 古代种`级存在；而到后续阶段，原文已经进一步明示它至少会封锁`HP`自然恢复、普通恢复手段、`密语`与服务器间移动，并伴随记忆丢失风险。与此同时，作品也保留了两个关键边界：原文并没有写“失去的记忆不能恢复”，而且对`典灾`夺取`HP`似乎仍是可行的绕开路径。

## 状态性质
- 现阶段最稳妥的理解方式，是把`魂冥咒`视为超规格复合型异常状态。
- 它不是单纯数值减益，而是同时影响状态显示、恢复链、通讯、跨服移动与记忆层面的异常。

## 已被明确称作具名坏状态
- 文中直接把`魂冥咒`称作一个有名字的`180级坏状态`。
- 这说明它不是普通叙述性不适，而是已经进入系统层可识别的状态异常。

## 强度远超一般冒险者常规范围
- 同一段还专门强调：即使在大灾害后的混沌世界里，`180级`也是离谱强度。
- 文中给出的比较基准是：`冒险者`可操作范围的上限大致从`90`开始，就算到`大和`也不过`100`。
- 这使`魂冥咒`一出现，就带上了“超出一般玩家常识”的性质。
- 关联出处：
  - PDF页码：2507
  - 章节标题：`097 Chapter 2.01`
  - 关键词：`一八〇レベル`、`九〇`、`一〇〇`
  - 日文短摘录：`一八〇レベルというのは法外な強度` / `上限は九〇から、ヤマトでさえ一〇〇`

## 已知影响之一是状态显示异常与身份误认
- 克拉斯提被花貂等人误认为`仙人`，文中给出的最直接解释就是“状态显示出现异常”。
- 叙述还进一步把这种异常归因到`魂冥咒`身上。
- 因此当前最稳的结论是：`魂冥咒`至少会影响他人读取或理解其状态信息，并进一步放大社会身份上的误认。
- 关联出处：
  - PDF页码：2505-2507
  - 章节标题：`097 Chapter 2.01`
  - 关键词：`ステータスの表示に異常`、`仙君`、`古来種`
  - 日文短摘录：`ステータスの表示に異常` / `仙君様` / `〈古来種〉などに間違えられている`

## 前期就已伴随记忆缺损疑点
- 在前一页，克拉斯提已经自觉“似乎有记忆缺损”。
- 他随后又把“坏状态是否是原因”列为自己的思考支线，但当场并没有得到结论。
- 因此，现阶段不能把`魂冥咒 = 失忆原因`写成既定事实，只能保留为强关联疑点。
- 判定：`待考证`

  - 日文短摘录：`記憶の欠損があるようだ` / `バッドステータスが原因では` / `〈魂冥呪〉`

## 后续已明确写出其复合封锁效果
- 到更后面的战斗与自我解析阶段，`魂冥咒`的效果已经不再只是“异常状态显示”或“记忆缺损猜测”，而是被明确写成一组系统层限制：
  - `HP`自然恢复停止
  - 无法通过恢复魔法、设施或道具回复
  - 无法使用`密语`
  - 无法进行服务器间移动
  - 会失去记忆
- 这说明`魂冥咒`是非常少见的复合型超规格异常状态。
- 关联出处：
  - PDF页码：2646-2650
  - 章节标题：`101 Chapter 4.09`
  - 关键词：`自然回復`、`回復魔法`、`施設`、`アイテム`、`念話`、`サーバー間移動`、`記憶`
  - 日文短摘录：`自然回復` / `回復魔法` / `施設` / `アイテム` / `念話` / `サーバー間移動` / `記憶`

## 后续战斗中出现了“以记忆为代价换取力量”的危险信号
- 再往后的战斗里，克拉斯提在绝境中明确产生了“捧出记忆也无所谓”的判断，并随之获得了`MP`恢复、`再使用规制时间`加速和身体能力提升。
- 这段材料本身不能直接证明“这一定是`魂冥咒`的效果”，因为原文没有明确下结论。
- 但把它和先前“记忆缺损”“坏状态是否是原因”的自我追问并列后，至少可以确认：`魂冥咒`与“记忆被当作代价或燃料”的危险链条高度接近。
- 判定：`待考证`

  - 日文短摘录：`捧げる記憶などいくらでもある` / `ＭＰを回復` / `再使用規制時間を加速`

## 原文并没有写死“失去的记忆不能恢复”
- 克拉斯提在重新审视这个坏状态时，明确抓住了一点：相关记述里并没有写“失去的记忆不可以取回”。
- 这使`魂冥咒`的后续处理边界变得很关键：它确实会造成记忆丧失风险，但作品并没有在当前时点把“不可逆”正式写死。

## 对典灾夺取HP可能是已知绕开路径之一
- 同一段里还给出了一个更操作性的例外：虽然`魂冥咒`封死了通常恢复路线，但“夺取`典灾`的`HP`”仍被明确保留为可行。
- 这意味着后续战斗中，克拉斯提并不是单纯“顶着诅咒硬撑”，而是在尝试围绕诅咒边界寻找可利用的缺口。

## 阶段变化
- 初次浮出时，`魂冥咒`首先以具名`180级坏状态`的形式出现，并伴随状态显示异常、身份误认与记忆缺损疑点。
- 中段战斗里，克拉斯提又出现了“以记忆换取力量”的危险表现，但原文尚未把这条链条完全写死为`魂冥咒`本体效果。
- 到后续明确说明阶段，正文已经把它推进为一种复合封锁异常：自然恢复、普通恢复、`密语`、服务器间移动与记忆层面都会受到影响，但可逆性和完整解除条件仍未完全明示。

## 对中原服的参考价值
- `魂冥咒`对你有价值的地方，不只是“有一个强力异常状态”。
- 更重要的是它把三层东西绑在一起了：
  - 超规格系统状态
  - 记忆或存在感异常的可能性
  - 中原社会语境下对身份的误读与神话化

## 当前未知或仍待补证的部分
- `魂冥咒`的完整成因、施加者与解除条件，当前仍未被系统化说明。
- 它是否必然导致记忆缺损，还是只在某些条件下才发生，目前也仍不能写死。
- “以记忆换取力量”的那组表现究竟是`魂冥咒`本身、战斗时的别种机制，还是多种异常叠加，正文仍缺最终确认。
