# WLD-005 大地人（大地人）

- 分类：世界观 / 居民群体
- 适用范围：全服共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`大地人`是《幻境神话》世界中原本就存在的非玩家住民群体。游戏时代通常被玩家当作`NPC`理解，但大灾害后，他们被明确表现为具备血肉、生活、权利与政治性的真实居民，因此不再只是设施附属物，而是整个世界社会结构的主体之一。

## 群体性质
- 当前最稳妥的写法是：`大地人`是赛尔迪希亚侧原生居民总类，而不是仅供玩家交互的功能角色。
- 在前期玩家视角里，他们先被延续性地理解成旧游戏中的`NPC`。
- 但后文不断补证，这种认识已不足以解释他们的生活、人口、权利与政治结构。

## 开篇时仍被当作旧游戏里的NPC理解
- 在开篇阶段，城惠等人首先仍以游戏时代的理解来认识`大地人`，即把他们视为原本负责卖货、提供服务、发布任务的非玩家角色。
- 他们常见于商店、`公会`登记所等设施，也会向玩家提供信息或任务。
- 关联出处：
  - PDF页码：42
  - 章节标题：`003`
  - 关键词：`ノンプレイヤーキャラクター`、`大地人`、`店員`、`クエスト`
  - 日文短摘录：`ゲーム内世界的には〈大地人〉` / `店員として様々な物を売っていたり` / `クエストをくれたりする`

## 城市秩序本来就由大地人承担
- 城市中的警备兵也属于`大地人`侧的秩序力量。
- 在战斗禁止区域中，若玩家发动攻击，警备兵会迅速介入；若攻击对象是警备兵本身，则会被视作对警备兵的反逆并当场讨伐。
- 开篇提到的强力警备兵等级可达`100`，说明至少在玩家城市里，`大地人`秩序机构并非可以随意挑战的背景装饰。
- 关联出处：
  - PDF页码：42-43
  - 章节标题：`003`
  - 关键词：`警備兵`、`牢屋`、`反逆`、`レベルで云えば１００`
  - 日文短摘录：`警備兵が飛んできて` / `反逆だと見なされて` / `レベルで云えば１００`

## 认知松动先于正式定义变化
- 开篇时主角仍常把`大地人`放在“原本的NPC”框架里理解。
- 但同一段叙述也已出现明显的认知松动：随着世界变得过于真实，玩家对“这只是游戏角色”的感受正在减弱。
- 这一层很重要，因为它说明“把大地人重新理解成真实居民”不是突然宣布，而是由体感先推动的。
- 关联出处：
  - PDF页码：42-43
  - 章节标题：`003`
  - 关键词：`ゲームという意識`、`五感`、`圧倒的にリアル`
  - 日文短摘录：`ゲームという意識が蒸発していく` / `圧倒的にリアル`

## 大灾害后被明确表现为血肉现实存在
- 后续明确确认：`大地人`在大灾害后不再只是有限脚本驱动的`NPC`，而是具备血肉、行动与对话能力的现实存在。
- 仅凭交谈，玩家有时甚至无法区分对方是玩家还是`大地人`。
- 这意味着“原本的任务提供者与店员”已经转化为真正的社会成员。
- 关联出处：
  - PDF页码：241-242
  - 章节标题：`009`
  - 关键词：`血肉を備えた実在`、`会話能力`、`行動能力`、`判らない`
  - 日文短摘录：`血肉を備えた実在の存在` / `会話能力も行動能力も` / `判らないこともしばしば`

## 生理与人口结构也被现实化
- 后续还指出，`大地人`现在也需要睡眠、休息与饮食。
- 文中推测，正因为他们开始需要轮班与生活维持，世界中的`大地人`数量才出现了明显增加。
- 这说明大灾害改变的不只是玩家，也改变了世界住民的人口结构与日常运转方式。
- 关联出处：
  - PDF页码：242-243
  - 章节标题：`009`
  - 关键词：`睡眠`、`休息`、`食事`、`数が爆発的に増えた`
  - 日文短摘录：`睡眠や休息、食事を必要` / `数が爆発的に増えた`
- 推断：中原服若照原作设定展开，地方城市、村镇与服务设施的人口规模，应默认比“纯游戏状态”更接近真实社会所需的人力配置。

## 灾后会成为掠夺与支配对象
- 由于`大地人`变得更像真实人类、数量也更多，他们开始成为恶性玩家掠夺、虐待与支配的对象。
- 这不是日本服专有底层规则，而是“大灾害后，大地人被现实化”所带来的普遍风险；日本服只是先在薄野这一场景里把问题暴露了出来。
- 关联出处：
  - PDF页码：240-243
  - 章节标题：`009`
  - 关键词：`搾取`、`虐待`、`娯楽としての人身売買`
  - 日文短摘录：`ノンプレイヤーキャラクターにも及んだ` / `虐待や搾取` / `娯楽としての人身売買`

## 后续被重新定义为世界主流居民
- 到圆桌会议前后，文中已经进一步明确：`大地人`不应再被当作没有人格的旧式`NPC`来处理，而应被视为这世界的主流居民群体。
- 更重要的是，`冒险者`并不是脱离`大地人`也能完整生活。银行、服务设施与大量社会运行都由`大地人`支撑；反过来，`大地人`却未必在同等程度上依赖`冒险者`。
- 这使“如何对待`大地人`”不再只是道德问题，而是世界稳定性与政治关系问题。
- 关联出处：
  - PDF页码：496-499
  - 章节标题：`017`
  - 关键词：`ノンプレイヤーキャラクターではありません`、`銀行`、`暮らせる`
  - 日文短摘录：`彼らはノンプレイヤーキャラクターではありません` / `銀行を始め` / `大地人のほうは僕ら抜きでも`

## 与冒险者可能形成长期制度冲突
- 文中还提出了一个关键危险：若`冒险者`继续把这个世界当成“可以随便来的游戏”，最终甚至可能与`大地人`社会发生不可逆的对立。
- 这并不是已经发生的既成事实，而是当时被认真提出的战略警告。
- 关联出处：
  - PDF页码：498-499
  - 章节标题：`017`
  - 关键词：`取り返しのつかないこと`、`戦争`
  - 日文短摘录：`取り返しのつかないことに` / `戦争の可能性`

## 区域购买制度会把大地人排除在外
- 到更后文，作品又补了一个非常关键的不平等来源：`大地人`无法使用`区域购买`系统。
- 这意味着，只要“可购买区域”继续作为现实治理工具存在，`大地人`就会在土地与公共空间秩序上被先天地排除在外。
- 因此，文中把`区域购买`进一步上升为可能诱发`冒险者`与`大地人`战争的制度问题，而不只是玩家内部的权力工具。
- 关联出处：
  - PDF页码：1731
  - 章节标题：`061`
  - 关键词：`大地人`、`ゾーン購入`、`異常`、`戦争の火種`

## 身份边界存在可改写的极端例外
- 后文还出现了一个极重要但必须谨慎处理的例外：至少存在通过高等级契约书与魔法素材，把`大地人`转变为`冒险者`的个案。
- 现阶段最稳妥的结论不是“所有`大地人`都能转成`冒险者`”，而是：
  - `大地人 / 冒险者` 的边界并非绝对不可改写
  - 这种改写需要极端条件，且伴随未知变质与高风险
- 关联出处：
  - PDF页码：1014-1017
  - 章节标题：`034`
  - 关键词：`大地人`、`冒険者の身分`、`契約書`
  - 日文短摘录：`〈冒険者〉の身分が含まれる` / `ルンデルハウスを〈冒険者〉にする` / `まったく違った存在になってしまう`

## 中原线进一步说明古代种属于大地人一支
- 后期中原线把`古代种`说明成`大地人`的一类，而非独立种族。
- 因此当前最稳的整体理解是：
  - `大地人`是更大的住民总类
  - `古代种`则是其中拥有超常力量、名望或守护职责的一支
- 这条区分对中原服尤其重要，因为当地会直接把这类存在称为`仙人 / 仙女`等。
- 关联出处：
  - PDF页码：2499
  - 章节标题：`096 Chapter 2.01`
  - 关键词：`古来種`、`大地人の一種`
  - 日文短摘录：`〈古来種〉は〈大地人〉の一種`

## 阶段变化
- 游戏残影阶段：先被当作旧游戏里的`NPC`。
- 现实化阶段：很快补出血肉、生活与人口扩张。
- 政治化阶段：再进一步被明确为世界主流居民、制度主体与潜在战争对象。
- 高层扩展阶段：中原线又补出`古代种`是其内部一支。
- 这几层必须同时保留，不能把“后期主流居民定义”写成开篇就被所有玩家彻底接受。

## 对中原服的参考价值
- `大地人`是中原服创作必须优先继承的核心设定之一。
- 日本服与中原服的差别更多应体现在地方政体、文化与势力结构，而不是是否存在`大地人`这一住民群体。
- 对中原服尤其重要的是：
  - 他们是现实社会主体，不是任务机
  - 他们可能与玩家合作、依存、冲突、对抗
  - 高位住民如`古代种 / 仙人`应视作这一总类内部的特殊层

## 当前未知或仍待补证的部分
- 各服务器`大地人`的人口规模、国家结构与制度细节差异，需要在本地词条中逐一追踪，不宜在本页一概而论。
- `大地人`与`冒险者`边界改写是否还有其他方法，当前未知。
- “数量爆发增加”究竟是生成补完、人口补全还是观察偏差，文本当前仍偏推测层。
