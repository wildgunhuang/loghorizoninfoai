# JPN-ORG-001 三日月同盟（三日月同盟）

- 分类：组织与制度 / 日本服专属组织
- 适用范围：日本服特有
- 对中原服的可用性：仅作参考
- 日本服依赖度：高

## 摘要
`三日月同盟`是秋叶原的一支小规模玩家公会，由`玛莉艾儿`担任核心领导者。它本身属于日本服本地组织，不能直接搬到中原服，但它所体现出的“小型互助型公会如何在大灾害初期保护成员、集中住宿、维持内部秩序”，对中原服创作具有参考价值。

## 组织定位
- 文中明确将`三日月同盟`归为小规模的`冒险支援型公会`。
- 它不是纯战斗公会，也不是纯生产公会，而是更偏向照顾成员日常、组织互助与提供缓冲空间的社区型组织。
- 关联出处：
  - PDF页码：61
  - 章节标题：`003`
  - 关键词：`三日月同盟`、`小規模`、`冒険サポート系ギルド`
  - 日文短摘录：`マリエール率いる〈三日月同盟〉` / `小規模な冒険サポート系ギルド`

## 初期应对方式
- 在大灾害发生后的极早期，`三日月同盟`倾向于把成员集中安置在公会厅中共同起居，以降低混乱与个体风险。
- 这说明小型公会在灾后最先发挥出来的功能，不一定是开荒战斗，而是住宿、互助、成员安抚与内部联络。
- 关联出处：
  - PDF页码：56-57
  - 章节标题：`003`
  - 关键词：`ギルドホール`、`寝泊まり`、`集団で`
  - 日文短摘录：`ギルドホールがありますし` / `しばらく寝泊まりは` / `集団で行なった方が良い`

## 人数与成员构成
- 在薄野救援前夕，文中提到`三日月同盟`成员约为`24`人。
- 公会中既有能承担战斗任务的成员，也有偏生产与城市活动的成员，但整体仍以“尚不成熟的年轻成员较多”为特征。
- 关联出处：
  - PDF页码：174, 179-180
  - 章节标题：`008`
  - 关键词：`２４人`、`ひよっこ`、`若手`
  - 日文短摘录：`いまはちょい増えて２４人` / `まだひよっこが多いやろ`

## 日本服专属情境
- `三日月同盟`的很多具体问题都绑定在日本服语境里，例如秋叶原据点、薄野救援、成员分布在日本服务器五大都市之间等。
- 因此它不适合当作中原服可直接继承的“设定模板”，但适合作为“小型善意公会”这一社会角色的参照样本。

## 阶段变化
- 初期阶段，`三日月同盟`首先被写成秋叶原的小规模`冒险支援型公会`。
- 大灾害早期，它又进一步显出“集中住宿、内部互助、安抚成员”的生存共同体功能。
- 到薄野救援线前后，它的角色再向前推进，成为会在成员分散异地时做出外出救援与本地留守分工判断的成熟小型公会样本。

## 对中原服的参考价值
- 可参考的不是名称、成员或地点，而是组织功能：
  - 小型互助型公会如何集中住宿
  - 如何维持成员信息联络
  - 如何在成员分散异地时决定是否救援
  - 如何在精锐外出时保留本地防守骨干

## 当前未知或仍待补证的部分
- `三日月同盟`在更后期的发展、扩员轨迹与内部制度，当前正文尚未完整回填。
- 它的财政来源、常态化任务分工与更具体的居住管理方式，现阶段也还不够清楚。
- 除`玛莉艾儿 / 荷丽艾塔 / 瑟拉拉`外，其他成员结构与长期角色分布仍需更多材料补证。
