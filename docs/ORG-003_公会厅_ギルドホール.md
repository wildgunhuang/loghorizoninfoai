# ORG-003 公会厅（ギルドホール）

- 分类：组织与制度 / 私有据点设施
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
`公会厅`是各个`公会`可以租借或购买的独立据点区域。它兼具居住、仓储、会合与内部管理空间的功能，是玩家组织真正“落地生活”的基础设施；与此同时，由于它本身就是可设置参数的私有区域，它也可能被用于拘束、压迫与内部暴力。

## 设施性质
- 当前最稳妥的写法是：`公会厅`属于`公会`私有据点区域，而不是公共行政设施。
- 它与`公会会馆`的区别必须明确：
  - `公会会馆`是公共手续与服务窗口
  - `公会厅`是单个`公会`实际持有和使用的区域

## 基本定义
- 文中明确说明，`公会厅`是独立的中规模区域，通常拥有数个房间，可作居住与事务空间使用。
- 规模较大的`公会`通常会在`公会会馆`租借或购置`公会厅`。
- 关联出处：
  - PDF页码：50-51
  - 章节标题：`003`
  - 关键词：`ギルドホール`、`中規模のゾーン`、`居住兼`、`会合`
  - 日文短摘录：`独立した中規模のゾーン` / `居住兼、事務用のスペース` / `会合に利用`

## 它同时承担仓储与集体生活功能
- `公会厅`可用于存放怪物掉落、素材、生产品等`公会`共享物资。
- 同时它也能被改造成长期居住与集体生活空间，在灾后初期尤其适合作为集中住宿点。
- 关联出处：
  - PDF页码：51, 57
  - 章节标题：`003`
  - 关键词：`戦利品`、`素材`、`寝泊まり`、`集団で`
  - 日文短摘录：`戦利品や素材` / `寝泊まりは` / `集団で行なった方が良い`

## 私有区域设置会放大持有者权力
- 作为被租借或购买的区域，`公会厅`的拥有者可以对出入限制与部分区域设置进行调整。
- 后文更明确显示，至少在某些`公会厅`内，区域持有者可以决定是否维持`禁止战斗区域`设定。
- 这意味着`公会厅`并不是天然安全屋，而是可能被善用，也可能被滥用的私有空间。
- 关联出处：
  - PDF页码：50-51, 471-472
  - 章节标题：`003`；`017`
  - 关键词：`出入りに制限`、`設定をいじる`、`戦闘行為を解禁`
  - 日文短摘录：`出入りに制限` / `ゾーンの設定をいじる` / `戦闘行為を解禁`

## 灾后善恶用途会明显分化
- 在善意`公会`手中，`公会厅`是住宿、互助与恢复秩序的据点。
- 在恶性`公会`手中，它也可能变成新人拘束、压迫与内部暴力的场所。
- 因此对中原服创作而言，重点不是“有没有`公会厅`”，而是“谁拥有它、如何设置它、用它来保护谁或压迫谁”。
- 关联出处：
  - PDF页码：56-57, 460-472
  - 章节标题：`003`；`017`
  - 关键词：`集団で`、`雑魚寝`、`軟禁`、`管理`
  - 日文短摘录：`集団で行なった方が良い` / `雑魚寝` / `軟禁状態`

## 阶段变化
- 据点阶段：先被说明为`公会`的居住兼事务空间。
- 权限阶段：很快补出它作为私有区域可调节出入与部分区域参数。
- 风险阶段：后文又明确显示，这些权限能被用来保护成员，也能被用来拘束与压迫。
- 这几层都应保留，不能只把`公会厅`写成中性建筑物。

## 对中原服的参考价值
- 可直接继承的部分：
  - `公会`拥有私有据点区域
  - 据点承担仓储、住宿、会议与内部管理功能
  - 区域设置权会放大`公会`的善意或恶意
- 这对写中原服边境公会、驻防公会、大型派系根据地和新人收容据点都很关键。

## 当前未知或仍待补证的部分
- 不同服务器、不同城市`公会厅`的维持成本、规模差异和参数权限上限，当前尚未全面汇总。
- 除`禁止战斗区域`开关与出入限制外，还能调整哪些区域参数，仍需继续补证。
