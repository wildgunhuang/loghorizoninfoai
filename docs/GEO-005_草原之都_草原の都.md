# GEO-005 草原之都（草原の都）

- 分类：场景与地理 / 都市
- 适用范围：特定服务器特有
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`草原之都`是后期中原线出现的重要城市节点。现有材料能够稳定确认：它位于`尤雷德`大陆中央交通线上，是从中国一侧理解时通往丝绸之路的起点之一，也是`中原`与`阿欧尔索伊`荒野交界处的边境城市。由于中原内部长期动荡，这里同时发展成了边境交通节点、`乐浪狼骑兵`的根据地，以及吸纳难民、商人和中小公会的缓冲城市。

## 城市性质
- 现阶段最稳妥的理解方式，是把`草原之都`视为中原外缘的边境综合节点，而不是单纯军城、单纯商埠或单纯避难所。
- 正文里关于它的描写同时涉及交通线、边境位置、公会据点、流民流入与共同体扎根。
- 这说明它的功能是复合型的。

## 是中央尤雷德的重要交通节点
- 文中把`草原之都`说明成`中央尤雷德`的终点。
- 如果从中国一侧理解，它又是正式丝绸之路的起点之一。
- 这让它不仅是地方城市，而是横贯大陆移动路线上的战略节点。

## 位于中原之外缘
- 同一段叙述明确说：从西往东看，`草原之都`再往前就是所谓的`中原`。
- 这意味着它并不是中原腹心，而是进入中原前的重要门槛城市。
- 文中还给出现实世界类比，大致相当于蒙古一带。

## 位于中原与阿欧尔索伊的边界
- 后文更明确补出其边界性质：`草原之都`位于绿色丰饶的`中原`与荒野`阿欧尔索伊`的交界。
- 这条设定很关键，因为它解释了为什么它既接得住中原流出的难民，也能维持边地型的独立生态。

## 是乐浪狼骑兵的根据地
- `草原之都`对旅人一行的重要性，还在于它是`乐浪狼骑兵`的本拠地。
- 对卡纳米一行来说，这里是继续向东前进的里程碑；对春翠来说，这里则是所属公会准备“归还战力”的回收点。
- 这说明此城并非单纯商旅中继站，而是带有公会军事与情报功能的据点。
- 关联出处：

## 会吸纳中原动荡中逃出的住民与冒险者
- 书中明确说，因为中原内部长期骚乱，大量`大地人`会拖家带口逃到这里。
- 逃来的人不只包括普通百姓，也包括商人、富农，以及部分`冒险者`。
- 这让`草原之都`具备了很强的“边境缓冲城 / 避难城”性质。
- 关联出处：
  - PDF页码：2484-2486

## 已与地方公会形成共同体关系
- 红王派逼近时，朱桓一侧明确提到：`草原之都`没有像样的防壁。
- 但他们仍不愿撤离，因为城里不仅有大量`大地人`，还已有成员家属与现实生活关系扎根于此。
- 这说明`草原之都`对`乐浪狼骑兵`而言，已经不只是临时据点，而是带有婚姻、生活和地方责任的共同体。

## 城市防御条件并不强
- 当前正文已明确写到，这座城市没有像样的防壁。
- 这使它的边境重要性带有明显脆弱性：它之所以能维持，不是因为天生难攻，而更多依赖地方势力与共同体自我支撑。
- 关联出处：

## 与狼君山、白桃庙一线距离不远
- 以剧情推进来看，`草原之都`并不是孤立于仙境线之外的俗世城市。
- `狼君山`、`天狼洞`、`白桃庙`等地都与它形成了实际行动半径内的山地网络。
- 这使它同时连接中原边境政治、公会战争与地方仙境危机三条线。
- 关联出处：
  - PDF页码：2475、2570-2572
  - 章节标题：`096 Chapter 1.03` / `099 Chapter 3.01`
  - 关键词：`本拠地`、`狼君山`
  - 日文短摘录：`本拠地` / `〈狼君山〉もまさにそうだ`

## 阶段变化
- 交通阶段：先被说明为中央尤雷德与丝绸之路方向的重要节点。
- 边境阶段：很快补出它位于中原与`阿欧尔索伊`的交界，是进入中原前的重要门槛城市。
- 缓冲阶段：后文又明确它吸纳难民、商人、中小公会，并与`乐浪狼骑兵`形成共同体关系。
- 战线阶段：再进一步，它与`狼君山 / 白桃庙 / 封禅大典`一线连成了现实战争与仙境危机的前沿。
- 这几层都应并列保留，不能只把它写成单一功能城市。

## 对中原服的参考价值
- `草原之都`对你写中原服非常有用，因为它天然能承载：
  - 边境贸易
  - 公会驻军
  - 难民流动
  - 中原内部战乱向外溢出的社会后果
  - 与山地仙境网络之间的往来关系
- 它提供的是一个很典型的“不是皇都、却高度关键”的地方城市模板。

## 当前未知或仍待补证的部分
- `草原之都`的正式治理结构、税收方式、常设守军与市政组织，正文还没有展开到可系统整编的程度。
- 它在游戏时代就已是大城市，还是大灾害后才因难民流入变得重要，现阶段也仍缺更早材料补证。
- 城中是否存在除`乐浪狼骑兵`之外的稳定大公会力量，目前亦未完全明示。
