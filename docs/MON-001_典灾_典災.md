# MON-001 典灾（典災）

- 分类：怪物与生物 / 特殊敌对存在
- 适用范围：日本服实现形式，底层规则可能共通
- 对中原服的可用性：可改造沿用
- 日本服依赖度：中

## 摘要
`典灾`是大灾害后逐渐浮出水面的特殊新型怪物类别。当前阅读范围内已能确认：它们并不只是普通高等级怪，而是具备更高知性、伪装、渗透、社会诱导或异常能力的敌对存在；但“典灾”是否覆盖全服、是否等同于所有灾后新怪物、其内部分类边界如何，目前仍不能下定论。

## 类别性质
- 现阶段最稳妥的理解方式，是把`典灾`视为大灾害后浮出的异常敌对类别。
- 它不是单一个体名，也不是所有新怪物的总称，而是一类具有稳定命名方式、异常能力和更高层背景联系的敌对存在。

## 作为大灾害后新怪物被确认
- 文中明确写到：`典灾`是大灾害后在这个世界暗中出现的新怪物。
- 相关情报并非孤例；当时至少已经有中州与秋叶原方向的确认案例。
- 这说明它不是单一迷宫里的特殊头目，而是需要被当成广域性情报来追踪的新敌类。
- 关联出处：
  - PDF页码：2181
  - 章节标题：`078`
  - 关键词：`典災`、`新しいモンスター`、`大災害`、`ナカス`、`アキバ`
  - 日文短摘录：`〈典災〉。そう呼ばれる新しいモンスター` / `ナカスでも一件、アキバでも二件確認`

## 作为具名类别而非单一个体
- 后文进一步显示，`典灾`并不是某一个怪物的专名，而更像一类带有称号与个体名的敌对存在。
- 文中已经列出多个个体，例如`结婚之典灾卡麦萨尔`、`疫病之典灾希斯痨`、`丑闻之典灾札讽`、`度量之典灾巴古利司`、`降灵术之典灾拉丝妃雅`等。
- 配合 `xlsx` 中对 `典災` 标注的注音 `ジーニアス`，当前最稳妥的理解是：`典灾`是一个类别名，而这些带前缀的名称是该类别下的具体个体。
- 关联出处：
  - PDF页码：2186、2247
  - 章节标题：`078` / `080`
  - 关键词：`降霊術の典災`、`結婚の典災`、`疫病の典災`
  - 日文短摘录：`〈降霊術の典災〉ラスフィア` / `結婚の典災カマイサル` / `疫病の典災シスラウ`

## 数量可能随时间增长，但不能与所有新怪物画等号
- 后文进一步指出：类似目击报告在大灾害后是“随着时间推移而增加”的。
- 叙述还特别做了边界说明：并不能断言所有灾后新怪物都属于`典灾`，但其中确实混有这种类别。
- 这意味着整理时不能把“新种怪物”与`典灾`完全画等号。
- 关联出处：
  - PDF页码：2246
  - 章节标题：`080`
  - 关键词：`時間が経過するほどに増えてきている`、`新種モンスター`、`全ては典災であるとは思わない`
  - 日文短摘录：`時間が経過するほどに増えてきている` / `新種モンスターの全ては典災であるとは思わない`

## 已知能力不只在战斗层，也在社会渗透层
- 现阶段至少能确认，部分`典灾`具备高度异常能力，而不只是数值更高。
- 已有个案显示：它们能够伪装成`大地人`、魅惑异性，并在城市内积累一时不可忽视的影响力。
- 这种“渗透社会内部”的能力，使`典灾`对城市秩序与人际关系的威胁方式明显不同于普通怪物。
- 关联出处：
  - PDF页码：2246
  - 章节标题：`080`
  - 关键词：`大地人を装う`、`異性を魅了する`、`影響力`
  - 日文短摘录：`〈大地人〉を装う` / `異性を魅了する` / `無視できない影響力`

## 与更早战争遗留的敌对体系相连
- 另一段战斗叙述又补出了更危险的层面：至少部分`典灾`并非单纯“最近才冒出来的野怪”，而与更早的大规模战争遗留有关。
- 文中直接把某个`典灾`称作“毁灭全界十三骑士团半数的那支 `ジーニアス` 军势的末端”。
- 这意味着`典灾`不一定只是大灾害后的随机新生物，也可能与这世界更古老、规模更大的敌对体系相连。
- 关联出处：
  - PDF页码：2194
  - 章节标题：`078`
  - 关键词：`ジーニアス軍勢`、`全界十三騎士団`
  - 日文短摘录：`ジーニアス軍勢の末端` / `全界十三騎士団の半分を滅ぼした`

## 与怪物体受肉、航界种等高层问题相连
- 后期书信材料与城惠的追想又把`典灾`与更高层的异界知性体问题连在了一起。
- 其中最关键的一点是：至少部分危险存在，很可能是利用月面预先准备的“高战斗能力怪物体”受肉到`赛尔迪希亚`的。
- 城惠后来也明确把`典灾`理解为“借用了《幻境神话》怪物身体的数据与根源”的存在。
- 这使`典灾`不再只是“会伪装的人形怪”，而更像借壳而来的敌对体系。
- 关联出处：
  - PDF页码：2291、3077-3078
  - 章节标题：`087` / `111`
  - 关键词：`怪物体`、`受肉`、`依代`、`モンスターの身体`
  - 日文短摘录：`戦闘能力の高い怪物体を利用` / `モンスターの身体を利用しているのが、〈典災〉だ`

## 不能被轻易解释为普通版本追加
- 文中还明确写出一个判断边界：把这些存在简单解释成《幻境神话》后续追加的新怪物，虽然方便，却未必可信。
- 城惠在当时就已经怀疑，这类说明恐怕既难以验证，也不足以解释眼前现象。
- 这使`典灾`更接近“世界规则层或敌方体系的变化”，而不是单纯的资料片补怪。
- 关联出处：
  - PDF页码：2245、2247
  - 章节标题：`080`
  - 关键词：`エネミーの変容`、`ノウアスフィアの開墾`、`真実とも思えなかった`
  - 日文短摘录：`〈大災害〉後におきたエネミーの変容` / `それが真実とも思えなかった`

## 不受“新增怪物数量”这种开发直觉严格限制
- 城惠后来承认，自己曾错误地以“扩展包开发量有限”为前提，低估了`典灾`的威胁上限。
- 其后续推理指出：它们完全可以重复利用既有素材、旧怪物根源和现成系统，继续延展出新的变体。
- 这说明`典灾`不是一次性“固定数量的新 Boss 列表”，而是能在既有世界资产上继续生长的敌对类别。
- 关联出处：
  - PDF页码：3077-3079
  - 章节标题：`111`
  - 关键词：`依代`、`再利用`、`素材`、`脅威度`
  - 日文短摘录：`そこらにある素材を再利用して` / `依代であるレイドボスモンスター`

## 也可能掌握类似口传的成长与系统逆用
- 后期战斗中，城惠进一步提出更不妙的判断：`典灾`不只是“会用怪物身体”的敌人，它们还可能像`冒险者`一样，沿着《幻境神话》的游戏根继续获得成长。
- 他甚至明确判断，造成大规模等级下降的手段，是`典灾`逆向利用了`师徒系统`所形成的、属于它们自己的“口传式攻击”。
- 这意味着只要世界仍保有游戏根，它们就可能持续开发出新的敌方技术，而不是停留在固定技能表上。
- 关联出处：

## 中原线已确认多种不同职能的个体
- 到`狼君山`大战线阶段，友军已能把某个女性怪物直接识别为`治疗之典灾泡浦斯`。
- 再往后，克拉斯提恢复片段记忆时，已经明确把自己在`昆仑`遭遇并施下`魂冥咒`的对象指向`魔女之典灾布可妃`。
- 这说明`典灾`个体之间至少存在鲜明的职能分化，不同个体可对应前线治疗、诅咒施加、高层干预等不同角色。
- 关联出处：

## 个别战例还显示了双层结构与精神攻击
- 秋叶原后期异变战里，出现了一组新的具名个体：外层巨体是`失望の典災エレイヌス`，而胸腔裂口中又藏着一体`35`级的`憧れの典災エレイヌス`。
- 同一战还补出另一类能力：它会围绕“你是否拥有崇敬对象”持续发问，并以紫色光线施加`精神属性攻击`、抽取`MP`，伴随与重要对象相关的记忆碎片剥离表现。
- 现阶段更稳妥的写法是：这是已确认的个别战例，不直接推广成所有`典灾`的通则。
- 判定：`情境性表现`

## 与共感子获取的关系
- 更后期的推理还显示，多个`典灾`的行为模式虽然外观不同，核心却可能都指向同一目标：获取`共感子`。
- 文中列举的方式包括以欲望、恋爱、归还地球、绝望等不同切口诱导目标。
- 城惠据此进一步推断：它们真正想取得的并不只是击杀成果，而是需要目标某种意义上的“同意”或主动交付。
- 判定：`高概率效果`
- 关联出处：
  - PDF页码：3295
  - 章节标题：`117`
  - 关键词：`共感子`、`同意`、`カマイサル`、`タリクタン`
  - 日文短摘录：`彼らは同意を求めている` / `提供者の同意を必要とする`

## 与大地人的“现实感”形成对照
- 角色还提出了一个很重要的体感判断：与越来越像真实人的`大地人`相比，`典灾`反而更像旧《幻境神话》里的“游戏怪物”。
- 这不是严密规则说明，而是角色在交战后的直观比较；但它提示`典灾`可能在存在方式上与`大地人`、普通怪物都不完全相同。
- 判定：`主观体感`
- 关联出处：

## 阶段变化
- 初期阶段，`典灾`首先以“大灾害后新出现的异常怪物类别”身份被确认。
- 随后，文本又逐步补出它作为“具名类别而非单一个体”的结构，以及渗透社会、伪装、魅惑与广域目击增加等特点。
- 到后期，它进一步与`怪物体受肉`、`航界种 / 共感子`、`师徒系统逆用`、中原线不同职能个体，以及秋叶原异变战中的特殊战例连成了一整条高层敌对体系线。

## 对中原服的参考价值
- 对中原服写作最有价值的不是照抄某个`典灾`个体，而是继承下面这些方向：
  - 大灾害后可能出现“带有异常知性与规则外能力的新敌类”
  - 这种敌类不一定只在野外出现，也可能渗透城市和社会关系
  - 它们与`大地人`、普通怪物、玩家之间，可能构成另一条后期主线

## 当前未知或仍待补证的部分
- `典灾`的起源、分类标准、与普通怪物的系统差异仍未完全明示。
- `ジーニアス`、`典灾`、`航界种`、`共感子`等高层概念之间的完整对应关系，当前也还不能写死。
- 个别战例里出现的“双层内核结构”“精神属性诱导”等表现，现阶段仍应视为已确认战例，而非整个类别的通用规则。
