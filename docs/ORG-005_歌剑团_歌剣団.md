# ORG-005 歌剑团（歌剣団）

- 分类：组织与制度 / 公会与派系
- 适用范围：特定服务器特有
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`歌剑团`是中原服务器当前已能确认的强势扩张型公会样本。现有材料最稳定地说明了它在边境现实中的角色：会反复向中小公会施压，要求结盟、归附乃至纳入麾下；其背后又明显连向`苍王派`一侧的更大政治框架。就目前正文范围而言，最稳妥的整理方式，是把它视为中原服大战争秩序下“派系触手型公会”的入口样本。

## 组织性质
- 现阶段最合理的理解方式，是把`歌剑团`视为中原服大型兼并阵营在地方运作时的实际执行公会。
- 正文里真正上门交涉和施压的是`歌剑团`使者，但要求加入的对象却是`苍王派`。
- 这说明它并不只是一个孤立公会，而是嵌在更大的派系竞争结构里。
- 关联出处：

## 会反复向边境公会施压
- 朱桓回到总部时，已经被告知来访者是`歌剑团`的使者。
- 文中说明，这不是一次性拜访，而是“这两个月来反复前来”，提出的要求也从结盟一路推进到“纳入伞下”。
- 这说明`歌剑团`的行为更接近持续政治施压，而不是普通外交试探。
- 关联出处：
  - PDF页码：2482
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`歌剣団`、`同盟`、`傘下`
  - 日文短摘录：`ギルドハウスに来たのは〈歌剣団〉の使者だ` / `同盟を結ぼうだの、傘下に入れ`

## 站在苍王派一侧，而非独立中立势力
- 紧接着的对话里，朱桓等人明确把来访者的要求描述为“加入苍王派”。
- 同一段里又出现`白王`、`红王派`这些对照对象。
- 因而当前更稳妥的写法，是把`歌剑团`视为`苍王派`阵营一侧的扩张公会，而不是完全独立于王号派系之外的中立组织。
- 关联出处：
  - PDF页码：2481-2482
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`蒼王閥`、`白王`、`紅王`
  - 日文短摘录：`蒼王閥に入らないと後悔をする` / `白王や紅王じゃ`

## 施压逻辑与交战区秩序直接相连
- 使者传达的逻辑并不掩饰现实威胁：
  - 如果不归附
  - 不仅公会自身会吃亏
  - 连这座城市的`大地人`也可能被卷进去
- 这说明`歌剑团`不是简单的“友好协商型大型公会”，而是活在交战区与领主化逻辑中的现实权力组织。
- 关联出处：
  - PDF页码：2481
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`大地人も皆殺し`、`臣従`
  - 日文短摘录：`この都の〈大地人〉も皆殺しだぞ` / `臣従せよ`

## 对边境中小公会形成持续压迫
- 在`乐浪狼骑兵`一侧看来，`歌剑团`代表的不是抽象政治，而是现实的扩张压力。
- 朱桓等人的反应也表明：边境中小公会要么被卷入大战略，要么被迫选边站。
- 这使`歌剑团`很适合作为中原服“大型强势公会如何向边境推进”的样本。
- 关联出处：
  - PDF页码：2481-2482
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`無理やり`、`下に入れ`
  - 日文短摘录：`なんだか連中、無理やりじゃないっすかね` / `下に入れって？`

## 其整编策略可能与战乱成本失控有关
- 马抱提出了一个带有推测性质的重要判断：这些大型势力也许终于意识到，战争打得太狠会导致`大地人`逃散、农业和商业停摆。
- 如果这一判断成立，那么`歌剑团`近来加强拉拢和整编中小公会，未必只是为了扩张，也可能是为了把既有支配区重新稳定下来。
- 判定：`高概率效果`
- 关联出处：
  - PDF页码：2485
  - 章节标题：`096 Chapter 1.04`
  - 关键词：`大地人に被害`、`逃げられちゃったら`、`畑も商業も止まっちゃう`
  - 日文短摘录：`〈大地人〉に被害出ちゃう` / `畑も商業も止まっちゃう`

## 当前已知的活动边界
- 以现有材料看，`歌剑团`已经能稳定把影响力施加到`草原之都`这类中原外缘城市。
- 但其正式据点、会长、兵力规模与内部层级，正文还没有展开到可系统整编的程度。
- 因而现阶段仍更适合把它写成“强势扩张型公会样本”，而不是已经完整还原的大型公会全传。

## 阶段变化
- 在当前材料里，`歌剑团`首先作为对`乐浪狼骑兵`施压的前出使者组织出现。
- 随着对话展开，它又进一步显出自己并非独立中立公会，而是`苍王派`阵营向边境延伸的实际触手。
- 到这一阶段，它已被稳定塑造成中原大战争秩序下“持续整编边境中小公会”的扩张型公会样本。

## 对中原服的参考价值
- `歌剑团`对你最有价值的不是某几个具体人物，而是它所代表的结构：
  - 大型势力会通过前出公会向边境扩张
  - 这种扩张往往伴随“保护你不被别家打掉”的现实威胁
  - 在领地化和交战区制度下，公会外交天然会带着半强制性质

## 当前未知或仍待补证的部分
- `歌剑团`与`苍王派`之间究竟是直属、附庸、盟友还是更松散的阵营关系，当前仍应保留边界。
- 其会长、中心据点、常驻成员规模与日常治理职能，目前正文还没有足够稳定的出处链。
- 它在中原主战区内部是否掌握实际领地，现阶段也尚未坐实。
