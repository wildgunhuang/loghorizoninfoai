# MON-002 鵺（鵺）

- 分类：怪物与生物 / 中原线大规模战斗怪物
- 适用范围：大概率共通
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`鵺`是在`狼君山`地下战线中正式出现的具名魔兽。现有材料能够稳定确认：当前登场样本是操纵电击的巨型`魔兽`，被克拉斯提等人直接辨认为`鵺`；战斗上至少属于`十二人战斗级别`，当前样本等级为`84`，弱点是`打击`，并具备`电击耐性`。更重要的是，地方住民明确表示它不是自己熟悉的`狼君山`守护生态，而后续又出现了“来自天外”的说法，因此它对中原线而言不只是普通副本怪。

## 类别性质
- 现阶段最稳妥的理解方式，是把`鵺`视为一类已被明确识别的大规模战斗级魔兽样本。
- 当前正文最完整展开的是`狼君山`一线的这个个体；因此本条应以“已确认样本”为主，不把单一战例过度推广成所有`鵺`的通则。

## 当前登场样本是操纵电击的巨大魔兽
- 花貂第一次目击它时，先把它辨认为`狼君山`从未见过的邪恶`魔兽`。
- 她还明确把那股焦臭味与其四肢、吐息缠绕的青白电光联系起来。
- 这说明当前样本的稳定特征至少包括：巨体、魔兽属性、电击系攻击与强烈异质感。
- 关联出处：
  - PDF页码：2604
  - 章节标题：`101 Chapter 4.03`
  - 关键词：`魔獣`、`電光`、`狼君山`
  - 日文短摘录：`この〈狼君山〉で見たことがない種類` / `魔獣だ` / `青白い電光`

## 被克拉斯提等人直接辨认为鵺
- 正式命名并不是旁白总结，而是克拉斯提与同行者在交手一瞬间直接说出的。
- 这让“`鵺`”成为当前样本的明确识别，而不是花貂的地方误称。
- 关联出处：
  - PDF页码：2604
  - 章节标题：`101 Chapter 4.03`
  - 关键词：`鵺`
  - 日文短摘录：`「鵺ですね」` / `「鵺だね！」`

## 当前样本属于十二人战斗级别
- 克拉斯提一侧在接敌瞬间，就给出了非常系统化的判断：这只怪物属于`十二人战斗级别`，大概是和天塔同型的`半团战`或近似类别。
- 这条信息非常有价值，因为它直接把文学描写落回了系统规模判断。
- 关联出处：
  - PDF页码：2605
  - 章节标题：`101 Chapter 4.03`
  - 关键词：`十二人戦闘クラス`、`ハーフレイド`
  - 日文短摘录：`この大きさだと十二人戦闘クラス` / `同型ハーフレイドか類似`

## 已明确出现弱点、耐性与等级判断
- 同一段战斗判断里，克拉斯提直接报出了两条重要参数：`弱点是打击`，并且具备`电击耐性`。
- 后续战斗分析进一步补出：这只`鵺`的等级是`84`。
- 因而当前样本的基础战斗情报已经足够稳定，可以单独保留。
- 关联出处：
  - PDF页码：2605、2608
  - 章节标题：`101 Chapter 4.03`
  - 关键词：`弱点は打撃`、`電撃耐性`、`レベルは八十四`
  - 日文短摘录：`弱点は打撃、電撃耐性` / `魔獣のレベルは八十四`

## 对少人数高等级队伍仍具有致命消耗性
- 同一段分析明确指出：如果有符合定员的`十二人`讨伐队，这场战斗原本会容易得多。
- 但现实情况是参战者人数只有标准编制的一小部分，于是问题转成了`MP`与持续力会先耗尽。
- 这说明`鵺`的危险，不仅在于瞬间秒杀，更在于大规模战斗怪在少人数遭遇下会形成典型的资源耗死局面。
- 关联出处：
  - PDF页码：2608-2609
  - 章节标题：`101 Chapter 4.03`
  - 关键词：`定数十二名`、`四分の一`、`ＭＰは人数に比例`
  - 日文短摘录：`適正レベル十二人` / `戦闘参加者が定数の四分の一` / `ＭＰは人数に比例する`

## 并非花貂所知的狼君山本地守护兽
- 花貂在战斗中不断想表达的一点是：自己从未见过这种怪物。
- 在她的地方知识里，`狼君山`应当由通往`昆仑`的圣地守护狼守卫，而不是这种外来丑恶的雷电魔兽。
- 这说明当前样本至少不应被轻率写成“狼君山原生常规守护怪”。
- 关联出处：
  - PDF页码：2609
  - 章节标题：`101 Chapter 4.03`
  - 关键词：`こんな魔獣は知らない`、`狼君`
  - 日文短摘录：`こんな魔獣は知らない` / `守護をしてくれているのは狼君`

## 后续出现了“来自天外”的解释线索
- 到叶莲仙女重新现身时，又出现了一个新的来源说法：`那东西是从天上来的`。
- 以当前证据来看，这更接近后续阶段的解释线索，而不能直接当成已完全证实的世界真相。
- 但它足以提示：当前这只`鵺`很可能不是普通迷宫刷新的本地怪。
- 判定：`高概率效果`
- 关联出处：

  - 关键词：`宙からきた`

## 阶段变化
- 当前材料里，`鵺`首先以`狼君山`地下战线上的巨大电击魔兽样本出现，并被直接识别为`十二人战斗级别`的强敌。
- 随着战斗分析推进，文本又进一步补出了它的等级、弱点、耐性和少人数遭遇下的资源耗死特征，使其从“怪异魔兽”提升为有明确系统参数的 raid 级样本。
- 到后续解释阶段，`叶莲仙女`提出“来自天外”的说法，使`鵺`不再只是地方怪物战例，而开始接入更高层的来源疑问。

## 对中原服的参考价值
- `鵺`这条词条对你有两层价值：
  - 怪物层面：它提供了“中原本地遭遇的大规模战斗怪”的明确样本
  - 系统层面：它把`十二人战斗级别 / 等级84 / 少人数资源耗尽`这一整套战斗判断落得很实

## 当前未知或仍待补证的部分
- 当前正文最完整展开的是`狼君山`上的这一只样本，尚不足以完整定义“鵺”这个类别在全服的常态生态。
- 它与`典灾`、`時限解放の獣`、`天外来物`之间的最终关系，目前仍待继续追证。
- 其是否必然属于 raid 怪，还是也可能以别种规模出现，现阶段也不能写死。
