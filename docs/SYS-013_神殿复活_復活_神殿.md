# SYS-013 神殿复活（復活 / 神殿）

- 分类：世界观 / 死亡机制
- 适用范围：大概率共通
- 对中原服的可用性：可直接沿用
- 日本服依赖度：低

## 摘要
异世界中的玩家死亡后并不会就此消灭，而是在短暂时间差后于城市的神殿复活。这一规则在开篇即被观测确认，后续又被`魂魄理论`推进为较完整的解释模型；到小说后段，还出现了可使`冒险者`不走`大神殿`路线的特殊例外。

## 设定性质
- 本词条记录的是“死亡后通常如何回到活动状态”的总入口规则。
- 它既包括前期的直接观察，也包括后期的解释模型与特殊例外。
- 因此不能只用“会复活”一句话概括，也不能用后期例外反过来推翻前期常规。

## 初期直接观察是死后会在神殿复活
- 开篇阶段通过目击玩家主动攻击卫兵、自杀式死亡并在神殿复活，确认了“死后复活”这一规则仍然有效。
- 这也使“死亡后就能回到现实世界”的早期猜想被初步否定。
- 关联出处：
  - PDF页码：73-74
  - 章节标题：`004`
  - 关键词：`死からの復活`、`神殿`
  - 日文短摘录：`聖堂で復活を遂げる` / `神殿へと向かい`

## 初期尚未确认完整惩罚结构
- 在最早的确认阶段，文本能直接证明的是“会复活”。
- 经验值、金钱或其他原游戏中的死亡惩罚是否仍完整存在，当时尚无直接实证。
- 因此早期结论应写成“已确认复活存在，但惩罚细节当时未明”，而不是事后回填成一开始就全都清楚。

## 后续解释模型指出会经由落魄与大神殿重构
- 后文借`李・耿恩`的研究提出了一套更细的解释。
- 死亡后，肉体侧的`魄`会开始扩散，这一过程被称为`落魄`。
- 对`大地人`来说，只要在魄尚未完全散尽前施加`苏生魔术`，仍有机会被拉回；一旦彻底散尽，死亡就会变成决定性的。
- 对`冒险者`来说，落魄结束时，或本人主动希望结束时，肉体会连同装备一起粒子化，并被转送到`大神殿`进行自动重构与自我苏生。
- 关联出处：
  - PDF页码：735-736
  - 章节标题：`025`
  - 关键词：`落魄`、`蘇生魔術`、`大神殿`、`自己蘇生`
  - 日文短摘录：`この過程を落魄と呼ぶ` / `大地人にも〈蘇生魔術〉は有効` / `大神殿に転送` / `自己蘇生`

## 死亡代价至少包括经验值损失
- 这套解释同时说明了经验值损失的来源：并不是单纯“系统扣罚”，而是苏生或重构时不可避免的`信息劣化`。
- 这意味着`冒险者`的死亡从设定上并非无成本。
- 关联出处：
  - PDF页码：735-736
  - 章节标题：`025`
  - 关键词：`経験値がロス`、`情報劣化`
  - 日文短摘录：`経験値がロスする` / `情報劣化`

## 后续又出现记忆损失风险
- 更重要的后续变化是：文中明确提出“记忆”横跨魂与魄两侧存在，因此每一次死亡与重构，都可能造成记忆缺损与劣化。
- 这里在当前阶段仍属于研究假说与角色推定，不应写成百分之百已被系统公告的铁律；但它已经足够重要，必须作为阶段性变化保留。
- 关联出处：
  - PDF页码：736-738
  - 章节标题：`025`
  - 关键词：`記憶`、`魂と魄`、`欠損`、`劣化`
  - 日文短摘录：`記憶とは魂と魄の両方にまたがって存在` / `記憶は欠損する` / `記憶は、死により欠損、劣化`

## 风险后来进入公开决策层
- 到后文，这一风险已经不是城惠私下持有的疑虑，而是被他明确向核心决策层报告的重大事项。
- 文中直接写到，大规模战斗之所以让人犹豫，原因之一正是`魂魄理论`下的死亡风险。
- 这说明“死亡有记忆代价”在剧情推进中已经开始转化为实际战略约束。
- 关联出处：
  - PDF页码：807, 828
  - 章节标题：`026`；`027`
  - 关键词：`魂魄理論`、`ためらい`、`わずかな記憶を失う`
  - 日文短摘录：`魂魄理論の問題` / `ためらいがある` / `わずかな記憶を失う`

## 后期出现了可绕开大神殿路线的特殊例外
- 到小说后段，文本终于出现了此前一直需要追踪的严格例外。
- 具名刀`六道輪廻`被直接说明为“在大和也没有第二把的、能够杀死`冒险者`的武器”。
- 同段还明确写到：持刀者`カズ彦`相信，被这把刀杀死的人会被送往与其相称的世界，至少不会前往`大神殿`。
- 这说明“`冒险者`通常会在神殿复活”这一常规规则，到这里已经出现了特殊道具层面的破口。
- 但现阶段仍需保留边界：
  - 文本明确的是`六道輪廻`的特殊性，而不是一般死亡规则已被推翻。
  - “一定回到地球”主要是`カズ彦`基于道具风味文本的信念，不应直接写死为系统公告。
- 关联出处：
  - PDF页码：3250-3254
  - 章节标题：`116`
  - 关键词：`六道輪廻`、`冒険者を殺せる武器`、`大神殿へは向かわない`
  - 日文短摘录：`〈冒険者〉を殺せる武器` / `大神殿へは向かわない`
- 备注：这条属于“后续例外”，不得反过来覆盖前文“通常会在神殿复活”的常规规则。详见 [ITM-003 六道轮回（六道輪廻）](ITM-003_六道轮回_六道輪廻.md)。

## 阶段变化
- 观察阶段：先确认“死后会在神殿复活”。
- 理论阶段：后文用`魂魄理论`解释`落魄`、`大神殿`与重构。
- 代价阶段：再进一步补出经验值损失与记忆损失风险。
- 例外阶段：后期出现`六道輪廻`这类可切断常规路线的特殊道具。
- 这四层必须同时保留，不能以后面的理论或例外覆盖前面的观察记录。

## 对中原服的参考价值
- 这条规则是中原服战争、远征、领地治理与牺牲判断的基础。
- 可直接继承的是“通常复活存在，但有代价、也可能有极少数例外”这一完整框架，而不是把`六道輪廻`式例外当作常态。

## 当前未知或仍待补证的部分
- 早期开篇阶段的完整死亡惩罚清单并未一次性公开。
- 记忆损失是否必然发生、是否有程度差异、是否存在修复路径，当前仍待补证。
- `六道輪廻`之外是否还存在其他可切断`大神殿`复活路线的手段，当前未知。
