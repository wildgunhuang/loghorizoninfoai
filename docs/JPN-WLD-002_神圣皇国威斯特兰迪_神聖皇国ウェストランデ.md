# JPN-WLD-002 神圣皇国威斯特兰迪（神聖皇国ウェストランデ）

- 分类：世界观 / 日本服地方政体
- 适用范围：日本服特有
- 对中原服的可用性：高参考价值
- 日本服依赖度：高

## 摘要
`神圣皇国威斯特兰迪`是日本服大和西部的大地人政体。它不能直接照搬到中原服，但它作为“拥有正统主张、军事压力与王朝继承叙事的地方国家”这一样本，对中原服地方政权设计很有参考价值。

## 基本性质
- 文中明确说明，大和在古代`威斯特兰德皇王朝`灭亡后，被东边的`自由都市同盟伊斯塔尔`与西边的`神圣皇国威斯特兰迪`所二分。
- 两者都被说成是古王朝的后裔，但`神圣皇国威斯特兰迪`会以“古王朝的正统后继者”身份自居。

## 正统主张与对伊斯塔尔的压力
- 当前阅读范围里，`神圣皇国威斯特兰迪`最明确的政治姿态，是以正统继承者名义要求`自由都市同盟伊斯塔尔`纳入其统治之下。
- 这说明它并不只是“西边另一个国家”，而是带有主动统一诉求和合法性话语的扩张性政体。
- 关联出处：

## 作为现存支配组织的地位
- 另一处叙述又从西方内部视角补了一个关键点：`神圣皇国威斯特兰迪`被称作“将威斯特兰德皇王朝血脉传到现在的唯一支配组织”。
- 这让它的正统主张不再只是外交口号，而是其内部政治自我理解的一部分。
- 同段还显示，在乱世背景下，军事人物与军功上升空间明显增大，说明这套政体与军政秩序联系紧密。

## 后期已被明确写成“统治组织脆弱”的国家
- 到更后面的西国政争讨论里，`神圣皇国威斯特兰迪`被直接判断为处在“未曾有的政争”之中。
- `アインス`进一步把它的根本问题概括成“统治组织的脆弱”。
- 这说明它虽然仍以大国与正统自居，但到当前阶段已经不能简单写成稳定国家，而应视作一个正在因多重对立而脆化的政体。

## 当前主要裂口
- 同一段对话已经把西国当前的主要裂口说得非常清楚：
  - `元老院`与其他势力的对立
  - `大地人`与`冒険者`的对立
  - `Ｐｌａｎｔ　ｈｗｙａｄｅｎ`与旧贵族的对立
- 这意味着`神圣皇国威斯特兰迪`的危机不是单一宫廷政变，而是多条社会裂口叠在一起的政体危机。

## 在冒险者力量面前并不强势
- `アインス`还明确给出一个非常尖锐的判断：无论`元老院`还是西国其他大地人势力，相比`冒険者`都算不上真正强大。
- 在这个视角里，当前动摇西大和的根源并不只是传统国家本身，而是`冒険者`巨型组织已经成长为足以压倒本地国家的力量。
- 这条判断很重要，因为它修正了“威斯特兰迪 = 西方强国”的表面印象：它更像是一个在新世界权力结构里被重新暴露出虚弱性的旧国家。

## 阶段变化
- 在较早材料里，`神圣皇国威斯特兰迪`首先被写成与`伊斯塔尔`并列、并自居古王朝正统后继的西方大地人政体。
- 随着西国相关材料增加，它又进一步显出“以正统主张对外施压、带有军政上升空间”的扩张型旧国家面貌。
- 到更后期，它则被明确写成因多条社会裂口叠加而显露“统治组织脆弱”的政体危机样本。

## 对中原服的参考价值
- 可参考的不是“日本西部有个威斯特兰迪”，而是这套结构：
  - 某一地方国家用古王朝血统来主张合法性
  - 它与邻近政体共享同一个历史源头，却争夺谁才是正统
  - 在灾后与乱世环境里，军功与军事化倾向会加强这种国家的扩张压力
