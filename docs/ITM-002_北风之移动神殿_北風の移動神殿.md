# ITM-002 北风之移动神殿（北風の移動神殿）

- 分类：道具与装备 / 圣具型设施道具
- 适用范围：日本服实现形式，底层规则可能共通
- 对中原服的可用性：可改造沿用
- 日本服依赖度：中

## 摘要
`北风之移动神殿`是后期文本中出现的具名圣具。当前阅读范围内可确认：它被`大地人`认知为由冒险者使用、可随行携带的`移动神殿`，至少具备“让附近死亡的冒险者复活”这一接近复活点替代物的功能；同时它似乎还伴随扰乱心声之类的副作用。

## 外观与携行方式
- 文中把它描述成一种由多人抬运的巨大器具，外形近似神轿，但装饰更偏西式。
- 它不是随身小道具，而更接近“可移动的圣坛 / 设施型器物”。
- 关联出处：
  - PDF页码：1849
  - 章节标题：`066`
  - 关键词：`北風の移動神殿`、`神輿`、`聖具`
  - 日文短摘录：`神輿のようにみえた` / `〈北風の移動神殿〉` / `聖具だとききました`

## 已知功能
- 当前最重要的已知功能，是它会让附近死亡的`冒险者`复活。
- 文中角色在理解这一效果时，甚至直接把它视作“复活地点的代替”乃至“复活地点本身”。
- 如果这一理解无误，那么它的意义已经不是普通辅助道具，而是能局部改写复活点逻辑的设施级圣具。
- 关联出处：
  - PDF页码：1850-1851
  - 章节标题：`066`
  - 关键词：`復活させる`、`復活地点の代替`、`移動神殿`
  - 日文短摘录：`〈冒険者〉の皆さんを復活させる` / `復活地点の代替` / `復活地点そのもの`

## 已知副作用与风险
- 同段还提到，这件圣具似乎会“搅乱心中的声音”。
- 当前文本没有把这一点解释成明确系统机制，因此还不能写成确定规则；但它至少已经说明，这类移动复活设施并非纯粹无代价的福音。
- 判定：`待考证`
- 关联出处：
  - PDF页码：1850
  - 章节标题：`066`
  - 关键词：`副作用`、`心の声をかき乱す`
  - 日文短摘录：`副作用なのかわかりませんが` / `心の声をかき乱す`

## 与大神殿的关系
- 目前最稳妥的理解是：`大神殿`仍是标准、固定的复活设施；`北风之移动神殿`则是能局部替代这一功能的移动型变体。
- 因此它不应覆盖`大神殿`词条，而应作为“复活设施的特殊实现形式”与之并列记录。

## 阶段变化
- 当前材料里，`北风之移动神殿`首先以被多人抬运的圣具型设施道具登场。
- 随后它又被进一步确认具备接近“移动复活点”的功能，因此从单纯圣具推进成了复活设施变体。
- 与此同时，文本也同步留下了“搅乱心声”的副作用线索，使它从一开始就不是纯粹无代价的便利道具。

## 对中原服的参考价值
- 这条设定对中原服很有启发性，因为它说明：
  - 复活设施功能未必永远固定在城市建筑上
  - 某些高阶圣具或组织专用器物，可能把复活点逻辑带到战场或远征途中
  - 但这种便利大概率会伴随风险、代价或副作用

## 当前未知或仍待补证的部分
- 这件圣具的发动条件、作用半径、复活次数限制与真正代价，当前都仍未被文本写死。
- 它与标准`大神殿`复活机制之间究竟共享多少底层规则，现阶段也还需要更多材料补证。
