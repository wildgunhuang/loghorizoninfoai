# JPN-SYS-001 凝魔锻术（凝魔鍛術）

- 分类：系统与制度 / 日本服技术附录
- 适用范围：日本服实现形式，底层规则可能共通
- 对中原服的可用性：可改造沿用
- 日本服依赖度：高

## 摘要
`凝魔锻术`是后期文本中明确提到的“西方”高层技术之一。现阶段可以确认：它并不只是单一魔法，而更接近一整套将魔力、古代技术与器械制造结合起来的魔导技术体系；但其完整原理、职业来源与可复制性，在当前阅读范围内仍未完全公开。

## 作为“西方技术”被单独提及
- 在`城惠`与`濡羽`对峙时，文中把`契约术式`与“西方的`凝魔锻术`”并列提及。
- 这说明在当时的高层情报视野里，`凝魔锻术`至少已被视作与`契约术式`同级别的重要技术情报之一。

## 已知技术谱系
- 后文给出了当前最关键的一段说明：
  - `杰鲁迪斯`被记述为`凝魔锻术`的开发者。
  - `杰雷德・耿恩`的`凝魔炉`则被描述为把它“大规模实用化”的装置或阶段。
  - 再加上`久尔永`的古代技术，三者共同生出了后续的魔导技术体系。
- 现阶段最稳妥的理解是：`凝魔锻术`并非孤立技能名，而是西方魔导技术谱系中的关键源头之一。

## 技术成果与大地人受益
- 同一段还明确指出：由这套谱系生出的魔导技术，原本当然是`冒险者`财产的重要部分，但其“外溢”也实实在在惠及了`大地人`。
- 文中的直接例子就是军用`铁钢车辆`。这些被冒险者视为失败作或玩具的技术产物，对`大地人`军政势力而言却是足以改变力量对比的宝物。
- 这说明在灾后世界里，高层魔导技术并不会只停留在玩家手中，而可能通过战争、交易、遗赠或抛弃物扩散到`大地人`社会。

## 阶段变化
- 在较早高层情报里，`凝魔锻术`首先作为“西方重要技术”被与`契约术式`并列提及。
- 随后，文本又补出它与`杰鲁迪斯`、`杰雷德・耿恩`的`凝魔炉`、以及`久尔永`古代技术之间的谱系关系。
- 到这一阶段，`凝魔锻术`已经不再只是神秘名词，而被明确成会外溢到`大地人`军政与器械层面的魔导技术源头。

## 对中原服的参考价值
- 这条设定对中原服最有价值的，不是照抄“西方的凝魔锻术”这个名字，而是继承它背后的结构：
  - 某个服务器地方势力可能掌握自己的高阶魔导工业传统
  - 这种传统可以和古代技术、遗产装置、量产器械结合
  - 它的成果会改变交通、军备、后勤与`大地人`势力结构
- 对中原服创作而言，这非常适合转化为“本地特有的魔导工艺体系”，但不应直接复制日本服西方技术谱系的人名和机构。

## 当前未知或仍待补证的部分
- `凝魔锻术`本身究竟是职业技能、工艺体系还是更上位的研究总称，当前仍未彻底写死。
- 它与`凝魔炉`之间究竟是“术式与装置”关系、还是更复杂的技术演化关系，现阶段也还需补证。
- 哪些西方政体、组织或人物掌握了这套技术的哪一部分，正文目前仍未完整展开。
