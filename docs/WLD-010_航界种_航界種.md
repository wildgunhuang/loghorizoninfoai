# WLD-010 航界种（航界種）

- 分类：世界观 / 异界知性体
- 适用范围：大概率共通
- 对中原服的可用性：可改造沿用
- 日本服依赖度：低

## 摘要
`航界种`是后期浮出的重要异界知性体类别。当前材料能够稳定确认：它们并非`大地人`或`冒险者`，而是来自其他世界、受肉于`赛尔迪希亚`的知性存在；其中至少又分为`监察者`与`采集者`两支。它们与`共感子`、`一致`、月面设施，以及`典灾`问题都有直接关联。

## 类别性质
- 现阶段最稳妥的理解方式，是把`航界种`视为“异界受肉知性体”的总类。
- 它们不是本地居民的亚种，而是跨世界进入`赛尔迪希亚`、借用本地肉体活动的外来知性体。

## 作为从异界受肉而来的存在被明示
- `成惠2`在信中明确自称：像`冒险者`从其他世界受肉到`赛尔迪希亚`一样，她们也是从其他世界受肉而来的存在。
- 她们自称`航界种`，并明确把自己说明成“你们看来属于异世界的知性体”。
- 关联出处：
  - PDF页码：2217
  - 章节标题：`084 ノウアスフィアの開墾`
  - 关键词：`航界種`、`受肉`、`異世界`
  - 日文短摘录：`私たちは私たちを〈航界種〉と呼ぶ` / `異なる世界からこのセルデシア世界へと受肉`

## 受肉时会借用本地既存肉体
- `成惠2`进一步说明：`航界种`原本并没有适合在这里直接使用的物理形状，因此到达`赛尔迪希亚`后，需要借用这个世界已经存在的肉体作为化身。
- 她甚至直说，自己现在使用的肉体，是保管在月面、且处于“没有与魂接续状态”的城惠肉体。
- 这条设定非常重要，因为它把`航界种`的“受肉”从抽象概念落到了具体实现方式。
- 关联出处：
  - PDF页码：2218
  - 章节标题：`084 ノウアスフィアの開墾`
  - 关键词：`既に存在した肉体`、`化身`、`月にて保管`、`魂との接続`
  - 日文短摘录：`既に存在した肉体を化身として借り受ける` / `月にて保管されていたあなたのもの`

## 内部分为监察者与采集者
- 同一封信里已经表明，`监察者`与`采集者`都属于`航界种`。
- 后文又进一步补充：`监察者`与`采集者`虽然同属`航界种`，但受肉策略不同。
- `监察者`更接近借用与冒险者相近的肉体活动；`采集者`则利用月面预先准备的高战斗能力怪物体受肉，因此对`冒险者`有高度危险。
- 关联出处：
  - PDF页码：2217、2291
  - 章节标题：`084 ノウアスフィアの開墾` / `087`
  - 关键词：`監察者`、`採取者`、`戦闘能力の高い怪物体`
  - 日文短摘录：`〈監察者〉` / `〈採取者〉` / `戦闘能力の高い怪物体を利用している`

## 属于人工知性体，并背负资源采集任务
- `成惠2`把她们一方解释为人工制造的知性体。
- 她们被制造出来的目的，是为创造者社会寻找并采集`共感子`这一关键资源。
- 这说明`航界种`并不是普通外星旅人，而更像背负明确任务的跨界调查 / 采集单位。
- 关联出处：
  - PDF页码：2289
  - 章节标题：`087`
  - 关键词：`人工的な知性体`、`共感子`、`探索と採取`
  - 日文短摘录：`人工的な知性体` / `〈共感子〉と呼ぶ資源の探索と採取`

## 与月面设施和归还可能性直接相关
- 后文指出：月球上存在`监察者`社区，观察着`赛尔迪希亚`的状况。
- 如果`冒险者`追求`归还`，理论上可以从月面的`监察者`那里获得协助。
- 但紧接着又补出一个现实障碍：敌人已经夺取了“通向月面的通信设备”。
- 这使“月上有协助者”与“现实上路线受阻”两件事同时成立。
- 关联出处：
  - PDF页码：2291、2367-2368
  - 章节标题：`087` / `091`
  - 关键词：`月`、`監察者コミュニティ`、`帰還`、`通信設備`
  - 日文短摘录：`セルデシアの月には〈監察者〉コミュニティがあり` / `月への通信設備を奪った` / `帰還の決定打になる保証はない`

## 受两大原则约束的“例外相遇”
- `成惠2`的书信还提出了两条高层原则：`天理地平线之原则`与`无矛盾之大原则`。
- 按她的说法，知性生命本不该彼此相遇，矛盾也会在诞生时被消灭；而`赛尔迪希亚`这个亜世界，正是被施加了许多限制后才得以让这种相遇暂时成立。
- 这部分目前仍属于高层解释模型，但它很重要，因为它说明`航界种`与`冒险者`的相遇本身，就不是通常宇宙结构下理所当然会发生的事。
- 关联出处：
  - PDF页码：2290-2291、2295
  - 章节标题：`087`
  - 关键词：`摂理地平線の原則`、`無矛盾の大原則`、`亜世界`
  - 日文短摘录：`〈摂理地平線の原則〉` / `〈無矛盾の大原則〉` / `セルデシア亜世界`

## 与典灾的关系是强关联，但尚不能完全画等号
- 现有材料和`xlsx`都提示出一个重要但需要谨慎处理的关系：
  - `采集者`的注音与`典灾（ジーニアス）`存在强关联
  - 文本里也明确说，部分危险存在是借用怪物体受肉而来
- 但在当前阅读范围内，还不能简单地把“所有`采集者`”与“所有`典灾`”粗暴画上等号。
- 更稳妥的写法是：`航界种`中的`采集者`与`典灾`问题存在强关联，且很可能是理解`典灾`来源的重要入口。
- 关联出处：
  - PDF页码：2291、3077
  - 章节标题：`087` / `111`
  - 关键词：`採取者`、`怪物体`、`典災`
  - 日文短摘录：`戦闘能力の高い怪物体` / `〈典災〉だって`

## 阶段变化
- 初期材料里，`航界种`首先被写成从异界受肉到`赛尔迪希亚`的外来知性体总类。
- 随后，它又被细分为`监察者`与`采集者`两支，并明确带出“借用本地肉体”“月面设施”“共感子采集任务”这些高层机制。
- 到更后期，它进一步发展成解释`一致`、`月面路线`、`典灾强关联`与“例外相遇原则”的关键世界观入口。

## 对中原服的参考价值
- `航界种`属于全世界共通层的后期宇宙论，不应被当成日本服地方设定。
- 中原服故事若推进到高层世界解释、月面设施、归还路线、异界观察者等主题时，可以直接继承这一框架。
- 但具体到`成惠2`、月面交涉过程、某次书信来往等剧情实现方式，则不必照搬日本服人物线。

## 当前未知或仍待补证的部分
- `航界种`创造者社会的规模、文明结构与更完整历史，当前仍只露出轮廓。
- `监察者`与`采集者`的数量、内部层级、是否还存在第三类分支，目前也未完全展开。
- 它们与`神隠し`、`归还`、`一致`之间的完整机制链路，现阶段仍应保留边界。
