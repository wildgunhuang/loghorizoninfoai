# JPN-ORG-002 布里甘提亚（ブリガンティア）

- 分类：组织与制度 / 日本服专属组织
- 适用范围：日本服特有
- 对中原服的可用性：仅作参考
- 日本服依赖度：高

## 摘要
`布里甘提亚`是薄野一带的恶性玩家组织。它不是中原服可直接继承的设定对象，但它非常适合作为“灾后恶性公会如何从玩家群体中形成地方性暴力秩序”的日本服案例。

## 初期性质
- 文中明确指出，`布里甘提亚`在游戏时代就已经名声很差。
- 大灾害之后，这个组织迅速从“风评恶劣的玩家团体”转变成几乎真正意义上的野盗集团。
- 关联出处：
  - PDF页码：240
  - 章节标题：`009`
  - 关键词：`評判の良くなかった`、`寄り集まって`、`野盗集団`
  - 日文短摘录：`評判の良くなかったこのギルド` / `寄り集まって作った` / `本物の野盗集団`

## 主要恶行
- 在当前阅读范围内，`布里甘提亚`从事的行为包括：
  - `PK`
  - 恐吓与长期纠缠
  - 对玩家进行强迫性勧诱与监禁
  - 对`大地人`实施奴隶买卖、恐吓、虐待与搾取
- 这说明它已经不只是“喜欢杀玩家的人”，而是灾后地方暴力秩序的具体执行者。
- 关联出处：
  - PDF页码：240-243, 272-273
  - 章节标题：`009`；`010`
  - 关键词：`ＰＫ`、`脅迫`、`奴隷商売`、`恐喝`、`虐待`
  - 日文短摘录：`ＰＫなどは日常茶飯事` / `脅迫や粘着行為` / `奴隷商売や恐喝` / `虐待や搾取`

## 组织秩序
- 文中明确指出，`布里甘提亚`虽然是无赖集团，但并非真正无秩序。
- 它内部的统治原则是`力量`与`暴力`，也就是谁能打、谁能压制别人，谁就能掌控组织。
- 这使它更接近“灾后地方武力集团”而不是普通意义上的游戏公会。
- 关联出处：
  - PDF页码：272-273
  - 章节标题：`010`
  - 关键词：`無法者`、`秩序`、`力`、`暴力`
  - 日文短摘录：`真実の無法では、集団は形作れない` / `秩序とは、極論すれば力。暴力`

## 地方性
- 当前叙事里，`布里甘提亚`的问题与薄野的人口规模、城市氛围和日本服北方据点的孤立状态紧密绑定。
- 因而它更适合被理解为“日本服薄野地区的一个具体案例”，而不是全服通用组织模板。

## 阶段变化
- 游戏时代，`布里甘提亚`首先只是名声恶劣的玩家团体。
- 到大灾害后，它迅速转化为以`PK`、恐吓、监禁与掠夺为生的地方性暴力集团。
- 同时，作品也明确显示它并非纯粹无序，而是发展出了以力量与暴力维持的内部秩序，因此更接近灾后地方武装组织而非普通问题公会。

## 对中原服的参考价值
- 可借鉴的不是组织名与成员，而是这类结构：
  - 灾后失序地区出现地方性暴力公会
  - 其收入来源不是单一 PK，而是复合型掠夺与支配
  - 其内部维持靠暴力威望，而不是正式制度
  - 其统治稳定度会随着首领威望受损而动摇

## 当前未知或仍待补证的部分
- `布里甘提亚`更完整的成员规模、上下层结构与持续运转方式，当前正文仍未完全展开。
- 它与薄野当地其他玩家势力、`大地人`网络之间的更复杂关系，也还缺少系统性回填。
- 其首领层和内部派别是否存在更明确分工，现阶段同样不能写死。
